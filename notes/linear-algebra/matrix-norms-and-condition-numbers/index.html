16. Matrix norms and condition numbers

{% include note-topic-nav.html back_label="Back" back_url="/notes/linear-algebra/" prev_label="Last page" prev_url="/notes/linear-algebra/positive-definite-matrices/" next_label="Next page" next_url="/notes/linear-algebra/dual-spaces-and-adjoints/" %}

Matrix norms and condition numbers records the objects, main results, proof mechanisms, and boundary cases needed for the linear algebra note.

Definition

Definition. A matrix norm measures the size of a linear map, usually as an operator norm \(\|A\|=\sup_{\|x\|=1}\|Ax\|\). The condition number of an invertible matrix is \(\kappa(A)=\|A\|\|A^{-1}\|\).

Proposition

Proposition. For solving \(Ax=b\), relative perturbations in \(b\) can be amplified by at most \(\kappa(A)\). Equality is approached along singular-vector directions for the Euclidean norm.

Proof

Proof. Write \(\Delta x=A^{-1}\Delta b\), then bound \(\|\Delta x\|\le\|A^{-1}\|\|\Delta b\|\) and relate \(\|b\|\) to \(\|A\|\|x\|\).

Remark

Remark. Conditioning is a property of the linear problem, not of a particular algorithm. This distinction is central in numerical analysis.
