13. Singular value decomposition

{% include note-topic-nav.html back_label="Back" back_url="/notes/linear-algebra/" prev_label="Last page" prev_url="/notes/linear-algebra/spectral-theorem-for-normal-operators/" next_label="Next page" next_url="/notes/linear-algebra/bilinear-and-quadratic-forms/" %}

Singular value decomposition records the objects, main results, proof mechanisms, and boundary cases needed for the linear algebra note.

Definition

Definition. The singular values of \(A:V\to W\) are the square roots of eigenvalues of \(A^*A\). They measure stretching along orthogonal input directions.

Singular value decomposition

Theorem. Every matrix has a singular value decomposition \(A=U\Sigma V^*\), with unitary or orthogonal \(U,V\) and diagonal nonnegative \(\Sigma\).

Proof

Proof. Apply the spectral theorem to \(A^*A\), use eigenvectors as right singular vectors, and normalize their images under \(A\) to obtain left singular vectors.

Remark

Remark. SVD is simultaneously a structural theorem, a metric theorem, and a rank theorem. It is the safest way to discuss low-rank approximation and ill conditioning.
