2. Linear combinations, span, and independence

{% include note-topic-nav.html back_label="Back" back_url="/notes/linear-algebra/" prev_label="Last page" prev_url="/notes/linear-algebra/vector-spaces-and-subspaces/" next_label="Next page" next_url="/notes/linear-algebra/bases-and-dimension/" %}

Linear combinations, span, and independence records the objects, main results, proof mechanisms, and boundary cases needed for the linear algebra note.

Definition

Definition. A finite linear combination is \(\sum_i a_i v_i\). The span of a set is the collection of all finite linear combinations, while linear independence means the zero combination has only zero coefficients.

Proposition

Proposition. A spanning list is minimal exactly when it is linearly independent; an independent list is maximal exactly when it spans. These equivalent characterizations define a basis.

Proof

Proof. If a spanning list is dependent, one vector is a combination of earlier vectors and can be removed. If an independent list does not span, adding a vector outside its span preserves independence.

Remark

Remark. This exchange mechanism is the engine behind dimension, rank, coordinates, and finite-dimensional classification.
