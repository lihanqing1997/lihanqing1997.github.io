7. Closed graph theorem

{% include note-topic-nav.html back_label="Back" back_url="/notes/functional-analysis/" prev_label="Last page" prev_url="/notes/functional-analysis/open-mapping-and-inverse-theorems/" next_label="Next page" next_url="/notes/functional-analysis/hahn-banach-theorem/" %}

Closed graph theorem records the objects, main results, proof mechanisms, and boundary cases needed for the functional analysis note.

Definition

Definition. The graph of a linear operator \(T:X\to Y\) is \(\{(x,Tx):x\in X\}\subset X\times Y\). A closed graph means graph limits remain graph points.

Closed graph theorem

Theorem. A linear map between Banach spaces is bounded if and only if its graph is closed.

Proof

Proof. View the graph as a Banach space with the product norm. The projection from the graph to \(X\) is bijective and bounded; the inverse theorem makes its inverse bounded, which controls \(T\).

Remark

Remark. Closed graph arguments are often the cleanest way to prove an operator is continuous when direct estimates are awkward.
