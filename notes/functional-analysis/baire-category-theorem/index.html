4. Baire category theorem

{% include note-topic-nav.html back_label="Back" back_url="/notes/functional-analysis/" prev_label="Last page" prev_url="/notes/functional-analysis/examples-of-function-spaces/" next_label="Next page" next_url="/notes/functional-analysis/uniform-boundedness-principle/" %}

Baire category theorem records the objects, main results, proof mechanisms, and boundary cases needed for the functional analysis note.

Definition

Definition. A complete metric space cannot be written as a countable union of nowhere dense closed sets. This is a topological largeness theorem, not a measure theorem.

Baire category theorem

Theorem. In a complete metric space, countable intersections of dense open sets are dense. Equivalently, the space is not meagre in itself.

Proof

Proof. Given dense open sets, construct nested closed balls with radii tending to zero, each contained in the next open set. Completeness gives a point in the intersection.

Remark

Remark. Baire category powers the uniform boundedness principle, open mapping theorem, and closed graph theorem. Completeness again produces structure.
