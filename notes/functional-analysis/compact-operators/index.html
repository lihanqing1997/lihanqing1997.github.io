15. Compact operators

{% include note-topic-nav.html back_label="Back" back_url="/notes/functional-analysis/" prev_label="Last page" prev_url="/notes/functional-analysis/riesz-representation-theorem/" next_label="Next page" next_url="/notes/functional-analysis/spectral-theory-for-compact-self-adjoint-operators/" %}

Compact operators records the objects, main results, proof mechanisms, and boundary cases needed for the functional analysis note.

Definition

Definition. A bounded linear operator is compact if it maps bounded sets to relatively compact sets. Compact operators behave like infinite-dimensional analogues of matrices with singular values tending to zero.

Proposition

Proposition. A compact operator sends weakly convergent sequences to norm-convergent sequences, provided the domain is reflexive or the weak convergence is already given.

Proof

Proof. Weakly convergent sequences are bounded. Compactness gives norm-convergent subsequences of their images, and weak convergence identifies every such subsequential limit with the image of the weak limit.

Remark

Remark. Compactness restores finite-dimensional spectral behavior, but also creates ill-posed inverse problems because compact inverses are usually unbounded.
