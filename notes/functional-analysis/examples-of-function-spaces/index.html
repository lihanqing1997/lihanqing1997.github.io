3. Examples of function spaces

{% include note-topic-nav.html back_label="Back" back_url="/notes/functional-analysis/" prev_label="Last page" prev_url="/notes/functional-analysis/banach-spaces-and-completeness/" next_label="Next page" next_url="/notes/functional-analysis/baire-category-theorem/" %}

Examples of function spaces records the objects, main results, proof mechanisms, and boundary cases needed for the functional analysis note.

Definition

Definition. Core examples include \(C(K)\) with the sup norm, \(L^p\) spaces with integral norms, sequence spaces \(\ell^p\), and Sobolev spaces controlling weak derivatives.

Proposition

Proposition. The space \(C(K)\) is Banach when \(K\) is compact, and \(L^p\) is Banach for \(1\le p\le\infty\). Hilbert structure occurs specifically at \(p=2\).

Proof

Proof. Uniform limits of continuous functions are continuous, proving completeness of \(C(K)\). Completeness of \(L^p\) follows from subsequence extraction and summability of differences.

Remark

Remark. Examples matter because functional analysis is not one geometry: sup-norm, integral-norm, and Hilbert geometries support different compactness and duality principles.
