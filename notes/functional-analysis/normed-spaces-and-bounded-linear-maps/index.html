1. Normed spaces and bounded linear maps

{% include note-topic-nav.html back_label="Back" back_url="/notes/functional-analysis/" next_label="Next page" next_url="/notes/functional-analysis/banach-spaces-and-completeness/" %}

Norms and metric structure

Definition. A norm on a vector space \(X\) is a function \(\|\cdot\|\) that is positive definite, homogeneous, and subadditive. It induces the metric \(d(x,y)=\|x-y\|\), so linear questions become topological questions.

Bounded linear maps

Definition. A linear map \(T:X\to Y\) is bounded if \(\|Tx\|\le C\|x\|\) for all \(x\). Its operator norm is \(\|T\|=\sup_{\|x\|\le1}\|Tx\|\). Bounded linear maps form \(\mathcal L(X,Y)\).

Boundedness and continuity

Theorem. For a linear map between normed spaces, the following are equivalent: boundedness, continuity at \(0\), continuity everywhere, and Lipschitz continuity. Continuity at one point therefore becomes a global estimate.

Scaling proof

Proof. If \(T\) is continuous at \(0\), choose \(\delta>0\) so \(\|x\|<\delta\) implies \(\|Tx\|<1\). For \(x\ne0\), apply this to \(\delta x/(2\|x\|)\) and rescale to get \(\|Tx\|\le(2/\delta)\|x\|\).

Equivalent norms

Definition. Two norms are equivalent if they induce the same topology, equivalently if \(c\|x\|_1\le\|x\|_2\le C\|x\|_1\). On finite-dimensional spaces all norms are equivalent; in infinite dimension this fails.

Finite versus infinite dimension

Remark. Functional analysis begins where finite-dimensional intuition stops: linear maps need not be continuous, closed bounded sets need not be compact, and dual spaces become large enough to encode topology.
