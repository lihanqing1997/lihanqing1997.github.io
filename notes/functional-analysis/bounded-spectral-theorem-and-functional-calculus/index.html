20. Bounded spectral theorem and functional calculus

{% include note-topic-nav.html back_label="Back" back_url="/notes/functional-analysis/" prev_label="Last page" prev_url="/notes/functional-analysis/spectral-theory-for-compact-self-adjoint-operators/" next_label="Next page" next_url="/notes/functional-analysis/unbounded-operators/" %}

Normal operators

Definition. A bounded operator \(T\) on a Hilbert space is normal if \(T^*T=TT^*\). Self-adjoint, unitary, and positive operators are basic subclasses of normal operators.

Continuous functional calculus

Theorem. For a bounded normal operator \(T\), there is a unique continuous functional calculus \(C(\sigma(T))\to\mathcal L(H)\), \(f\mapsto f(T)\), extending polynomials in \(z\) and \(\overline z\) and satisfying \(\|f(T)\|=\|f\|_\infty\).

Spectral measure form

Theorem. There exists a projection-valued measure \(E\) on \(\sigma(T)\) such that \(T=\int_{\sigma(T)}\lambda\,\mathrm{d}E(\lambda)\). This generalizes diagonalization from matrices to Hilbert spaces.

Self-adjoint order

Corollary. For self-adjoint \(T\), the spectrum is real. If \(T\ge0\), then \(\sigma(T)\subset[0,\infty)\), and the functional calculus defines \(T^{1/2}\).

Multiplication model

Example. Multiplication by a bounded measurable function on \(L^2\) is normal. The spectral theorem says general normal operators can be represented, after decomposition, like multiplication operators.

Beyond compactness

Remark. Compact self-adjoint operators have eigenvector expansions. General bounded normal operators may have continuous spectrum, so projections onto spectral sets replace sums over eigenvectors.
