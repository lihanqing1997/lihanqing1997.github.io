5. Uniform boundedness principle

{% include note-topic-nav.html back_label="Back" back_url="/notes/functional-analysis/" prev_label="Last page" prev_url="/notes/functional-analysis/baire-category-theorem/" next_label="Next page" next_url="/notes/functional-analysis/open-mapping-and-inverse-theorems/" %}

Statement

Theorem. Let \(X\) be Banach and \(Y\) normed. If \(\mathcal T\subset\mathcal L(X,Y)\) is pointwise bounded, meaning \(\sup_{T\in\mathcal T}\|Tx\|<\infty\) for every \(x\in X\), then \(\sup_{T\in\mathcal T}\|T\|<\infty\).

Baire proof

Proof. Set \(E_n=\{x:\sup_{T\in\mathcal T}\|Tx\|\le n\}\). These closed sets cover \(X\). By Baire, some \(E_n\) contains a ball. Translating and scaling that ball yields a uniform operator-norm bound.

Banach-Steinhaus theorem

Remark. Uniform boundedness is also called Banach-Steinhaus. It upgrades infinitely many pointwise estimates into a single norm estimate, but only because the domain is complete.

Pointwise convergence of operators

Corollary. If \(T_nx\) converges for every \(x\in X\), where \(X\) is Banach, then \(\sup_n\|T_n\|<\infty\). Pointwise convergence secretly contains uniform boundedness.

Fourier partial sums

Example. Uniform boundedness explains why certain pointwise convergence phenomena fail: if Fourier partial sum operators had uniformly good convergence on all of \(C(\mathbb T)\), their norms would need to remain bounded, but the Dirichlet kernels force growth.

Diagnostic use

Remark. To disprove universal convergence of a sequence of bounded linear procedures, show their norms blow up. Uniform boundedness then guarantees some input has bad behavior.
