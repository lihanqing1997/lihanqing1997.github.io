18. Distributions and weak derivatives

{% include note-topic-nav.html back_label="Back" back_url="/notes/functional-analysis/" prev_label="Last page" prev_url="/notes/functional-analysis/unbounded-operators/" next_label="Next page" next_url="/notes/functional-analysis/sobolev-spaces/" %}

Distributions and weak derivatives records the objects, main results, proof mechanisms, and boundary cases needed for the functional analysis note.

Definition

Definition. A distribution is a continuous linear functional on a space of test functions. Weak derivatives are defined by moving derivatives onto test functions with a sign change.

Proposition

Proposition. Locally integrable functions define distributions, and every distribution has weak derivatives of all orders. If a classical derivative exists, it agrees with the weak derivative.

Proof

Proof. Define \(\partial^\alpha T\) by \(\langle \partial^\alpha T,\varphi\rangle=(-1)^{|\alpha|}\langle T,\partial^\alpha\varphi\rangle\). Integration by parts verifies agreement for smooth functions.

Remark

Remark. Distribution theory lets discontinuous and singular objects satisfy differential equations in a precise linear-functional sense.
