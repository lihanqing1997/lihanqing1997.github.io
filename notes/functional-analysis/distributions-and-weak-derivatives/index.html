23. Distributions and weak derivatives

{% include note-topic-nav.html back_label="Back" back_url="/notes/functional-analysis/" prev_label="Last page" prev_url="/notes/functional-analysis/locally-convex-spaces-and-seminorms/" next_label="Next page" next_url="/notes/functional-analysis/sobolev-spaces/" %}

Test functions and distributions

Definition. A distribution on an open set \(U\subset\mathbb R^d\) is a continuous linear functional on \(C_c^\infty(U)\). We write its action as \(\langle T,\varphi\rangle\).

Locally integrable functions

Example. Every \(f\in L^1_{\mathrm{loc}}(U)\) defines a distribution by \(\langle T_f,\varphi\rangle=\int_U f(x)\varphi(x)\,\mathrm{d}x\). Thus distributions extend functions.

Weak derivatives

Definition. The distributional derivative \(\partial^\alpha T\) is defined by \(\langle \partial^\alpha T,\varphi\rangle=(-1)^{|\alpha|}\langle T,\partial^\alpha\varphi\rangle\). This moves derivatives from rough objects onto smooth test functions.

Integration by parts origin

Proof. For smooth \(f\), integration by parts gives \(\int \partial^\alpha f\,\varphi\,\mathrm{d}x=(-1)^{|\alpha|}\int f\,\partial^\alpha\varphi\,\mathrm{d}x\), with no boundary term because \(\varphi\) has compact support. The weak definition preserves this identity.

Dirac delta

Example. The Dirac distribution \(\delta_a\) is \(\langle\delta_a,\varphi\rangle=\varphi(a)\). Its derivative satisfies \(\langle\partial_j\delta_a,\varphi\rangle=-\partial_j\varphi(a)\).

Weak solutions

Remark. Distributions allow differential equations to be interpreted when classical derivatives do not exist. Sobolev spaces refine this by asking weak derivatives to be functions in \(L^p\).
