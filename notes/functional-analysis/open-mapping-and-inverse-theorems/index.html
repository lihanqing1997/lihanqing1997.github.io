6. Open mapping and inverse theorems

{% include note-topic-nav.html back_label="Back" back_url="/notes/functional-analysis/" prev_label="Last page" prev_url="/notes/functional-analysis/uniform-boundedness-principle/" next_label="Next page" next_url="/notes/functional-analysis/closed-graph-theorem/" %}

Open mapping and inverse theorems records the objects, main results, proof mechanisms, and boundary cases needed for the functional analysis note.

Definition

Definition. An open map sends open sets to open sets. In Banach spaces, surjective bounded linear maps are automatically open.

Open mapping theorem

Theorem. If \(T:X\to Y\) is a surjective bounded linear map between Banach spaces, then \(T\) is open. If \(T\) is bijective, then \(T^{-1}\) is bounded.

Proof

Proof. Use Baire category to show the image of the unit ball contains a ball around zero after scaling and closure. Linearity then transfers this local openness to every open set.

Remark

Remark. The result says algebraic invertibility plus Banach completeness implies topological invertibility. This is a cornerstone of operator equations.
