20. Strong converses and error exponents

{% include note-topic-nav.html back_label="Back" back_url="/notes/information-theory/" prev_label="Last page" prev_url="/notes/information-theory/channel-coding-converse/" next_label="Next page" next_url="/notes/information-theory/continuous-alphabets-differential-entropy/" %}

First-order coding theorems separate possible from impossible rates. Strong converses and error exponents refine that boundary.

Strong converse

Definition. A channel has a strong converse if every code sequence with rate strictly above capacity has error probability tending to one, not merely bounded away from zero.

DMC strong converse

Theorem. Finite-alphabet discrete memoryless channels satisfy the strong converse: if \(\liminf n^{-1}\log M_n>C\), then the average probability of decoding error tends to one.

Error exponent

Definition. For rates below capacity, the reliability function is the best achievable exponential decay rate \[ E(R)=\sup\liminf_{n\to\infty}-\frac1n\log P_e^{(n)}. \] Random-coding and sphere-packing bounds give lower and upper bounds on \(E(R)\).

Change-of-measure idea

Proof. Strong converses compare the true channel output law with auxiliary output laws under which too many messages cannot be separated. Error exponents sharpen the same comparison by retaining exponential constants instead of only first-order rates.
