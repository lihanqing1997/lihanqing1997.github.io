The direct part of Shannon’s channel theorem shows that random codebooks achieve every rate below capacity. The proof is a packing theorem for typical sets.

Channel code

Definition. An \((n,M)\) channel code chooses codewords \(x^n(m)\), sends one through \(W^n\), and decodes the received \(Y^n\) to \(\widehat M\). The average error is \(P_e=P(\widehat M\ne M)\) for a uniform message.

Remark. The theorem is a high-dimensional packing statement: output typical sets generated by different messages can be made almost disjoint as long as their exponential count is below the mutual-information exponent.
