15. Universal source coding and minimax redundancy

{% include note-topic-nav.html back_label="Back" back_url="/notes/information-theory/" prev_label="Last page" prev_url="/notes/information-theory/redundancy-mismatched-coding/" next_label="Next page" next_url="/notes/information-theory/kolmogorov-complexity-algorithmic-information/" %}

Universal coding asks for one code that performs well for many possible sources. The mathematical object is regret or redundancy relative to the best code that knew the source.

Pointwise regret

Definition. For a model class \(\mathcal P\) on \(\mathcal X^n\) and coding distribution \(Q_n\), the pointwise regret is \[ \sup_{x^n}\left\{-\log Q_n(x^n)-\inf_{P\in\mathcal P}[-\log P(x^n)]\right\}. \]

Normalized maximum likelihood

Proposition. If \[ S_n=\sum_{x^n}\sup_{P\in\mathcal P}P(x^n)<\infty, \] then the normalized maximum likelihood law \[ Q_n^\star(x^n)=\frac{\sup_{P\in\mathcal P}P(x^n)}{S_n} \] has minimax pointwise regret \(\log S_n\).

Proof by domination

Proof. For any \(Q_n\), the worst-case regret is at least \(r\) only if \(Q_n(x^n)\ge e^{-r}\sup_P P(x^n)\) for all \(x^n\). Summing over \(x^n\) gives \(1\ge e^{-r}S_n\), hence \(r\ge\log S_n\). The normalized maximum likelihood distribution attains equality.

Asymptotic parametric scale

Remark. For regular \(d\)-dimensional parametric families, minimax redundancy often grows like \((d/2)\log n\) plus lower-order constants. The main lesson is not the constant but the statistical dimension penalty.
