16. Kolmogorov complexity and algorithmic information

{% include note-topic-nav.html back_label="Back" back_url="/notes/information-theory/" prev_label="Last page" prev_url="/notes/information-theory/universal-source-coding-minimax-redundancy/" next_label="Next page" next_url="/notes/information-theory/channels-stochastic-kernels-capacity/" %}

Entropy describes average description length under a distribution. Kolmogorov complexity asks for the shortest effective description of an individual string.

Prefix complexity

Definition. Fix a universal prefix Turing machine \(U\). The prefix Kolmogorov complexity of a binary string \(x\) is \[ K(x)=\min\{|p|:U(p)=x\}. \] Changing the universal machine changes \(K\) by at most an additive constant.

Incompressibility counting

Proposition. Among binary strings of length \(n\), at most \(2^{n-k}-1\) strings satisfy \(K(x)

Proof by counting programs

Proof. There are fewer than \(2^{n-k}\) binary programs of length less than \(n-k\). Each such program outputs at most one string, so fewer than \(2^{n-k}\) strings can have a description that short.

Relation to entropy

Remark. For computable sources, expected prefix complexity is bounded above by entropy plus a source-dependent constant. Entropy is therefore the distributional version of shortest description, while \(K(x)\) is an individual-sequence version.
