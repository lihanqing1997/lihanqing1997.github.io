The source coding theorem gives entropy its first operational identity: entropy is the sharp asymptotic rate for lossless representation.

Fixed-length compression

Definition. An \((n,M_n,\varepsilon_n)\) lossless source code maps \(X^n\) to one of \(M_n\) messages and decodes with error probability at most \(\varepsilon_n\). Its rate is \(R_n=n^{-1}\log M_n\).

Proof. Encode typical strings injectively and send a special error symbol for atypical strings. The AEP gives a typical set of size at most \(\exp(n(H+\eta))\) and probability tending to one. Thus any \(R>H\) eventually leaves enough codewords.
