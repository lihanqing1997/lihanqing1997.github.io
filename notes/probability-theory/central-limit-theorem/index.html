Central Limit Theorem

The central limit theorem gives the universal first approximation to fluctuations of sums.

Statement

  • Let X1,X2,... be i.i.d. with E X1 = μ and 0 < Var(X1) = σ2 < ∞.
  • With Sn = X1 + ... + Xn, (Sn - nμ) / (σ√n) converges in distribution to N(0,1).

How to read it

  • LLN gives the center: Sn is near nμ.
  • CLT gives the scale and shape: fluctuations are of order √n and approximately normal.
  • For continuity points x, P((Sn - nμ) / (σ√n) ≤ x) → Φ(x).

Durrett guide: Chapter 3, especially Section 3.4.

Théorème central limite

Le théorème central limite donne la première approximation universelle des fluctuations de sommes.

Énoncé

  • Soient X1,X2,... i.i.d. avec E X1 = μ et 0 < Var(X1) = σ2 < ∞.
  • Avec Sn = X1 + ... + Xn, (Sn - nμ) / (σ√n) converge en loi vers N(0,1).

Lecture

  • La loi des grands nombres donne le centre : Sn est proche de nμ.
  • Le TCL donne l'échelle et la forme : les fluctuations sont d'ordre √n et approximativement normales.
  • Aux points de continuité x, P((Sn - nμ) / (σ√n) ≤ x) → Φ(x).

Guide Durrett : chapitre 3, surtout section 3.4.

中心极限定理

中心极限定理给出和的波动的最基本、最普遍的近似。

定理表述

  • 设 X1,X2,... 独立同分布，E X1 = μ，且 0 < Var(X1) = σ2 < ∞。
  • 令 Sn = X1 + ... + Xn，则 (Sn - nμ) / (σ√n) 依分布收敛到 N(0,1)。

如何理解

  • 大数定律给出中心位置：Sn 接近 nμ。
  • 中心极限定理给出尺度和形状：波动量级为 √n，形状近似正态。
  • 对连续点 x，P((Sn - nμ) / (σ√n) ≤ x) → Φ(x)。

Durrett 参考：第 3 章，尤其第 3.4 节。
