2. Measures, extension, and monotone classes

{% include probability-note-nav.html back_label="Back" prev_label="Last page" prev_url="/notes/probability-theory/probability-spaces/" next_label="Next page" next_url="/notes/probability-theory/distributions-random-variables/" %}

Definition. A \(\pi\)-system \(\mathcal P\) is a nonempty family of sets closed under finite intersections. A \(\lambda\)-system \(\mathcal L\) contains \(\Omega\), is closed under complements inside \(\Omega\), and is closed under countable disjoint unions. The notation \(\sigma(\mathcal P)\) means the smallest \(\sigma\)-field containing \(\mathcal P\).

Theorem. If \(\mathcal P\) is a \(\pi\)-system and \(\mathcal L\) is a \(\lambda\)-system with \(\mathcal P\subset\mathcal L\), then \(\sigma(\mathcal P)\subset\mathcal L\). Hence two probability measures that agree on \(\mathcal P\) agree on \(\sigma(\mathcal P)\).

Proof. Let \(\mathcal D\) be the smallest \(\lambda\)-system containing \(\mathcal P\). Fix \(A\in\mathcal P\) and define \(\mathcal D_A=\{B\in\mathcal D:A\cap B\in\mathcal D\}\). This is a \(\lambda\)-system and contains \(\mathcal P\), so it contains \(\mathcal D\). Thus \(A\cap B\in\mathcal D\) for all \(A\in\mathcal P\), \(B\in\mathcal D\). Now fix \(B\in\mathcal D\) and repeat the argument with \(\mathcal D_B=\{A\in\mathcal D:A\cap B\in\mathcal D\}\). It follows that \(\mathcal D\) is closed under finite intersections. If \(A_1,A_2,\ldots\in\mathcal D\), then \(\bigcup_n A_n\) is the disjoint union of \(A_1\), \(A_2\setminus A_1\), \(A_3\setminus(A_1\cup A_2)\), and so on, so \(\mathcal D\) is a \(\sigma\)-field. The uniqueness statement follows by taking \(\mathcal L=\{A:\mu(A)=\nu(A)\}\).

Extending identities from generators

This theorem is the usual way to extend an identity from intervals, rectangles, or cylinder sets to the full generated \(\sigma\)-field.

2. Mesures, extension et classes monotones

{% include probability-note-nav.html back_label="Retour" prev_label="Page précédente" prev_url="/notes/probability-theory/probability-spaces/" next_label="Page suivante" next_url="/notes/probability-theory/distributions-random-variables/" %}

Définition. Un \(\pi\)-système \(\mathcal P\) est une famille non vide stable par intersections finies. Un \(\lambda\)-système \(\mathcal L\) contient \(\Omega\), est stable par complémentaire dans \(\Omega\), et est stable par unions dénombrables disjointes. La notation \(\sigma(\mathcal P)\) désigne la plus petite \(\sigma\)-algèbre contenant \(\mathcal P\).

Théorème. Si \(\mathcal P\) est un \(\pi\)-système et si \(\mathcal L\) est un \(\lambda\)-système tel que \(\mathcal P\subset\mathcal L\), alors \(\sigma(\mathcal P)\subset\mathcal L\). Ainsi deux probabilités qui coïncident sur \(\mathcal P\) coïncident sur \(\sigma(\mathcal P)\).

Preuve. Soit \(\mathcal D\) le plus petit \(\lambda\)-système contenant \(\mathcal P\). Fixons \(A\in\mathcal P\) et définissons \(\mathcal D_A=\{B\in\mathcal D:A\cap B\in\mathcal D\}\). C’est un \(\lambda\)-système qui contient \(\mathcal P\), donc il contient \(\mathcal D\). Ainsi \(A\cap B\in\mathcal D\) pour tout \(A\in\mathcal P\) et tout \(B\in\mathcal D\). Fixons ensuite \(B\in\mathcal D\) et répétons l’argument avec \(\mathcal D_B=\{A\in\mathcal D:A\cap B\in\mathcal D\}\). On obtient que \(\mathcal D\) est stable par intersections finies. Si \(A_1,A_2,\ldots\in\mathcal D\), alors \(\bigcup_n A_n\) est l’union disjointe de \(A_1\), \(A_2\setminus A_1\), \(A_3\setminus(A_1\cup A_2)\), etc. Donc \(\mathcal D\) est une \(\sigma\)-algèbre. L’unicité suit en prenant \(\mathcal L=\{A:\mu(A)=\nu(A)\}\).

Extension des identités depuis les générateurs

Ce théorème sert à prolonger une identité depuis les intervalles, rectangles ou cylindres vers toute la \(\sigma\)-algèbre engendrée.

2. 测度、扩张与单调类

{% include probability-note-nav.html back_label="返回" prev_label="上一页" prev_url="/notes/probability-theory/probability-spaces/" next_label="下一页" next_url="/notes/probability-theory/distributions-random-variables/" %}

定义. \(\pi\)-系统 \(\mathcal P\) 是非空集合族，并且对有限交封闭。\(\lambda\)-系统 \(\mathcal L\) 包含 \(\Omega\)，对 \(\Omega\) 中的补集封闭，并且对可列不交并封闭。记号 \(\sigma(\mathcal P)\) 表示包含 \(\mathcal P\) 的最小 \(\sigma\)-代数。

定理. 若 \(\mathcal P\) 是 \(\pi\)-系统，\(\mathcal L\) 是 \(\lambda\)-系统，且 \(\mathcal P\subset\mathcal L\)，则 \(\sigma(\mathcal P)\subset\mathcal L\)。因此，如果两个概率测度在 \(\mathcal P\) 上相同，那么它们在 \(\sigma(\mathcal P)\) 上相同。

证明. 令 \(\mathcal D\) 为包含 \(\mathcal P\) 的最小 \(\lambda\)-系统。固定 \(A\in\mathcal P\)，定义 \(\mathcal D_A=\{B\in\mathcal D:A\cap B\in\mathcal D\}\)。它是一个 \(\lambda\)-系统，并且包含 \(\mathcal P\)，所以包含 \(\mathcal D\)。因此对所有 \(A\in\mathcal P\)、\(B\in\mathcal D\)，都有 \(A\cap B\in\mathcal D\)。再固定 \(B\in\mathcal D\)，对 \(\mathcal D_B=\{A\in\mathcal D:A\cap B\in\mathcal D\}\) 重复同样论证，得到 \(\mathcal D\) 对有限交封闭。若 \(A_1,A_2,\ldots\in\mathcal D\)，则 \(\bigcup_n A_n\) 是 \(A_1\)、\(A_2\setminus A_1\)、\(A_3\setminus(A_1\cup A_2)\) 等集合的不交并，因此 \(\mathcal D\) 是 \(\sigma\)-代数。测度唯一性取 \(\mathcal L=\{A:\mu(A)=\nu(A)\}\) 即可。

从生成类扩张恒等式

这个定理用于把区间、矩形或柱集上的恒等式推广到它们生成的整个 \(\sigma\)-代数。
