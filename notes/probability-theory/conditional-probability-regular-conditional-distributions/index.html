Conditional probability and regular conditional distributions

{% include probability-note-nav.html back_label="Back" prev_label="Last page" prev_url="/notes/probability-theory/pushforward-measures-densities/" next_label="Next page" next_url="/notes/probability-theory/conditional-expectation/" %}

Definition. For an event \(A\) and a sub-\(\sigma\)-field \(\mathcal G\), the conditional probability \(P(A\mid\mathcal G)\) is \(\mathbb E[1_A\mid\mathcal G]\). A regular conditional distribution of \(X\) given \(\mathcal G\) is a kernel \(K(\omega,B)\) such that \(K(\cdot,B)\) is \(\mathcal G\)-measurable, \(K(\omega,\cdot)\) is a probability measure, and \(K(\omega,B)=P(X\in B\mid\mathcal G)(\omega)\) almost surely.

Theorem. If \(S\) is a standard Borel space, then regular conditional distributions of \(X:\Omega\to S\) given \(\mathcal G\) exist. They are unique up to \(P\)-null sets in the sense that two versions agree on all \(B\) from a countable generator, hence on all Borel \(B\).

Proof. The existence part is a standard measurable-disintegration theorem for standard Borel spaces. For uniqueness, let \(K\) and \(L\) be two versions. Choose a countable \(\pi\)-system \(\mathcal P\) generating \(\mathcal B(S)\). For each \(B\in\mathcal P\), the equality \(K(\cdot,B)=L(\cdot,B)\) holds almost surely because both are versions of \(P(X\in B\mid\mathcal G)\). Intersecting the countably many full-probability sets gives one full-probability set on which the two probability measures agree on \(\mathcal P\). The \(\pi\)-\(\lambda\) theorem then gives equality on all Borel sets.

Kernels as conditional laws

Regular conditional distributions let one write \(P(X\in B\mid Y=y)\) as an honest probability kernel rather than only an equivalence class of conditional expectations.

Probabilité conditionnelle et lois conditionnelles régulières

{% include probability-note-nav.html back_label="Retour" prev_label="Page précédente" prev_url="/notes/probability-theory/pushforward-measures-densities/" next_label="Page suivante" next_url="/notes/probability-theory/conditional-expectation/" %}

Définition. Pour un événement \(A\) et une sous-\(\sigma\)-algèbre \(\mathcal G\), la probabilité conditionnelle \(P(A\mid\mathcal G)\) est \(\mathbb E[1_A\mid\mathcal G]\). Une loi conditionnelle régulière de \(X\) sachant \(\mathcal G\) est un noyau \(K(\omega,B)\) tel que \(K(\cdot,B)\) soit \(\mathcal G\)-mesurable, \(K(\omega,\cdot)\) soit une probabilité, et \(K(\omega,B)=P(X\in B\mid\mathcal G)(\omega)\) presque sûrement.

Théorème. Si \(S\) est un espace borélien standard, alors il existe des lois conditionnelles régulières de \(X:\Omega\to S\) sachant \(\mathcal G\). Elles sont uniques à ensembles nuls près : deux versions coïncident sur une famille génératrice dénombrable, donc sur tous les boréliens.

Preuve. L’existence est le théorème classique de désintégration mesurable pour les espaces boréliens standards. Pour l’unicité, soient \(K\) et \(L\) deux versions. Prenons un \(\pi\)-système dénombrable \(\mathcal P\) qui engendre \(\mathcal B(S)\). Pour chaque \(B\in\mathcal P\), l’égalité \(K(\cdot,B)=L(\cdot,B)\) vaut presque sûrement, car les deux côtés sont des versions de \(P(X\in B\mid\mathcal G)\). L’intersection dénombrable de ces ensembles de probabilité un donne un ensemble de probabilité un sur lequel les deux probabilités coïncident sur \(\mathcal P\). Le théorème \(\pi\)-\(\lambda\) donne alors l’égalité sur tous les boréliens.

Noyaux comme lois conditionnelles

Les lois conditionnelles régulières permettent d’écrire \(P(X\in B\mid Y=y)\) comme un noyau de probabilité véritable, et pas seulement comme une classe de versions d’espérance conditionnelle.

条件概率与正则条件分布

{% include probability-note-nav.html back_label="返回" prev_label="上一页" prev_url="/notes/probability-theory/pushforward-measures-densities/" next_label="下一页" next_url="/notes/probability-theory/conditional-expectation/" %}

定义. 对事件 \(A\) 和子 \(\sigma\)-代数 \(\mathcal G\)，条件概率定义为 \(P(A\mid\mathcal G)=\mathbb E[1_A\mid\mathcal G]\)。给定 \(\mathcal G\) 的 \(X\) 的正则条件分布，是核 \(K(\omega,B)\)，满足 \(K(\cdot,B)\) 为 \(\mathcal G\)-可测，\(K(\omega,\cdot)\) 是概率测度，并且 \(K(\omega,B)=P(X\in B\mid\mathcal G)(\omega)\) 几乎处处成立。

定理. 若 \(S\) 是标准 Borel 空间，则 \(X:\Omega\to S\) 关于 \(\mathcal G\) 的正则条件分布存在。它在零测集意义下唯一：两个版本若在一个可列生成族上相同，则在所有 Borel 集上相同。

证明. 存在性是标准 Borel 空间上的可测分解定理。下面说明唯一性。设 \(K,L\) 是两个版本。取生成 \(\mathcal B(S)\) 的可列 \(\pi\)-系统 \(\mathcal P\)。对每个 \(B\in\mathcal P\)，由于 \(K(\cdot,B)\) 和 \(L(\cdot,B)\) 都是 \(P(X\in B\mid\mathcal G)\) 的版本，二者几乎处处相等。把这些概率一的集合做可列交，得到一个概率一的集合，在其上 \(K(\omega,\cdot)\) 与 \(L(\omega,\cdot)\) 在 \(\mathcal P\) 上相同。由 \(\pi\)-\(\lambda\) 定理，它们在所有 Borel 集上相同。

作为条件分布的核

正则条件分布让 \(P(X\in B\mid Y=y)\) 成为真正的概率核，而不只是条件期望的某个版本。
