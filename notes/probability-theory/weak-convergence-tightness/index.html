Weak convergence and tightness

{% include probability-note-nav.html back_label="Back" prev_label="Last page" prev_url="/notes/probability-theory/modes-of-convergence/" next_label="Next page" next_url="/notes/probability-theory/characteristic-functions/" %}

Definition. Probability measures \(\mu_n\) on a metric space \(S\) converge weakly to \(\mu\), written \(\mu_n\Rightarrow\mu\), if \[ \int f\,\mathrm{d}\mu_n\to\int f\,\mathrm{d}\mu \quad\text{for every bounded continuous }f:S\to\mathbb R. \] A family \(\mathcal M\) is tight if for every \(\varepsilon\gt 0\) there is compact \(K\subset S\) such that \(\sup_{\mu\in\mathcal M}\mu(K^c)\lt \varepsilon\).

Theorem. On \(\mathbb R^d\), a sequence of probability measures is relatively compact for weak convergence if and only if it is tight. This is Prokhorov’s theorem in Euclidean spaces.

Proof. If \(\mu_{n_k}\Rightarrow\mu\), then choose compact \(K\) with \(\mu(K)\gt 1-\varepsilon\). By the portmanteau theorem, \(\limsup_k\mu_{n_k}(K^c)\le\mu(K^c)\lt \varepsilon\) after slightly enlarging \(K\). This gives tightness for convergent subsequences. Conversely, tightness puts almost all mass in a compact cube. On each compact cube, bounded Lipschitz test functions have a countable dense determining class. A diagonal argument extracts a subsequence for which all integrals in this class converge. The limiting functional defines a probability measure, and the determining class gives weak convergence.

Remark

Tightness is the mathematical way of saying that no mass escapes to infinity.

Convergence faible et tension

{% include probability-note-nav.html back_label="Retour" prev_label="Page précédente" prev_url="/notes/probability-theory/modes-of-convergence/" next_label="Page suivante" next_url="/notes/probability-theory/characteristic-functions/" %}

Définition. Des probabilités \(\mu_n\) sur un espace métrique \(S\) convergent faiblement vers \(\mu\), noté \(\mu_n\Rightarrow\mu\), si \[ \int f\,\mathrm{d}\mu_n\to\int f\,\mathrm{d}\mu \quad\text{pour toute fonction continue bornée }f:S\to\mathbb R. \] Une famille \(\mathcal M\) est tendue si, pour tout \(\varepsilon\gt 0\), il existe un compact \(K\subset S\) tel que \(\sup_{\mu\in\mathcal M}\mu(K^c)\lt \varepsilon\).

Théorème. Sur \(\mathbb R^d\), une suite de probabilités est relativement compacte pour la convergence faible si et seulement si elle est tendue. C’est le théorème de Prokhorov dans le cas euclidien.

Preuve. Si \(\mu_{n_k}\Rightarrow\mu\), on choisit un compact \(K\) avec \(\mu(K)\gt 1-\varepsilon\). Par le théorème de portmanteau, \(\limsup_k\mu_{n_k}(K^c)\le\mu(K^c)\lt \varepsilon\) après un léger élargissement de \(K\). Cela donne la tension pour les sous-suites convergentes. Réciproquement, la tension concentre presque toute la masse dans un cube compact. Sur ce compact, les fonctions lipschitziennes bornées possèdent une classe déterminante dénombrable dense. Un argument diagonal extrait une sous-suite pour laquelle toutes les intégrales de cette classe convergent. La fonctionnelle limite définit une probabilité, et la classe déterminante donne la convergence faible.

Remarque

La tension signifie précisément que la masse ne s’échappe pas vers l’infini.

弱收敛与紧性

{% include probability-note-nav.html back_label="返回" prev_label="上一页" prev_url="/notes/probability-theory/modes-of-convergence/" next_label="下一页" next_url="/notes/probability-theory/characteristic-functions/" %}

定义. 度量空间 \(S\) 上的概率测度 \(\mu_n\) 弱收敛到 \(\mu\)，记作 \(\mu_n\Rightarrow\mu\)，若 \[ \int f\,\mathrm{d}\mu_n\to\int f\,\mathrm{d}\mu \quad\text{对每个有界连续 }f:S\to\mathbb R. \] 测度族 \(\mathcal M\) 称为紧的，若对每个 \(\varepsilon\gt 0\)，存在紧集 \(K\subset S\)，使得 \(\sup_{\mu\in\mathcal M}\mu(K^c)\lt \varepsilon\)。

定理. 在 \(\mathbb R^d\) 上，概率测度序列关于弱收敛相对紧，当且仅当它是紧的。这是欧氏空间中的 Prokhorov 定理。

证明. 若 \(\mu_{n_k}\Rightarrow\mu\)，取紧集 \(K\) 使 \(\mu(K)\gt 1-\varepsilon\)。由 portmanteau 定理，适当扩大 \(K\) 后有 \(\limsup_k\mu_{n_k}(K^c)\le\mu(K^c)\lt \varepsilon\)。这说明收敛子列不会把质量送到无穷远。反过来，紧性保证大部分质量落在某个紧立方体中。在该紧集上，有界 Lipschitz 测试函数有可列稠密的决定类。用对角线法取子列，使该可列类中每个函数的积分都收敛。极限泛函定义一个概率测度，而决定类保证弱收敛。

说明

紧性的直觉含义是：概率质量不会逃到无穷远。
