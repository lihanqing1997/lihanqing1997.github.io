Optional Stopping and Optional Sampling

{% include probability-note-nav.html back_label="Back" prev_label="Last page" prev_url="/notes/probability-theory/doob-inequalities-martingale-convergence/" next_label="Next page" next_url="/notes/probability-theory/uniform-integrability-martingale-convergence/" %}

This page records the core mathematical facts for Optional Stopping and Optional Sampling.

Definition. Optional stopping asks when a martingale remains fair after observing it at random stopping times. The statement is true under boundedness or suitable integrability hypotheses, and false without them.

Central result. If \(M_n\) is a martingale and \(\sigma\le\tau\) are bounded stopping times, then \(\mathbb E(M_\tau\mid\mathcal F_\sigma)=M_\sigma\). In particular, \(\mathbb EM_\tau=\mathbb EM_0\).

Proof. In discrete time, write \(M_\tau-M_\sigma=\sum_{k=1}^N(M_k-M_{k-1})1_{\{\sigma\lt k\le\tau\}}\). The multiplier is \(\mathcal F_{k-1}\)-measurable, so each summand has expectation zero conditionally on \(\mathcal F_{k-1}\). Summing proves the identity.

How to read it

Most paradoxes about gambling systems come from violating the hypotheses: unbounded stopping times or non-integrable terminal values can create apparent unfairness.

The point is not to memorize the displayed formulas separately, but to understand how the definition, the result, and the proof mechanism fit together.
