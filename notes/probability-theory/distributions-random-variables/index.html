3. Random variables and distributions

{% include probability-note-nav.html back_label="Back" prev_label="Last page" prev_url="/notes/probability-theory/measures-extension-monotone-classes/" next_label="Next page" next_url="/notes/probability-theory/independence/" %}

Definition. Let \((\Omega,\mathcal F,P)\) and \((S,\mathcal S)\) be measurable spaces. A map \(X:\Omega\to S\) is a random variable if \[ X^{-1}(B)=\{\omega\in\Omega:X(\omega)\in B\}\in\mathcal F \quad\text{for every }B\in\mathcal S. \] Its law is the probability measure \(P_X\) on \((S,\mathcal S)\) defined by \(P_X(B)=P(X^{-1}(B))\).

Proposition. For a real random variable \(X\), the distribution function \(F_X(x)=P(X\le x)\) is increasing, right-continuous, and satisfies \(\lim_{x\to-\infty}F_X(x)=0\), \(\lim_{x\to\infty}F_X(x)=1\). Conversely, every function with these four properties determines a unique probability measure on \((\mathbb R,\mathcal B(\mathbb R))\).

Proof. If \(x\le y\), then \(\{X\le x\}\subset\{X\le y\}\), so \(F_X(x)\le F_X(y)\). If \(x_n\downarrow x\), then \(\{X\le x_n\}\downarrow\{X\le x\}\), and continuity from above gives \(F_X(x_n)\downarrow F_X(x)\). The two endpoint limits follow from \(\{X\le n\}\uparrow\Omega\) and \(\{X\le -n\}\downarrow\varnothing\). Conversely, define \(\mu((a,b])=F(b)-F(a)\) on half-open intervals, extend the resulting premeasure by Carathéodory’s theorem, and use the \(\pi\)-\(\lambda\) theorem for uniqueness.

Distributions as pushforwards

A random variable is a measurable map; its distribution is the pushforward of \(P\) by that map.

3. Variables aléatoires et lois

{% include probability-note-nav.html back_label="Retour" prev_label="Page précédente" prev_url="/notes/probability-theory/measures-extension-monotone-classes/" next_label="Page suivante" next_url="/notes/probability-theory/independence/" %}

Définition. Soient \((\Omega,\mathcal F,P)\) et \((S,\mathcal S)\) des espaces mesurables. Une application \(X:\Omega\to S\) est une variable aléatoire si \[ X^{-1}(B)=\{\omega\in\Omega:X(\omega)\in B\}\in\mathcal F \quad\text{pour tout }B\in\mathcal S. \] Sa loi est la probabilité \(P_X\) sur \((S,\mathcal S)\) définie par \(P_X(B)=P(X^{-1}(B))\).

Proposition. Pour une variable aléatoire réelle \(X\), la fonction de répartition \(F_X(x)=P(X\le x)\) est croissante, continue à droite, et vérifie \(\lim_{x\to-\infty}F_X(x)=0\), \(\lim_{x\to\infty}F_X(x)=1\). Réciproquement, toute fonction ayant ces quatre propriétés détermine une unique probabilité sur \((\mathbb R,\mathcal B(\mathbb R))\).

Preuve. Si \(x\le y\), alors \(\{X\le x\}\subset\{X\le y\}\), donc \(F_X(x)\le F_X(y)\). Si \(x_n\downarrow x\), alors \(\{X\le x_n\}\downarrow\{X\le x\}\), et la continuité par le haut donne \(F_X(x_n)\downarrow F_X(x)\). Les limites aux deux extrémités découlent de \(\{X\le n\}\uparrow\Omega\) et \(\{X\le -n\}\downarrow\varnothing\). Pour la réciproque, on définit \(\mu((a,b])=F(b)-F(a)\) sur les intervalles semi-ouverts, on étend la prémesure par le théorème de Carathéodory, puis on utilise le théorème \(\pi\)-\(\lambda\) pour l’unicité.

Lois comme mesures images

Une variable aléatoire est une application mesurable ; sa loi est l’image directe de \(P\) par cette application.

3. 随机变量与分布

{% include probability-note-nav.html back_label="返回" prev_label="上一页" prev_url="/notes/probability-theory/measures-extension-monotone-classes/" next_label="下一页" next_url="/notes/probability-theory/independence/" %}

定义. 设 \((\Omega,\mathcal F,P)\) 与 \((S,\mathcal S)\) 为可测空间。映射 \(X:\Omega\to S\) 称为随机变量，若 \[ X^{-1}(B)=\{\omega\in\Omega:X(\omega)\in B\}\in\mathcal F \quad\text{对每个 }B\in\mathcal S. \] 它的分布是在 \((S,\mathcal S)\) 上的概率测度 \(P_X\)，定义为 \(P_X(B)=P(X^{-1}(B))\)。

命题. 对实值随机变量 \(X\)，分布函数 \(F_X(x)=P(X\le x)\) 单调递增、右连续，并满足 \(\lim_{x\to-\infty}F_X(x)=0\)、\(\lim_{x\to\infty}F_X(x)=1\)。反过来，任何满足这四个性质的函数都唯一决定 \((\mathbb R,\mathcal B(\mathbb R))\) 上的一个概率测度。

证明. 若 \(x\le y\)，则 \(\{X\le x\}\subset\{X\le y\}\)，所以 \(F_X(x)\le F_X(y)\)。若 \(x_n\downarrow x\)，则 \(\{X\le x_n\}\downarrow\{X\le x\}\)，由概率的上连续性得到 \(F_X(x_n)\downarrow F_X(x)\)。两个端点极限分别由 \(\{X\le n\}\uparrow\Omega\) 和 \(\{X\le -n\}\downarrow\varnothing\) 得到。反向构造中，在半开区间上定义 \(\mu((a,b])=F(b)-F(a)\)，再用卡拉泰奥多里扩张定理扩张该预测度，最后用 \(\pi\)-\(\lambda\) 定理证明唯一性。

作为推前测度的分布

随机变量是可测映射；它的分布就是概率测度 \(P\) 在该映射下的推前测度。
