Distributions & Random Variables

{% include probability-note-nav.html back_label="Back" prev_label="Last page" prev_url="/notes/probability-theory/probability-spaces/" next_label="Next page" next_url="/notes/probability-theory/pushforward-measures-densities/" %}

A random variable is not just a function; it must be measurable so that its probabilistic questions are events.

Definition

A real-valued random variable on \((\Omega,\mathcal F,P)\) is a measurable map \(X:(\Omega,\mathcal F)\to(\mathbb R,\mathcal B(\mathbb R))\), meaning \(\{X\in B\}\in\mathcal F\) for every Borel set \(B\).

The distribution of \(X\) is the measure \(\mu_X\) on \(\mathbb R\) defined by \(\mu_X(B)=P(X\in B)\). Its distribution function is \(F_X(x)=P(X\le x)\).

Proposition

\(\mu_X\) is a probability measure. Also, \(F_X\) is nondecreasing, right-continuous, and satisfies \(\lim_{x\to-\infty}F_X(x)=0\) and \(\lim_{x\to\infty}F_X(x)=1\).

Proof

Since \(X^{-1}(\mathbb R)=\Omega\), \(\mu_X(\mathbb R)=1\). If \(B_i\) are disjoint Borel sets, then \(\{X\in B_i\}\) are disjoint events and \(\{X\in\bigcup_iB_i\}=\bigcup_i\{X\in B_i\}\). Thus \(\mu_X(\bigcup_iB_i)=\sum_i\mu_X(B_i)\).

If \(x\le y\), then \(\{X\le x\}\subset\{X\le y\}\), so \(F_X(x)\le F_X(y)\).

For \(x_n\downarrow x\), the events \(\{X\le x_n\}\downarrow\{X\le x\}\), so continuity from above gives \(F_X(x_n)\downarrow F_X(x)\). This is right-continuity.

Finally, \(\{X\le n\}\uparrow\Omega\) and \(\{X\le -n\}\downarrow\varnothing\), so continuity of probability gives the two limits.

Two common forms

\(X\) is discrete if \(\mu_X(C)=1\) for some countable \(C\subset\mathbb R\). It has density \(f\) if \(f\ge0\), \(\int_{\mathbb R}f(x)\,dx=1\), and \(P(X\in B)=\int_B f(x)\,dx\) for all Borel \(B\).

Lois et variables aléatoires

{% include probability-note-nav.html back_label="Retour" prev_label="Page précédente" prev_url="/notes/probability-theory/probability-spaces/" next_label="Page suivante" next_url="/notes/probability-theory/pushforward-measures-densities/" %}

Une variable aléatoire n'est pas seulement une fonction : elle doit être mesurable pour que ses questions probabilistes soient des événements.

Définition

Une variable aléatoire réelle sur \((\Omega,\mathcal F,P)\) est une application mesurable \(X:(\Omega,\mathcal F)\to(\mathbb R,\mathcal B(\mathbb R))\), c'est-à-dire \(\{X\in B\}\in\mathcal F\) pour tout borélien \(B\).

La loi de \(X\) est la mesure \(\mu_X\) définie par \(\mu_X(B)=P(X\in B)\). Sa fonction de répartition est \(F_X(x)=P(X\le x)\).

Proposition

\(\mu_X\) est une mesure de probabilité. De plus, \(F_X\) est croissante, continue à droite, et vérifie \(\lim_{x\to-\infty}F_X(x)=0\), \(\lim_{x\to\infty}F_X(x)=1\).

Preuve

Comme \(X^{-1}(\mathbb R)=\Omega\), \(\mu_X(\mathbb R)=1\). Si les \(B_i\) sont boréliens disjoints, les événements \(\{X\in B_i\}\) sont disjoints et \(\{X\in\bigcup_iB_i\}=\bigcup_i\{X\in B_i\}\). Donc \(\mu_X(\bigcup_iB_i)=\sum_i\mu_X(B_i)\).

Si \(x\le y\), alors \(\{X\le x\}\subset\{X\le y\}\), donc \(F_X(x)\le F_X(y)\).

Si \(x_n\downarrow x\), alors \(\{X\le x_n\}\downarrow\{X\le x\}\), et la continuité décroissante donne \(F_X(x_n)\downarrow F_X(x)\).

Enfin, \(\{X\le n\}\uparrow\Omega\) et \(\{X\le -n\}\downarrow\varnothing\), d'où les deux limites.

Deux formes courantes

\(X\) est discrète si \(\mu_X(C)=1\) pour un ensemble dénombrable \(C\subset\mathbb R\). Elle a une densité \(f\) si \(f\ge0\), \(\int_{\mathbb R}f(x)\,dx=1\), et \(P(X\in B)=\int_B f(x)\,dx\) pour tout borélien \(B\).

Guide Durrett : sections 1.2-1.3.

分布与随机变量

{% include probability-note-nav.html back_label="返回" prev_label="上一页" prev_url="/notes/probability-theory/probability-spaces/" next_label="下一页" next_url="/notes/probability-theory/pushforward-measures-densities/" %}

随机变量不只是函数；它必须可测，才能保证由它提出的概率问题确实是事件。

定义

概率空间 \((\Omega,\mathcal F,P)\) 上的实值随机变量是可测映射 \(X:(\Omega,\mathcal F)\to(\mathbb R,\mathcal B(\mathbb R))\)，即对任意 Borel 集 \(B\)，都有 \(\{X\in B\}\in\mathcal F\)。

\(X\) 的分布是 \(\mathbb R\) 上的测度 \(\mu_X(B)=P(X\in B)\)。它的分布函数是 \(F_X(x)=P(X\le x)\)。

命题

\(\mu_X\) 是概率测度。并且 \(F_X\) 单调不减、右连续，且 \(\lim_{x\to-\infty}F_X(x)=0\)、\(\lim_{x\to\infty}F_X(x)=1\)。

证明

因为 \(X^{-1}(\mathbb R)=\Omega\)，所以 \(\mu_X(\mathbb R)=1\)。若 \(B_i\) 两两不交，则 \(\{X\in B_i\}\) 两两不交，且 \(\{X\in\bigcup_iB_i\}=\bigcup_i\{X\in B_i\}\)。故 \(\mu_X(\bigcup_iB_i)=\sum_i\mu_X(B_i)\)。

若 \(x\le y\)，则 \(\{X\le x\}\subset\{X\le y\}\)，所以 \(F_X(x)\le F_X(y)\)。

若 \(x_n\downarrow x\)，则 \(\{X\le x_n\}\downarrow\{X\le x\}\)，由概率的上连续性，\(F_X(x_n)\downarrow F_X(x)\)，即右连续。

最后，\(\{X\le n\}\uparrow\Omega\)，\(\{X\le -n\}\downarrow\varnothing\)，由概率的连续性得到两个极限。

两种常见形式

若存在可数集 \(C\subset\mathbb R\) 使 \(\mu_X(C)=1\)，则称 \(X\) 为离散型。若存在 \(f\ge0\) 使 \(\int_{\mathbb R}f(x)\,dx=1\)，且对任意 Borel 集 \(B\)，\(P(X\in B)=\int_B f(x)\,dx\)，则称 \(X\) 有密度 \(f\)。

Durrett 参考：第 1.2-1.3 节。
