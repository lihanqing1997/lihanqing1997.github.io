Distributions & Random Variables

A random variable is a measurable function X : Ω → R; measurability makes events like {X ≤ x} legitimate.

Distribution

  • The distribution of X is the probability measure μX(B) = P(X ∈ B).
  • The distribution function is F(x) = P(X ≤ x).
  • A distribution function is nondecreasing, right-continuous, and has limits 0 at -∞ and 1 at ∞.

Useful forms

  • Discrete: probabilities are assigned to countably many points.
  • Continuous with density f: P(X ∈ B) = ∫B f(x) dx.
  • Same distribution means all probabilities determined by X agree, even if the variables live on different spaces.

Lois et variables aléatoires

Une variable aléatoire est une fonction mesurable X : Ω → R ; la mesurabilité rend les événements comme {X ≤ x} admissibles.

Loi

  • La loi de X est la mesure de probabilité μX(B) = P(X ∈ B).
  • La fonction de répartition est F(x) = P(X ≤ x).
  • Une fonction de répartition est croissante, continue à droite, et tend vers 0 en -∞ et 1 en ∞.

Formes utiles

  • Discrète : les masses sont portées par un ensemble au plus dénombrable.
  • Continue avec densité f : P(X ∈ B) = ∫B f(x) dx.
  • Avoir la même loi signifie que toutes les probabilités déterminées par X coïncident.

Guide Durrett : sections 1.2-1.3.

分布与随机变量

随机变量是可测函数 X : Ω → R；可测性保证 {X ≤ x} 这类集合是事件。

分布

  • X 的分布是概率测度 μX(B) = P(X ∈ B)。
  • 分布函数为 F(x) = P(X ≤ x)。
  • 分布函数单调不减、右连续，并且在 -∞ 处极限为 0，在 ∞ 处极限为 1。

常见形式

  • 离散型：概率集中在至多可数个点上。
  • 有密度 f 的连续型：P(X ∈ B) = ∫B f(x) dx。
  • 同分布表示由 X 决定的所有概率相同，即使变量定义在不同概率空间上。
