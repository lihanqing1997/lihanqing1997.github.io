3. Random variables and distributions

{% include probability-note-nav.html back_label="Back" prev_label="Last page" prev_url="/notes/probability-theory/measures-extension-monotone-classes/" next_label="Next page" next_url="/notes/probability-theory/independence/" %}

A random variable is a measurable map, and its distribution is the pushforward measure it induces. This viewpoint separates the random object from the particular probability space used to realize it.

Random variables as measurable maps

Measurability is what makes events of the form \(\{X\in B\}\) legitimate.

Definition. A random variable with values in a measurable space \((S,\mathcal S)\) is a measurable map \(X:(\Omega,\mathcal F)\to(S,\mathcal S)\). Its distribution, or law, is the probability measure \(P_X\) on \(S\) defined by \(P_X(B)=P(X\in B)\).

Distribution determines expectations

Every measurable calculation involving only \(X\) can be transferred to its law.

Theorem. If \(g:S\to\mathbb R\) is measurable and either nonnegative or \(g(X)\in L^1(P)\), then \[ \mathbb E[g(X)]=\int_S g(s)\,P_X(\mathrm{d}s). \]

Proof. For indicators \(g=1_B\), the identity is the definition of \(P_X\). By linearity it holds for nonnegative simple functions. Monotone convergence gives the nonnegative measurable case, and decomposing \(g=g^+-g^-\) gives the integrable case.

Equality in distribution

Two random variables may live on different probability spaces and still have the same probabilistic behavior.

Proposition. Random variables \(X\) and \(Y\) are equal in distribution, written \(X\stackrel{d}=Y\), if \(P_X=P_Y\). If \(S\) is a metric space with Borel \(\sigma\)-field, it is enough to check probabilities on a pi-system generating the Borel sets.

Proof. If probabilities agree on a generating pi-system, the pi-lambda theorem implies agreement on the generated \(\sigma\)-field. Therefore all bounded measurable expectations agree, and the variables have the same law.

Remark. Almost-sure equality implies equality in distribution, but the converse is much weaker: two independent standard normal variables have the same law without being equal almost surely.
