51. Girsanov theorem and change of measure

{% include probability-note-nav.html back_label="Back" prev_label="Last page" prev_url="/notes/probability-theory/applications-ito-formula/" next_label="Next page" next_url="/notes/probability-theory/stochastic-differential-equations/" %}

Girsanov's theorem describes how absolutely continuous changes of measure alter Brownian drift. It is the stochastic-calculus form of likelihood reweighting.

Stochastic exponential

The density process is an exponential martingale.

Definition. For a progressively measurable \(\theta_t\) with suitable integrability, define \[ Z_t=\exp\left(\int_0^t\theta_s\,\mathrm{d}B_s-\frac12\int_0^t\theta_s^2\,\mathrm{d}s\right). \]

Girsanov theorem

Under the tilted measure, Brownian motion gains the opposite drift.

Theorem. If \(Z_t\) is a true martingale on \([0,T]\) and \(\mathrm{d}Q=Z_T\,\mathrm{d}P\), then \[ \widetilde B_t=B_t-\int_0^t\theta_s\,\mathrm{d}s \] is a \(Q\)-Brownian motion on \([0,T]\).

Proof. For bounded \(\theta\), use Itô's formula to compute products \(Z_tM_t\), where \(M_t\) is an exponential martingale generated by \(\widetilde B\). The drift terms cancel exactly, showing that \(\exp(iu(\widetilde B_t-\widetilde B_s)+u^2(t-s)/2)\) has conditional \(Q\)-expectation one. This identifies independent Gaussian increments under \(Q\). General integrands follow by localization and approximation, provided \(Z\) is a true martingale.

Novikov condition

A common sufficient condition ensures the stochastic exponential is a martingale.

Proposition. If \[ \mathbb E\exp\left(\frac12\int_0^T\theta_s^2\,\mathrm{d}s\right)\lt\infty, \] then \(Z_t\) is a true martingale on \([0,T]\).

Proof. The proof localizes at levels where \(\int\theta^2\) is bounded, so the stopped exponentials are martingales. A convexity and conditional Hölder argument gives uniform integrability of the stopped terminal values under the displayed exponential integrability condition. Letting the localization levels increase preserves expectation one.

Remark. Girsanov changes drift, not quadratic variation; the transformed process remains Brownian because its bracket is still \(t\).
