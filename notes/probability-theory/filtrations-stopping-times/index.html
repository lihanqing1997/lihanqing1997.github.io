Filtrations, Adapted Processes, and Stopping Times

{% include probability-note-nav.html back_label="Back" prev_label="Last page" prev_url="/notes/probability-theory/stable-infinitely-divisible-laws/" next_label="Next page" next_url="/notes/probability-theory/discrete-time-martingales/" %}

This page records the core mathematical facts for Filtrations, Adapted Processes, and Stopping Times.

Definition. A filtration \((\mathcal F_t)\) is an increasing family of \(\sigma\)-fields. A process \(X_t\) is adapted if \(X_t\) is \(\mathcal F_t\)-measurable. A random time \(\tau\) is a stopping time if \(\{\tau\le t\}\in\mathcal F_t\) for every \(t\).

Central result. If \(X_t\) is adapted and \(\tau\) is a stopping time, then the stopped process \(X_{t\wedge\tau}\) is adapted. In discrete time, \(\{\tau=n\}\in\mathcal F_n\), so information available at the stopping time is well-defined.

Proof. For each fixed \(t\), the variable \(t\wedge\tau\) only uses the events \(\{\tau\le s\}\) with \(s\le t\). In discrete time, \(X_\tau1_{\{\tau\le n\}}=\sum_{k=0}^nX_k1_{\{\tau=k\}}\), and each summand is \(\mathcal F_n\)-measurable.

How to read it

Stopping times encode non-anticipation. They allow random decisions based only on information already seen.

The point is not to memorize the displayed formulas separately, but to understand how the definition, the result, and the proof mechanism fit together.
