Stochastic integration with respect to semimartingales

{% include probability-note-nav.html back_label="Back" prev_label="Last page" prev_url="/notes/probability-theory/stochastic-integration-brownian-motion/" next_label="Next page" next_url="/notes/probability-theory/ito-formula/" %}

This topic belongs to stochastic calculus. Ordinary calculus must be modified because martingale increments have non-negligible quadratic variation. Here the focus is Stochastic integration with respect to semimartingales.

Start with the precise object. The definition states what the object is and what assumptions make the later formulas meaningful.

Definition. For a semimartingale \(X=M+A\), define \(\int H\,dX=\int H\,dM+\int H\,dA\), where the martingale integral is built by localization and the finite-variation integral is pathwise Lebesgue-Stieltjes.

These assumptions are part of the mathematics. They tell us where each expression lives and prevent informal notation from hiding a real domain issue.

The result below is the fact one usually uses in calculations or proofs. Read it as the operational content of the definition.

Result. If \(X,Y\) are continuous semimartingales, then integration by parts holds: \[X_tY_t=X_0Y_0+\int_0^tX_s\,dY_s+\int_0^tY_s\,dX_s+[X,Y]_t.\]

Proof.

The proof is easiest to read by separating the elementary case from the extension step.

Apply the discrete identity \(\Delta(XY)=X\Delta Y+Y\Delta X+\Delta X\Delta Y\) along partitions. The first two sums converge to stochastic integrals with predictable left endpoints, while the last sum converges to covariation.

Remarks

The extra covariation term is the product-rule signature of stochastic calculus.
