Radon-Nikodym derivatives, densities, and change of measure

{% include probability-note-nav.html back_label="Back" prev_label="Last page" prev_url="/notes/probability-theory/lp-spaces-inequalities-uniform-integrability/" next_label="Next page" next_url="/notes/probability-theory/conditional-probability-regular-conditional-distributions/" %}

Definition. For a measurable map \(T:(S,\mathcal S)\to(E,\mathcal E)\) and a measure \(\mu\) on \(S\), the pushforward measure is \(T_\#\mu(B)=\mu(T^{-1}(B))\). If \(\nu\ll\mu\), the Radon-Nikodym derivative \(\mathrm{d}\nu/\mathrm{d}\mu\) is the measurable function \(h\ge0\) satisfying \(\nu(A)=\int_Ah\,\mathrm{d}\mu\) for all \(A\in\mathcal S\).

Theorem. If \(Y=T(X)\), then for every nonnegative or integrable \(g\), \[ \mathbb E[g(Y)]=\int_E g(y)\,P_Y(\mathrm{d}y)=\int_\Omega g(T(X(\omega)))\,P(\mathrm{d}\omega). \] If \(\nu\ll\mu\), then \(\int f\,\mathrm{d}\nu=\int f(\mathrm{d}\nu/\mathrm{d}\mu)\,\mathrm{d}\mu\).

Proof. For \(g=1_B\), the first identity reads \(P(Y\in B)=P_X(T^{-1}(B))\), which is the definition of pushforward. Linearity gives simple functions, and monotone convergence gives nonnegative \(g\); signed integrable \(g\) follows by positive and negative parts. The Radon-Nikodym integration formula is proved the same way: it is true for indicators by the defining identity \(\nu(A)=\int_A h\,\mathrm{d}\mu\), then extends to simple, nonnegative, and integrable functions.

Remark

Densities are not tied to Lebesgue measure only; \(\mathrm{d}\nu/\mathrm{d}\mu\) is a density of \(\nu\) relative to any dominating measure \(\mu\).

Dérivées de Radon-Nikodym, densités et changement de mesure

{% include probability-note-nav.html back_label="Retour" prev_label="Page précédente" prev_url="/notes/probability-theory/lp-spaces-inequalities-uniform-integrability/" next_label="Page suivante" next_url="/notes/probability-theory/conditional-probability-regular-conditional-distributions/" %}

Définition. Pour une application mesurable \(T:(S,\mathcal S)\to(E,\mathcal E)\) et une mesure \(\mu\) sur \(S\), l’image directe est \(T_\#\mu(B)=\mu(T^{-1}(B))\). Si \(\nu\ll\mu\), la dérivée de Radon-Nikodym \(\mathrm{d}\nu/\mathrm{d}\mu\) est une fonction mesurable \(h\ge0\) telle que \(\nu(A)=\int_Ah\,\mathrm{d}\mu\) pour tout \(A\in\mathcal S\).

Théorème. Si \(Y=T(X)\), alors pour toute fonction \(g\) positive ou intégrable, \[ \mathbb E[g(Y)]=\int_E g(y)\,P_Y(\mathrm{d}y)=\int_\Omega g(T(X(\omega)))\,P(\mathrm{d}\omega). \] Si \(\nu\ll\mu\), alors \(\int f\,\mathrm{d}\nu=\int f(\mathrm{d}\nu/\mathrm{d}\mu)\,\mathrm{d}\mu\).

Preuve. Pour \(g=1_B\), la première identité dit \(P(Y\in B)=P_X(T^{-1}(B))\), qui est la définition de l’image directe. La linéarité donne les fonctions simples, puis la convergence monotone donne les fonctions positives ; le cas intégrable signé suit par parties positive et négative. La formule de Radon-Nikodym se prouve de la même manière : elle est vraie pour les indicatrices par \(\nu(A)=\int_Ah\,\mathrm{d}\mu\), puis s’étend aux fonctions simples, positives et intégrables.

Remarque

Une densité n’est pas forcément relative à la mesure de Lebesgue ; \(\mathrm{d}\nu/\mathrm{d}\mu\) est la densité de \(\nu\) par rapport à n’importe quelle mesure dominante \(\mu\).

Radon-Nikodym 导数、密度与换测度

{% include probability-note-nav.html back_label="返回" prev_label="上一页" prev_url="/notes/probability-theory/lp-spaces-inequalities-uniform-integrability/" next_label="下一页" next_url="/notes/probability-theory/conditional-probability-regular-conditional-distributions/" %}

定义. 设 \(T:(S,\mathcal S)\to(E,\mathcal E)\) 可测，\(\mu\) 是 \(S\) 上的测度。推前测度定义为 \(T_\#\mu(B)=\mu(T^{-1}(B))\)。若 \(\nu\ll\mu\)，则 Radon-Nikodym 导数 \(\mathrm{d}\nu/\mathrm{d}\mu\) 是满足 \(\nu(A)=\int_Ah\,\mathrm{d}\mu\) 的非负可测函数 \(h\)，其中 \(A\in\mathcal S\)。

定理. 若 \(Y=T(X)\)，则对任意非负或可积函数 \(g\)， \[ \mathbb E[g(Y)]=\int_E g(y)\,P_Y(\mathrm{d}y)=\int_\Omega g(T(X(\omega)))\,P(\mathrm{d}\omega). \] 若 \(\nu\ll\mu\)，则 \(\int f\,\mathrm{d}\nu=\int f(\mathrm{d}\nu/\mathrm{d}\mu)\,\mathrm{d}\mu\)。

证明. 对 \(g=1_B\)，第一个等式就是 \(P(Y\in B)=P_X(T^{-1}(B))\)，这正是推前测度的定义。由线性性推广到简单函数，再由单调收敛推广到非负函数；有符号可积函数由正部和负部分解得到。Radon-Nikodym 积分公式同理：对示性函数由 \(\nu(A)=\int_Ah\,\mathrm{d}\mu\) 成立，再推广到简单、非负和可积函数。

说明

密度不一定是相对于 Lebesgue 测度的；\(\mathrm{d}\nu/\mathrm{d}\mu\) 是 \(\nu\) 相对于任何支配测度 \(\mu\) 的密度。
