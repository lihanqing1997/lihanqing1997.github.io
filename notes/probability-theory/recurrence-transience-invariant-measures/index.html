Recurrence, Transience, and Invariant Measures

This page records the core mathematical facts for Recurrence, Transience, and Invariant Measures.

Definition. A measure \(\pi\) is invariant for a kernel \(K\) if \(\pi K=\pi\). A state is recurrent if the process returns to it with probability one, and transient if the expected number of visits is finite.

Central result. For an irreducible positive recurrent Markov chain on a countable state space, there is a unique invariant probability \(\pi\), and \(\pi_i=1/\mathbb E_iT_i^+\), where \(T_i^+\) is the first return time to \(i\).

Proof. Regenerate the chain at successive returns to state \(i\). The expected time spent in state \(j\) during one cycle divided by the expected cycle length defines \(\pi_j\). Renewal arguments show \(\pi K=\pi\), and irreducibility gives uniqueness.

How to read it

Invariant measures are equilibrium laws. Recurrence determines whether equilibrium can be normalized into a probability distribution.

The point is not to memorize the displayed formulas separately, but to understand how the definition, the result, and the proof mechanism fit together.
