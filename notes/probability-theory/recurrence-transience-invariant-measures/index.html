Recurrence, transience, and invariant measures

{% include probability-note-nav.html back_label="Back" prev_label="Last page" prev_url="/notes/probability-theory/markov-processes-semigroups/" next_label="Next page" next_url="/notes/probability-theory/ergodic-theorems/" %}

This topic belongs to stochastic processes. The central move is to regard randomness as a whole path rather than as a single random variable. Here the focus is Recurrence, transience, and invariant measures.

Start with the precise object. The definition states what the object is and what assumptions make the later formulas meaningful.

Definition. A measure \(\pi\) is invariant for a kernel \(K\) if \(\pi K=\pi\). A state is recurrent if the process returns to it with probability one, and transient if the expected number of visits is finite.

These assumptions are part of the mathematics. They tell us where each expression lives and prevent informal notation from hiding a real domain issue.

The result below is the fact one usually uses in calculations or proofs. Read it as the operational content of the definition.

Result. For an irreducible positive recurrent Markov chain on a countable state space, there is a unique invariant probability \(\pi\), and \(\pi_i=1/\mathbb E_iT_i^+\), where \(T_i^+\) is the first return time to \(i\).

Proof.

The proof is easiest to read by separating the elementary case from the extension step.

Regenerate the chain at successive returns to state \(i\). The expected time spent in state \(j\) during one cycle divided by the expected cycle length defines \(\pi_j\). Renewal arguments show \(\pi K=\pi\), and irreducibility gives uniqueness.

Remarks

Invariant measures are equilibrium laws. Recurrence determines whether equilibrium can be normalized into a probability distribution.
