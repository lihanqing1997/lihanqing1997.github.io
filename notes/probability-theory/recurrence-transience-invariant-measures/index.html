32. Recurrence, transience, and invariant measures

{% include probability-note-nav.html back_label="Back" prev_label="Last page" prev_url="/notes/probability-theory/markov-processes-semigroups/" next_label="Next page" next_url="/notes/probability-theory/ergodic-theorems/" %}

Definition. For a Markov chain, a state \(i\) is recurrent if \(P_i(T_i\lt \infty)=1\), where \(T_i=\inf\{n\ge1:X_n=i\}\). It is transient if this probability is less than \(1\). A measure \(\pi\) is invariant if \(\pi P=\pi\).

Proposition. For a countable-state chain, \(i\) is recurrent if and only if \(\sum_{n=0}^\infty P_i(X_n=i)=\infty\). It is transient if and only if this sum is finite.

Proof. Let \(f=P_i(T_i\lt \infty)\). After each return to \(i\), the strong Markov property restarts the chain from \(i\). Hence the number of visits to \(i\) has expectation \(1+f+f^2+\cdots\), equal to \(\infty\) when \(f=1\) and \(1/(1-f)\) when \(f\lt 1\). But this expected number is also \(\sum_{n=0}^\infty P_i(X_n=i)\).

Equilibrium and return behavior

Invariant measures describe equilibrium; recurrence describes whether the chain repeatedly comes back to a state.

32. Récurrence, transience et mesures invariantes

{% include probability-note-nav.html back_label="Retour" prev_label="Page précédente" prev_url="/notes/probability-theory/markov-processes-semigroups/" next_label="Page suivante" next_url="/notes/probability-theory/ergodic-theorems/" %}

Définition. Pour une chaîne de Markov, un état \(i\) est récurrent si \(P_i(T_i\lt \infty)=1\), où \(T_i=\inf\{n\ge1:X_n=i\}\). Il est transient si cette probabilité est strictement inférieure à \(1\). Une mesure \(\pi\) est invariante si \(\pi P=\pi\).

Proposition. Pour une chaîne à espace d’états dénombrable, \(i\) est récurrent si et seulement si \(\sum_{n=0}^\infty P_i(X_n=i)=\infty\). Il est transient si et seulement si cette somme est finie.

Preuve. Soit \(f=P_i(T_i\lt \infty)\). Après chaque retour en \(i\), la propriété forte de Markov redémarre la chaîne depuis \(i\). Le nombre de visites en \(i\) a donc pour espérance \(1+f+f^2+\cdots\), infinie si \(f=1\), et égale à \(1/(1-f)\) si \(f\lt 1\). Cette espérance est aussi \(\sum_{n=0}^\infty P_i(X_n=i)\).

Équilibre et comportement de retour

Les mesures invariantes décrivent l’équilibre ; la récurrence décrit le retour répété vers un état.

32. 常返、暂留与不变测度

{% include probability-note-nav.html back_label="返回" prev_label="上一页" prev_url="/notes/probability-theory/markov-processes-semigroups/" next_label="下一页" next_url="/notes/probability-theory/ergodic-theorems/" %}

定义. 对 Markov 链，状态 \(i\) 称为常返，若 \(P_i(T_i\lt \infty)=1\)，其中 \(T_i=\inf\{n\ge1:X_n=i\}\)。若该概率小于 \(1\)，则称为暂留。测度 \(\pi\) 称为不变测度，若 \(\pi P=\pi\)。

命题. 对可数状态 Markov 链，\(i\) 常返当且仅当 \(\sum_{n=0}^\infty P_i(X_n=i)=\infty\)；暂留当且仅当该和有限。

证明. 令 \(f=P_i(T_i\lt \infty)\)。每次回到 \(i\) 后，由强 Markov 性，链从 \(i\) 重新开始。因此访问 \(i\) 的次数期望为 \(1+f+f^2+\cdots\)，当 \(f=1\) 时为无穷，当 \(f\lt 1\) 时为 \(1/(1-f)\)。另一方面，该期望也等于 \(\sum_{n=0}^\infty P_i(X_n=i)\)。

平衡与返回行为

不变测度描述平衡；常返性描述链是否会反复回到某个状态。
