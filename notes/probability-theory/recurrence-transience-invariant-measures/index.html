32. Recurrence, transience, and invariant measures

{% include probability-note-nav.html back_label="Back" prev_label="Last page" prev_url="/notes/probability-theory/markov-processes-semigroups/" next_label="Next page" next_url="/notes/probability-theory/ergodic-theorems/" %}

Recurrence, transience, and invariant measures describe long-run behavior of Markov chains and processes. They distinguish states that are repeatedly revisited from states that eventually disappear.

Return probabilities

Recurrence is a statement about eventual returns.

Definition. For a Markov chain on a countable state space, let \(T_i^+=\inf\{n\ge1:X_n=i\}\). State \(i\) is recurrent if \(P_i(T_i^+\lt\infty)=1\), and transient otherwise.

Green function criterion

Expected visit counts determine recurrence.

Theorem. For an irreducible countable Markov chain, \[ i\ \text{is recurrent}\quad\Longleftrightarrow\quad \sum_{n=0}^\infty P_i(X_n=i)=\infty. \]

Proof. Each return to \(i\) restarts the chain by the strong Markov property. The number of visits to \(i\) is therefore geometric in the number of returns. If the return probability is \(f_i\lt1\), the expected number of visits is \(1/(1-f_i)\lt\infty\). If \(f_i=1\), repeated returns occur almost surely and the expected number of visits is infinite.

Invariant measures

Invariant measures are fixed points of the transition kernel.

Proposition. A measure \(\pi\) is invariant for a kernel \(K\) if \[ \pi(A)=\int K(x,A)\,\pi(\mathrm{d}x) \] for every measurable \(A\), equivalently \(\pi K=\pi\).

Proof. If \(X_0\sim\pi\), then \(P(X_1\in A)=\int K(x,A)\pi(\mathrm{d}x)=\pi(A)\). Inductively \(X_n\sim\pi\) for all \(n\). Conversely, if the one-step distribution remains \(\pi\) whenever \(X_0\sim\pi\), then the displayed identity follows.

Remark. Positive recurrent irreducible chains have invariant probability measures; null recurrent chains have recurrent behavior without finite invariant mass.
