Continuous-Time Martingales and Local Martingales

{% include probability-note-nav.html back_label="Back" prev_label="Last page" prev_url="/notes/probability-theory/brownian-motion-pdes/" next_label="Next page" next_url="/notes/probability-theory/quadratic-variation-covariation/" %}

This page records the core mathematical facts for Continuous-Time Martingales and Local Martingales.

Definition. A continuous-time adapted process \(M_t\) is a martingale if it is integrable and \(\mathbb E(M_t\mid\mathcal F_s)=M_s\) for \(s\le t\). It is a local martingale if there are stopping times \(\tau_n\uparrow\infty\) such that \(M_{t\wedge\tau_n}\) is a martingale for each \(n\).

Central result. Every nonnegative local martingale is a supermartingale. A local martingale that is uniformly integrable on each finite horizon is a true martingale.

Proof. For a nonnegative local martingale, apply Fatou's lemma to the stopped martingales: \(\mathbb E(M_t\mid\mathcal F_s)\le\liminf_n\mathbb E(M_{t\wedge\tau_n}\mid\mathcal F_s)=\liminf_nM_{s\wedge\tau_n}=M_s\). Uniform integrability permits passing limits inside conditional expectations with equality.

How to read it

Localization lets one prove identities under bounded conditions first, then remove the bounds by stopping before bad behavior occurs.

The point is not to memorize the displayed formulas separately, but to understand how the definition, the result, and the proof mechanism fit together.
