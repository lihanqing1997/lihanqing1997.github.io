45. Continuous-time martingales and local martingales

{% include probability-note-nav.html back_label="Back" prev_label="Last page" prev_url="/notes/probability-theory/brownian-motion-pdes/" next_label="Next page" next_url="/notes/probability-theory/quadratic-variation-covariation/" %}

Continuous-time martingales require path regularity and localization. Local martingales behave like martingales up to an increasing sequence of stopping times.

Continuous-time martingales

The martingale identity is imposed at all ordered times.

Definition. An adapted integrable process \(M_t\) is a martingale if \(\mathbb E[M_t\mid\mathcal F_s]=M_s\) for all \(s\le t\). It is usually assumed to have càdlàg paths.

Localization

Local martingales are martingales before suitable stopping times.

Theorem. A process \(M_t\) is a local martingale if there exist stopping times \(\tau_n\uparrow\infty\) such that \(M^{\tau_n}_t=M_{t\wedge\tau_n}\) is a martingale for every \(n\).

Proof. Localization permits one to prove martingale identities under boundedness, then remove boundedness by stopping. If \(M^{\tau_n}\) is a martingale for each \(n\), then for bounded stopping times \(\sigma\le\tau\), optional sampling applies to \(M^{\tau_n}\); letting \(n\to\infty\) gives local versions of martingale identities whenever uniform integrability or bounded convergence is available.

Class D and true martingales

A local martingale becomes a true martingale under uniform integrability across stopping times.

Proposition. If a càdlàg local martingale \(M\) has the family \(\{M_\tau:\tau\le T\}\) uniformly integrable for every fixed \(T\), then \(M\) is a martingale on \([0,T]\).

Proof. Choose a localizing sequence \(\tau_n\). For \(s\le t\le T\), optional sampling gives \(\mathbb E[M_{t\wedge\tau_n}\mid\mathcal F_s]=M_{s\wedge\tau_n}\). Uniform integrability lets \(n\to\infty\) in \(L^1\), yielding \(\mathbb E[M_t\mid\mathcal F_s]=M_s\).

Remark. Positive local martingales are supermartingales, but may fail to be true martingales.
