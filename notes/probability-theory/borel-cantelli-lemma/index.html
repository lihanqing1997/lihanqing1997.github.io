Borel-Cantelli Lemma

Borel-Cantelli translates summability of probabilities into almost sure eventual behavior.

Infinitely often

  • {An i.o.} means infinitely many An occur.
  • Formally, {An i.o.} = ∩m≥1 ∪n≥m An.

The lemmas

  • If ∑ P(An) < ∞, then P(An i.o.) = 0.
  • If the An are independent and ∑ P(An) = ∞, then P(An i.o.) = 1.
  • Main use: summable error probabilities imply only finitely many errors occur almost surely.

Durrett guide: Section 2.3.

Lemme de Borel-Cantelli

Borel-Cantelli transforme la sommabilité des probabilités en comportement presque sûr à terme.

Une infinité de fois

  • {An i.o.} signifie qu'une infinité des événements An se produisent.
  • Formellement, {An i.o.} = ∩m≥1 ∪n≥m An.

Les lemmes

  • Si ∑ P(An) < ∞, alors P(An i.o.) = 0.
  • Si les An sont indépendants et ∑ P(An) = ∞, alors P(An i.o.) = 1.
  • Usage principal : des probabilités d'erreur sommables donnent seulement un nombre fini d'erreurs p.s.

Guide Durrett : section 2.3.

Borel-Cantelli 引理

Borel-Cantelli 引理把概率的可和性转化为几乎必然的最终行为。

无穷多次发生

  • {An i.o.} 表示有无穷多个 An 发生。
  • 形式上，{An i.o.} = ∩m≥1 ∪n≥m An。

两个引理

  • 若 ∑ P(An) < ∞，则 P(An i.o.) = 0。
  • 若 An 相互独立且 ∑ P(An) = ∞，则 P(An i.o.) = 1。
  • 主要用途：若错误概率可和，则几乎必然只发生有限多次错误。

Durrett 参考：第 2.3 节。
