Borel-Cantelli lemmas and zero-one laws

{% include probability-note-nav.html back_label="Back" prev_label="Last page" prev_url="/notes/probability-theory/weak-law-of-large-numbers/" next_label="Next page" next_url="/notes/probability-theory/strong-law-of-large-numbers/" %}

Definition. For events \(A_n\), the event that infinitely many of them occur is \[ \{A_n\ \text{i.o.}\}=\limsup_{n\to\infty}A_n=\bigcap_{m=1}^\infty\bigcup_{n\ge m}A_n. \]

Theorem. If \(\sum_nP(A_n)\lt \infty\), then \(P(A_n\text{ i.o.})=0\). If the events \(A_n\) are independent and \(\sum_nP(A_n)=\infty\), then \(P(A_n\text{ i.o.})=1\).

Proof. For the first part, \[ P\left(\bigcup_{n\ge m}A_n\right)\le\sum_{n\ge m}P(A_n)\to0. \] Since \(A_n\text{ i.o.}\subset\bigcup_{n\ge m}A_n\) for every \(m\), the probability is \(0\). For the second part, \[ P\left(\bigcap_{n=m}^N A_n^c\right)=\prod_{n=m}^N(1-P(A_n)) \le \exp\left(-\sum_{n=m}^NP(A_n)\right)\to0. \] Thus with probability one at least one \(A_n\) occurs after every \(m\), which is exactly occurrence infinitely often.

Remark

Borel-Cantelli converts summability of probabilities into almost sure statements about eventual behavior.

Lemmes de Borel-Cantelli et lois zéro-un

{% include probability-note-nav.html back_label="Retour" prev_label="Page précédente" prev_url="/notes/probability-theory/weak-law-of-large-numbers/" next_label="Page suivante" next_url="/notes/probability-theory/strong-law-of-large-numbers/" %}

Définition. Pour des événements \(A_n\), l’événement « une infinité de \(A_n\) se réalisent » est \[ \{A_n\ \text{i.o.}\}=\limsup_{n\to\infty}A_n=\bigcap_{m=1}^\infty\bigcup_{n\ge m}A_n. \]

Théorème. Si \(\sum_nP(A_n)\lt \infty\), alors \(P(A_n\text{ i.o.})=0\). Si les \(A_n\) sont indépendants et si \(\sum_nP(A_n)=\infty\), alors \(P(A_n\text{ i.o.})=1\).

Preuve. Pour la première partie, \[ P\left(\bigcup_{n\ge m}A_n\right)\le\sum_{n\ge m}P(A_n)\to0. \] Comme \(A_n\text{ i.o.}\subset\bigcup_{n\ge m}A_n\) pour tout \(m\), la probabilité est nulle. Pour la deuxième partie, \[ P\left(\bigcap_{n=m}^N A_n^c\right)=\prod_{n=m}^N(1-P(A_n)) \le \exp\left(-\sum_{n=m}^NP(A_n)\right)\to0. \] Donc, avec probabilité un, au moins un \(A_n\) se produit après chaque \(m\), ce qui signifie exactement une infinité d’occurrences.

Remarque

Borel-Cantelli transforme une condition de sommabilité en un énoncé presque sûr sur le comportement éventuel.

Borel-Cantelli 引理与零一律

{% include probability-note-nav.html back_label="返回" prev_label="上一页" prev_url="/notes/probability-theory/weak-law-of-large-numbers/" next_label="下一页" next_url="/notes/probability-theory/strong-law-of-large-numbers/" %}

定义. 对事件 \(A_n\)，无穷多个 \(A_n\) 发生的事件为 \[ \{A_n\ \text{i.o.}\}=\limsup_{n\to\infty}A_n=\bigcap_{m=1}^\infty\bigcup_{n\ge m}A_n. \]

定理. 若 \(\sum_nP(A_n)\lt \infty\)，则 \(P(A_n\text{ i.o.})=0\)。若 \(A_n\) 相互独立且 \(\sum_nP(A_n)=\infty\)，则 \(P(A_n\text{ i.o.})=1\)。

证明. 第一部分， \[ P\left(\bigcup_{n\ge m}A_n\right)\le\sum_{n\ge m}P(A_n)\to0. \] 因为 \(A_n\text{ i.o.}\subset\bigcup_{n\ge m}A_n\) 对每个 \(m\) 都成立，所以其概率为 \(0\)。第二部分，由独立性， \[ P\left(\bigcap_{n=m}^N A_n^c\right)=\prod_{n=m}^N(1-P(A_n)) \le \exp\left(-\sum_{n=m}^NP(A_n)\right)\to0. \] 因而以概率一，对每个 \(m\)，在 \(m\) 之后至少有一个 \(A_n\) 发生，这正是无穷多次发生。

说明

Borel-Cantelli 引理把概率可和性转化为关于最终行为的几乎处处结论。
