Borel-Cantelli lemmas and zero-one laws

{% include probability-note-nav.html back_label="Back" prev_label="Last page" prev_url="/notes/probability-theory/weak-law-of-large-numbers/" next_label="Next page" next_url="/notes/probability-theory/strong-law-of-large-numbers/" %}

This topic belongs to classical limit theory. The main question is how random quantities behave as the sample size or time scale grows. Here the focus is Borel-Cantelli lemmas and zero-one laws.

Start with the precise object. The definition states what the object is and what assumptions make the later formulas meaningful.

Definition. For events \(A_n\), the event \(A_n\ \mathrm{i.o.}\) is \(\limsup_nA_n=\bigcap_m\bigcup_{n\ge m}A_n\). It means infinitely many of the events occur.

These assumptions are part of the mathematics. They tell us where each expression lives and prevent informal notation from hiding a real domain issue.

The result below is the fact one usually uses in calculations or proofs. Read it as the operational content of the definition.

Result. If \(\sum_nP(A_n)\lt\infty\), then \(P(A_n\ \mathrm{i.o.})=0\). If the \(A_n\) are independent and \(\sum_nP(A_n)=\infty\), then \(P(A_n\ \mathrm{i.o.})=1\). Tail events of independent sequences have probability \(0\) or \(1\).

Proof.

The proof is easiest to read by separating the elementary case from the extension step.

For the first part, \(P(\bigcup_{n\ge m}A_n)\le\sum_{n\ge m}P(A_n)\to0\). For the independent converse, \(P(\cap_{n=m}^N A_n^c)=\prod_{n=m}^N(1-P(A_n))\le\exp(-\sum_{n=m}^NP(A_n))\to0\). The zero-one law follows because tail events are independent of every finite initial block, hence independent of the whole generated \(\sigma\)-field.

Remarks

Borel-Cantelli converts summability into almost sure eventual behavior. It is a small theorem with enormous reach.
