Uniform integrability and L1/Lp martingale convergence

{% include probability-note-nav.html back_label="Back" prev_label="Last page" prev_url="/notes/probability-theory/optional-stopping-sampling/" next_label="Next page" next_url="/notes/probability-theory/square-integrable-martingales-quadratic-variation/" %}

This topic belongs to martingale theory. The guiding idea is to describe random evolution relative to information that grows over time. Here the focus is Uniform integrability and L1/Lp martingale convergence.

Start with the precise object. The definition states what the object is and what assumptions make the later formulas meaningful.

Definition. A martingale \((M_n)\) is uniformly integrable if \(\{M_n:n\ge0\}\) is uniformly integrable. Uniform integrability is stronger than boundedness in \(L^1\), but weaker than boundedness in \(L^p\) for some \(p>1\).

These assumptions are part of the mathematics. They tell us where each expression lives and prevent informal notation from hiding a real domain issue.

The result below is the fact one usually uses in calculations or proofs. Read it as the operational content of the definition.

Result. A martingale \((M_n)\) converges in \(L^1\) if and only if it is uniformly integrable. In that case \(M_n=\mathbb E(M_\infty\mid\mathcal F_n)\).

Proof.

The proof is easiest to read by separating the elementary case from the extension step.

If \(M_n\to M_\infty\) in \(L^1\), uniform integrability follows from \(L^1\)-relative compactness. Conversely, Doob convergence gives almost sure convergence to \(M_\infty\). Uniform integrability upgrades almost sure convergence to \(L^1\) convergence by Vitali's theorem, and the representation follows from passing \(M_n=\mathbb E(M_m\mid\mathcal F_n)\) to the limit \(m\to\infty\).

Remarks

Uniform integrability is the exact condition that makes the limiting martingale still carry the original mass.
