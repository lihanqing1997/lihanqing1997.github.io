Path spaces and finite-dimensional distributions

{% include probability-note-nav.html back_label="Back" prev_label="Last page" prev_url="/notes/probability-theory/kolmogorov-extension/" next_label="Next page" next_url="/notes/probability-theory/markov-chains-transition-kernels/" %}

This topic belongs to stochastic processes. The central move is to regard randomness as a whole path rather than as a single random variable. Here the focus is Path spaces and finite-dimensional distributions.

Start with the precise object. The definition states what the object is and what assumptions make the later formulas meaningful.

Definition. For a process \(X=(X_t)_{t\in T}\), the finite-dimensional distributions are the laws of \((X_{t_1},\ldots,X_{t_n})\). The canonical path space is often \(S^T\), \(C[0,1]\), or \(D[0,1]\), with coordinate maps as the process.

These assumptions are part of the mathematics. They tell us where each expression lives and prevent informal notation from hiding a real domain issue.

The result below is the fact one usually uses in calculations or proofs. Read it as the operational content of the definition.

Result. On the canonical space, the cylinder \(\sigma\)-field is generated by coordinate projections. On \(C[0,1]\), the Borel \(\sigma\)-field for the uniform norm is generated by the coordinate maps.

Proof.

The proof is easiest to read by separating the elementary case from the extension step.

Evaluation maps are continuous on \(C[0,1]\), so their generated \(\sigma\)-field is contained in the Borel \(\sigma\)-field. Conversely, the uniform norm is determined by values on rational times for continuous functions, so open balls can be generated from countably many coordinate conditions.

Remarks

Finite-dimensional distributions describe pointwise behavior. To prove convergence of processes, one also needs tightness in the chosen path topology.
