Triangular arrays and truncation methods

{% include probability-note-nav.html back_label="Back" prev_label="Last page" prev_url="/notes/probability-theory/strong-law-of-large-numbers/" next_label="Next page" next_url="/notes/probability-theory/central-limit-theorem/" %}

This topic belongs to classical limit theory. The main question is how random quantities behave as the sample size or time scale grows. Here the focus is Triangular arrays and truncation methods.

Start with the precise object. The definition states what the object is and what assumptions make the later formulas meaningful.

Definition. A triangular array is a collection \(X_{n,k}\), \(1\le k\le k_n\), usually with independence within each row. It lets the distribution of summands vary with \(n\). Lindeberg's condition controls large row terms: \(\sum_k\mathbb E[X_{n,k}^2 1_{\{|X_{n,k}|>\varepsilon\}}]\to0\).

These assumptions are part of the mathematics. They tell us where each expression lives and prevent informal notation from hiding a real domain issue.

The result below is the fact one usually uses in calculations or proofs. Read it as the operational content of the definition.

Result. If row sums are centered, independent, have total variance \(\sum_k\operatorname{Var}(X_{n,k})\to1\), and satisfy Lindeberg's condition, then \(\sum_kX_{n,k}\Rightarrow N(0,1)\).

Proof.

The proof is easiest to read by separating the elementary case from the extension step.

Use characteristic functions. Taylor expansion gives \(\log\prod_k\mathbb E e^{itX_{n,k}}=\sum_k(-t^2\mathbb EX_{n,k}^2/2)+o(1)\); the Lindeberg condition makes the Taylor remainders negligible uniformly over \(k\). The limit is \(e^{-t^2/2}\), so Lévy's theorem gives the result.

Remarks

Truncation separates the typical part from rare large observations. It is the basic method for moving beyond finite bounded moments.
