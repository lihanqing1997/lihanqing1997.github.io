33. Ergodic theorems

{% include probability-note-nav.html back_label="Back" prev_label="Last page" prev_url="/notes/probability-theory/recurrence-transience-invariant-measures/" next_label="Next page" next_url="/notes/probability-theory/renewal-random-walks/" %}

Definition. A measure-preserving map \(T:(\Omega,\mathcal F,P)\to(\Omega,\mathcal F,P)\) satisfies \(P(T^{-1}A)=P(A)\). It is ergodic if every invariant event \(A=T^{-1}A\) has probability \(0\) or \(1\).

Theorem. If \(T\) is measure-preserving and \(f\in L^1\), then \[ \frac1n\sum_{k=0}^{n-1}f(T^k\omega)\to \mathbb E[f\mid\mathcal I](\omega) \quad\text{a.s.}, \] where \(\mathcal I\) is the invariant \(\sigma\)-field. If \(T\) is ergodic, the limit is \(\mathbb Ef\).

Proof. For \(f=g-g\circ T\), the averages telescope: \[ n^{-1}\sum_{k=0}^{n-1}(g-g\circ T)(T^k\omega) =\frac{g(\omega)-g(T^n\omega)}{n}\to0 \] in the appropriate truncated \(L^1\) sense. General \(L^1\) functions are handled by approximating them by invariant parts plus such coboundaries and using the maximal ergodic inequality. The limit is \(T\)-invariant, hence equals \(\mathbb E[f\mid\mathcal I]\); if \(\mathcal I\) is trivial, this is the constant \(\mathbb Ef\).

Time averages and invariant information

The ergodic theorem turns long-run time averages into conditional expectations on invariant information.

33. Théorèmes ergodiques

{% include probability-note-nav.html back_label="Retour" prev_label="Page précédente" prev_url="/notes/probability-theory/recurrence-transience-invariant-measures/" next_label="Page suivante" next_url="/notes/probability-theory/renewal-random-walks/" %}

Définition. Une application préservant la mesure \(T:(\Omega,\mathcal F,P)\to(\Omega,\mathcal F,P)\) vérifie \(P(T^{-1}A)=P(A)\). Elle est ergodique si tout événement invariant \(A=T^{-1}A\) a probabilité \(0\) ou \(1\).

Théorème. Si \(T\) préserve la mesure et \(f\in L^1\), alors \[ \frac1n\sum_{k=0}^{n-1}f(T^k\omega)\to \mathbb E[f\mid\mathcal I](\omega) \quad\text{p.s.}, \] où \(\mathcal I\) est la \(\sigma\)-algèbre invariante. Si \(T\) est ergodique, la limite est \(\mathbb Ef\).

Preuve. Pour \(f=g-g\circ T\), les moyennes télescopent : \[ n^{-1}\sum_{k=0}^{n-1}(g-g\circ T)(T^k\omega) =\frac{g(\omega)-g(T^n\omega)}{n}\to0 \] dans le sens \(L^1\) tronqué approprié. Les fonctions générales de \(L^1\) s’obtiennent par approximation par une partie invariante plus des cobords, en utilisant l’inégalité ergodique maximale. La limite est \(T\)-invariante, donc égale à \(\mathbb E[f\mid\mathcal I]\). Si \(\mathcal I\) est triviale, cette quantité est la constante \(\mathbb Ef\).

Moyennes temporelles et information invariante

Le théorème ergodique transforme les moyennes temporelles longues en espérances conditionnelles sur l’information invariante.

33. 遍历定理

{% include probability-note-nav.html back_label="返回" prev_label="上一页" prev_url="/notes/probability-theory/recurrence-transience-invariant-measures/" next_label="下一页" next_url="/notes/probability-theory/renewal-random-walks/" %}

定义. 保测映射 \(T:(\Omega,\mathcal F,P)\to(\Omega,\mathcal F,P)\) 满足 \(P(T^{-1}A)=P(A)\)。若每个不变事件 \(A=T^{-1}A\) 的概率只能是 \(0\) 或 \(1\)，则称 \(T\) 遍历。

定理. 若 \(T\) 保测且 \(f\in L^1\)，则 \[ \frac1n\sum_{k=0}^{n-1}f(T^k\omega)\to \mathbb E[f\mid\mathcal I](\omega) \quad\text{几乎处处}, \] 其中 \(\mathcal I\) 是不变 \(\sigma\)-代数。若 \(T\) 遍历，则极限为 \(\mathbb Ef\)。

证明. 对 \(f=g-g\circ T\)，平均会望远镜相消： \[ n^{-1}\sum_{k=0}^{n-1}(g-g\circ T)(T^k\omega) =\frac{g(\omega)-g(T^n\omega)}{n}\to0 \] 在适当截断的 \(L^1\) 意义下成立。一般 \(L^1\) 函数通过“不变部分 + coboundary”逼近，并使用最大遍历不等式处理。极限是 \(T\)-不变的，因此等于 \(\mathbb E[f\mid\mathcal I]\)；若 \(\mathcal I\) 平凡，则它就是常数 \(\mathbb Ef\)。

时间平均与不变信息

遍历定理把长期时间平均转化为关于不变信息的条件期望。
