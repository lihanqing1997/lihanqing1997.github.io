Discrete-time martingales and submartingales

{% include probability-note-nav.html back_label="Back" prev_label="Last page" prev_url="/notes/probability-theory/filtrations-stopping-times/" next_label="Next page" next_url="/notes/probability-theory/doob-inequalities-martingale-convergence/" %}

This topic belongs to martingale theory. The guiding idea is to describe random evolution relative to information that grows over time. Here the focus is Discrete-time martingales and submartingales.

Start with the precise object. The definition states what the object is and what assumptions make the later formulas meaningful.

Definition. A discrete-time integrable adapted process \((M_n,\mathcal F_n)\) is a martingale if \(\mathbb E(M_{n+1}\mid\mathcal F_n)=M_n\). It is a submartingale if the conditional expectation is at least \(M_n\).

These assumptions are part of the mathematics. They tell us where each expression lives and prevent informal notation from hiding a real domain issue.

The result below is the fact one usually uses in calculations or proofs. Read it as the operational content of the definition.

Result. If \(M_n\) is a martingale and \(\tau\) is a bounded stopping time, then \(\mathbb EM_\tau=\mathbb EM_0\). If \(X\in L^1\), then \(M_n=\mathbb E(X\mid\mathcal F_n)\) is a martingale.

Proof.

The proof is easiest to read by separating the elementary case from the extension step.

For the conditional-expectation martingale, the tower property gives \(\mathbb E(M_{n+1}\mid\mathcal F_n)=\mathbb E(\mathbb E(X\mid\mathcal F_{n+1})\mid\mathcal F_n)=\mathbb E(X\mid\mathcal F_n)=M_n\). Bounded optional stopping follows by summing martingale differences stopped at \(\tau\).

Remarks

A martingale is a fair process relative to its own information. This fairness is conditional, not unconditional.
