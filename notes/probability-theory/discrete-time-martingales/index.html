Discrete-Time Martingales and Submartingales

{% include probability-note-nav.html back_label="Back" prev_label="Last page" prev_url="/notes/probability-theory/filtrations-stopping-times/" next_label="Next page" next_url="/notes/probability-theory/doob-inequalities-martingale-convergence/" %}

This page records the core mathematical facts for Discrete-Time Martingales and Submartingales.

Definition. A discrete-time integrable adapted process \((M_n,\mathcal F_n)\) is a martingale if \(\mathbb E(M_{n+1}\mid\mathcal F_n)=M_n\). It is a submartingale if the conditional expectation is at least \(M_n\).

Central result. If \(M_n\) is a martingale and \(\tau\) is a bounded stopping time, then \(\mathbb EM_\tau=\mathbb EM_0\). If \(X\in L^1\), then \(M_n=\mathbb E(X\mid\mathcal F_n)\) is a martingale.

Proof. For the conditional-expectation martingale, the tower property gives \(\mathbb E(M_{n+1}\mid\mathcal F_n)=\mathbb E(\mathbb E(X\mid\mathcal F_{n+1})\mid\mathcal F_n)=\mathbb E(X\mid\mathcal F_n)=M_n\). Bounded optional stopping follows by summing martingale differences stopped at \(\tau\).

How to read it

A martingale is a fair process relative to its own information. This fairness is conditional, not unconditional.

The point is not to memorize the displayed formulas separately, but to understand how the definition, the result, and the proof mechanism fit together.
