Conditional expectation

{% include probability-note-nav.html back_label="Back" prev_label="Last page" prev_url="/notes/probability-theory/conditional-probability-regular-conditional-distributions/" next_label="Next page" next_url="/notes/probability-theory/modes-of-convergence/" %}

This topic belongs to the measure-theoretic foundation of probability. The main purpose is to make the objects measurable before one starts computing. Here the focus is Conditional expectation.

Start with the precise object. The definition states what the object is and what assumptions make the later formulas meaningful.

Definition. For an integrable \(X\) and a sub-\(\sigma\)-field \(\mathcal G\), the conditional expectation \(\mathbb E(X\mid\mathcal G)\) is the \(\mathcal G\)-measurable integrable random variable \(Y\) such that \(\int_GY\,dP=\int_GX\,dP\) for all \(G\in\mathcal G\).

These assumptions are part of the mathematics. They tell us where each expression lives and prevent informal notation from hiding a real domain issue.

The result below is the fact one usually uses in calculations or proofs. Read it as the operational content of the definition.

Result. Conditional expectation is a projection: if \(X\in L^2\), then \(\mathbb E(X\mid\mathcal G)\) is the orthogonal projection of \(X\) onto the closed subspace \(L^2(\mathcal G)\). Also, \(\mathbb E\{\mathbb E(X\mid\mathcal G)\mid\mathcal H\}=\mathbb E(X\mid\mathcal H)\) whenever \(\mathcal H\subset\mathcal G\).

Proof.

The proof is easiest to read by separating the elementary case from the extension step.

Existence follows from the Radon-Nikodym theorem applied to the signed measure \(G\mapsto\int_GX\,dP\) on \(\mathcal G\). Uniqueness follows because two versions with the same integrals over all \(G\in\mathcal G\) must be equal almost surely. The projection property follows by testing against all \(\mathcal G\)-measurable square-integrable functions.

Remarks

Conditioning means replacing unavailable information by the best measurable summary relative to the information \(\mathcal G\). Martingales are built by repeatedly applying this projection.
