Applications of Itô formula

{% include probability-note-nav.html back_label="Back" prev_label="Last page" prev_url="/notes/probability-theory/ito-formula/" next_label="Next page" next_url="/notes/probability-theory/girsanov-theorem-change-of-measure/" %}

This topic belongs to stochastic calculus. Ordinary calculus must be modified because martingale increments have non-negligible quadratic variation. Here the focus is Applications of Itô formula.

Start with the precise object. The definition states what the object is and what assumptions make the later formulas meaningful.

Definition. Itô formula produces martingales, solves linear stochastic equations, and computes transformations. Standard choices are \(f(x)=x^2\), \(f(x)=e^{\theta x}\), \(f(x)=\log x\), and harmonic functions.

These assumptions are part of the mathematics. They tell us where each expression lives and prevent informal notation from hiding a real domain issue.

The result below is the fact one usually uses in calculations or proofs. Read it as the operational content of the definition.

Result. If \(u(t,x)\in C^{1,2}\), then \[du(t,B_t)=\left(\partial_tu+\frac12\Delta u\right)(t,B_t)dt+\nabla u(t,B_t)\,dB_t.\] Hence if \(u\) solves the heat equation, \(u(t,B_t)\) is a local martingale.

Proof.

The proof is easiest to read by separating the elementary case from the extension step.

Apply multidimensional Itô formula to the time-space process \((t,B_t)\). The drift term is exactly \(\partial_tu+\frac12\Delta u\). If that vanishes, only the stochastic integral remains, and stochastic integrals are local martingales.

Remarks

The usual strategy is: choose a function whose drift vanishes, apply Itô, then use optional stopping to compute probabilities or expectations.
