Expected Value

Expectation is integration with respect to probability: it is the average value predicted by the distribution.

Definition

  • For X ≥ 0, E X = ∫ X dP, possibly ∞.
  • For general X, write X = X+ - X-. E X is defined if at least one of E X+, E X- is finite.
  • If μX is the distribution of X, then E g(X) = ∫ g dμX whenever the integral is defined.

Key tools

  • Linearity: E(aX + bY) = aE X + bE Y for integrable variables.
  • Markov: for X ≥ 0, P(X ≥ a) ≤ E X / a.
  • Chebyshev: P(|X - E X| ≥ a) ≤ Var(X) / a2.

Durrett guide: Sections 1.4-1.6.

Espérance

L'espérance est l'intégrale par rapport à la probabilité : c'est la valeur moyenne prédite par la loi.

Définition

  • Si X ≥ 0, E X = ∫ X dP, éventuellement ∞.
  • Pour X général, écrire X = X+ - X-. E X est définie si au moins une des deux espérances est finie.
  • Si μX est la loi de X, alors E g(X) = ∫ g dμX quand l'intégrale est définie.

Outils clés

  • Linéarité : E(aX + bY) = aE X + bE Y pour les variables intégrables.
  • Markov : si X ≥ 0, P(X ≥ a) ≤ E X / a.
  • Chebyshev : P(|X - E X| ≥ a) ≤ Var(X) / a2.

Guide Durrett : sections 1.4-1.6.

期望

期望就是关于概率测度的积分，表示由分布给出的平均值。

定义

  • 若 X ≥ 0，则 E X = ∫ X dP，允许为 ∞。
  • 一般情形写 X = X+ - X-。若 E X+ 与 E X- 至少有一个有限，则 E X 有定义。
  • 若 μX 是 X 的分布，则在积分有定义时 E g(X) = ∫ g dμX。

关键工具

  • 线性性：对可积变量，E(aX + bY) = aE X + bE Y。
  • Markov 不等式：若 X ≥ 0，则 P(X ≥ a) ≤ E X / a。
  • Chebyshev 不等式：P(|X - E X| ≥ a) ≤ Var(X) / a2。
