Expected Value

{% include probability-note-nav.html back_label="Back" prev_label="Last page" prev_url="/notes/probability-theory/pushforward-measures-densities/" next_label="Next page" next_url="/notes/probability-theory/independence/" %}

Expectation is Lebesgue integration with respect to probability. It is linear because integration is linear, not because outcomes are equally likely.

Definition

For \(X\ge0\), define \(EX=\int_\Omega X\,dP\in[0,\infty]\). For general \(X\), write \(X^+=\max(X,0)\), \(X^-=\max(-X,0)\), so \(X=X^+-X^-\). If at least one of \(EX^+,EX^-\) is finite, define \(EX=EX^+-EX^-\). If \(E|X|<\infty\), then \(X\) is integrable.

Proposition: change of variables

If \(g:\mathbb R\to\mathbb R\) is Borel and the integral is defined, then

\[E[g(X)] = \int_{\mathbb R} g(x)\,\mu_X(dx).\]

Proof

For \(g=1_B\), both sides equal \(P(X\in B)=\mu_X(B)\). By linearity this holds for nonnegative simple \(g\). For nonnegative Borel \(g\), take simple \(g_n\uparrow g\) and apply monotone convergence on both sides. For general \(g\), apply the nonnegative case to \(g^+\) and \(g^-\).

Proposition: Markov and Chebyshev

If \(Y\ge0\) and \(a>0\), then \(P(Y\ge a)\le EY/a\). If \(X\) has finite variance and \(a>0\), then \(P(|X-EX|\ge a)\le \operatorname{Var}(X)/a^2\).

Proof

Since \(Y\ge a\,1_{\{Y\ge a\}}\), integrating gives \(EY\ge aP(Y\ge a)\). This is Markov's inequality.

Apply Markov to \(Y=(X-EX)^2\): \(P(|X-EX|\ge a)=P((X-EX)^2\ge a^2)\le E[(X-EX)^2]/a^2\).

Durrett guide: Sections 1.4-1.6.

Espérance

{% include probability-note-nav.html back_label="Retour" prev_label="Page précédente" prev_url="/notes/probability-theory/pushforward-measures-densities/" next_label="Page suivante" next_url="/notes/probability-theory/independence/" %}

L'espérance est l'intégration de Lebesgue par rapport à la probabilité. Elle est linéaire parce que l'intégrale est linéaire.

Définition

Pour \(X\ge0\), définir \(EX=\int_\Omega X\,dP\in[0,\infty]\). Pour \(X\) général, écrire \(X^+=\max(X,0)\), \(X^-=\max(-X,0)\), donc \(X=X^+-X^-\). Si au moins une des quantités \(EX^+,EX^-\) est finie, définir \(EX=EX^+-EX^-\). Si \(E|X|<\infty\), \(X\) est intégrable.

Proposition : changement de variable

Si \(g:\mathbb R\to\mathbb R\) est borélienne et si l'intégrale est définie, alors

\[E[g(X)] = \int_{\mathbb R} g(x)\,\mu_X(dx).\]

Preuve

Pour \(g=1_B\), les deux côtés valent \(P(X\in B)=\mu_X(B)\). Par linéarité, le résultat vaut pour les fonctions simples positives. Pour \(g\ge0\) borélienne, prendre des fonctions simples \(g_n\uparrow g\) et appliquer la convergence monotone des deux côtés. Pour \(g\) générale, appliquer ce résultat à \(g^+\) et \(g^-\).

Proposition : Markov et Chebyshev

Si \(Y\ge0\) et \(a>0\), alors \(P(Y\ge a)\le EY/a\). Si \(X\) a une variance finie et \(a>0\), alors \(P(|X-EX|\ge a)\le \operatorname{Var}(X)/a^2\).

Preuve

Comme \(Y\ge a\,1_{\{Y\ge a\}}\), l'intégration donne \(EY\ge aP(Y\ge a)\).

Appliquer Markov à \(Y=(X-EX)^2\) : \(P(|X-EX|\ge a)=P((X-EX)^2\ge a^2)\le E[(X-EX)^2]/a^2\).

Guide Durrett : sections 1.4-1.6.

期望

{% include probability-note-nav.html back_label="返回" prev_label="上一页" prev_url="/notes/probability-theory/pushforward-measures-densities/" next_label="下一页" next_url="/notes/probability-theory/independence/" %}

期望就是关于概率测度的 Lebesgue 积分。它的线性性来自积分的线性性。

定义

若 \(X\ge0\)，定义 \(EX=\int_\Omega X\,dP\in[0,\infty]\)。一般情形令 \(X^+=\max(X,0)\)、\(X^-=\max(-X,0)\)，则 \(X=X^+-X^-\)。若 \(EX^+\) 与 \(EX^-\) 至少一个有限，则定义 \(EX=EX^+-EX^-\)。若 \(E|X|<\infty\)，称 \(X\) 可积。

命题：换元公式

若 \(g:\mathbb R\to\mathbb R\) 是 Borel 函数且积分有定义，则

\[E[g(X)] = \int_{\mathbb R} g(x)\,\mu_X(dx).\]

证明

当 \(g=1_B\) 时，两边都等于 \(P(X\in B)=\mu_X(B)\)。由线性性可得结论对非负简单函数成立。对非负 Borel 函数 \(g\)，取简单函数 \(g_n\uparrow g\)，两边使用单调收敛定理。一般 \(g\) 分别作用于 \(g^+\) 和 \(g^-\)。

命题：Markov 与 Chebyshev 不等式

若 \(Y\ge0\) 且 \(a>0\)，则 \(P(Y\ge a)\le EY/a\)。若 \(X\) 方差有限且 \(a>0\)，则 \(P(|X-EX|\ge a)\le \operatorname{Var}(X)/a^2\)。

证明

因为 \(Y\ge a\,1_{\{Y\ge a\}}\)，积分后得 \(EY\ge aP(Y\ge a)\)。

把 Markov 不等式用于 \(Y=(X-EX)^2\)，得 \(P(|X-EX|\ge a)=P((X-EX)^2\ge a^2)\le E[(X-EX)^2]/a^2\)。

Durrett 参考：第 1.4-1.6 节。
