Lebesgue Integration and Expectation

{% include probability-note-nav.html back_label="Back" prev_label="Last page" prev_url="/notes/probability-theory/independence/" next_label="Next page" next_url="/notes/probability-theory/lp-spaces-inequalities-uniform-integrability/" %}

This page records the core mathematical facts for Lebesgue Integration and Expectation.

Definition. For \(X\ge0\), define \(\mathbb EX=\int_\Omega X\,dP\in[0,\infty]\). For general \(X\), write \(X=X^+-X^-\). If at least one of \(\mathbb EX^+,\mathbb EX^-\) is finite, define \(\mathbb EX=\mathbb EX^+-\mathbb EX^-\). If \(\mathbb E|X|\lt\infty\), then \(X\) is integrable.

Central result. If \(0\le X_n\uparrow X\), then \(\mathbb EX_n\uparrow\mathbb EX\). If \(|X_n|\le Y\) with \(\mathbb EY\lt\infty\) and \(X_n\to X\) almost surely, then \(\mathbb EX_n\to\mathbb EX\).

Proof. The first assertion is the monotone convergence theorem, proved by approximating \(X\) from below by simple functions and using countable additivity. For dominated convergence, apply Fatou's lemma to \(Y+X_n\) and \(Y-X_n\) to obtain the two opposite inequalities for the limsup and liminf of the expectations.

How to read it

Expectation is linear integration, not averaging over equally likely outcomes. The convergence theorems are the main tools for exchanging limits and expectations.

The point is not to memorize the displayed formulas separately, but to understand how the definition, the result, and the proof mechanism fit together.
