Probability Spaces

A probability space is the formal place where random experiments live.

Core objects

  • Sample space: Ω, the set of possible outcomes.
  • Events: a sigma-field F of subsets of Ω, closed under complements and countable unions.
  • Probability: P : F → [0,1], with P(Ω) = 1 and countable additivity on disjoint events.

Pivotal facts

  • Monotonicity: if A ⊆ B, then P(A) ≤ P(B).
  • Continuity from below: if An ↑ A, then P(An) ↑ P(A).
  • Continuity from above: if An ↓ A, then P(An) ↓ P(A).

Durrett guide: Section 1.1.

Espaces probabilisés

Un espace probabilisé est le cadre formel dans lequel vivent les expériences aléatoires.

Objets centraux

  • Espace d'états : Ω, l'ensemble des issues possibles.
  • Événements : une tribu F de parties de Ω, stable par complémentaire et unions dénombrables.
  • Probabilité : P : F → [0,1], avec P(Ω) = 1 et additivité dénombrable sur les événements disjoints.

Faits pivots

  • Monotonie : si A ⊆ B, alors P(A) ≤ P(B).
  • Continuité croissante : si An ↑ A, alors P(An) ↑ P(A).
  • Continuité décroissante : si An ↓ A, alors P(An) ↓ P(A).

Guide Durrett : section 1.1.

概率空间

概率空间是描述随机试验的基本框架。

核心对象

  • 样本空间：Ω，所有可能结果的集合。
  • 事件：Ω 的子集构成的 σ-代数 F，对补集和可数并封闭。
  • 概率：P : F → [0,1]，满足 P(Ω) = 1，且对两两不交事件可数可加。

关键性质

  • 单调性：若 A ⊆ B，则 P(A) ≤ P(B)。
  • 下连续性：若 An ↑ A，则 P(An) ↑ P(A)。
  • 上连续性：若 An ↓ A，则 P(An) ↓ P(A)。

Durrett 参考：第 1.1 节。
