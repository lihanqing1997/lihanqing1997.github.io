A probability space is the measure-theoretic stage on which random objects live. The main point is that probabilities are assigned only to measurable events, and almost-sure statements are statements modulo null sets.

Events and probability measures

The axioms are the same as measure theory, normalized to total mass one.

Definition. A probability space is a triple \((\Omega,\mathcal F,P)\), where \(\mathcal F\) is a \(\sigma\)-field on \(\Omega\) and \(P:\mathcal F\to[0,1]\) is countably additive with \(P(\Omega)=1\). Events are elements of \(\mathcal F\). A property holds almost surely if it fails on a set \(N\in\mathcal F\) with \(P(N)=0\).

Continuity of probability

Countable additivity is most often used through continuity along monotone set limits.

Theorem. If \(A_n\uparrow A\), then \(P(A_n)\uparrow P(A)\). If \(A_n\downarrow A\), then \(P(A_n)\downarrow P(A)\).

Proof. For \(A_n\uparrow A\), write \(B_1=A_1\) and \(B_n=A_n\setminus A_{n-1}\) for \(n\ge2\). The \(B_n\) are disjoint, \(A_n=\bigcup_{k\le n}B_k\), and \(A=\bigcup_{k\ge1}B_k\), so countable additivity gives \(P(A_n)=\sum_{k\le n}P(B_k)\to\sum_{k\ge1}P(B_k)=P(A)\). For \(A_n\downarrow A\), apply the increasing result to \(A_n^c\uparrow A^c\) and use \(P(A_n)=1-P(A_n^c)\).

Null sets and completion

Almost-sure equality is harmless only when the measurable structure has enough null subsets.

Proposition. The completion of \((\Omega,\mathcal F,P)\) is obtained by adding every subset of every \(P\)-null set to \(\mathcal F\), and extending \(P\) by assigning those subsets probability zero.

Proof. Let \(\overline{\mathcal F}\) consist of sets \(A\cup N\), where \(A\in\mathcal F\) and \(N\subseteq Z\) for some \(Z\in\mathcal F\) with \(P(Z)=0\). This class is a \(\sigma\)-field because countable unions of null subsets are contained in countable unions of null sets. Define \(\overline P(A\cup N)=P(A)\). If the same set has two representations, their symmetric difference is contained in a null set, so the value is well-defined.

Remark. Completeness is convenient, but not automatic. Many canonical spaces first appear with a Borel \(\sigma\)-field and are completed only after a probability law is fixed.
