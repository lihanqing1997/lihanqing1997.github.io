1. Probability spaces

{% include probability-note-nav.html back_label="Back" next_label="Next page" next_url="/notes/probability-theory/measures-extension-monotone-classes/" %}

Definition. A probability space is a triple \((\Omega,\mathcal F,P)\). Here \(\mathcal F\) is a \(\sigma\)-field on \(\Omega\): \(\Omega\in\mathcal F\), \(A\in\mathcal F\) implies \(A^c\in\mathcal F\), and \((A_n)_{n\ge1}\subset\mathcal F\) implies \(\bigcup_{n=1}^\infty A_n\in\mathcal F\). The map \(P:\mathcal F\to[0,1]\) satisfies \(P(\Omega)=1\) and countable additivity: \[ A_i\cap A_j=\varnothing\ (i\ne j) \quad\Longrightarrow\quad P\left(\bigcup_{n=1}^\infty A_n\right)=\sum_{n=1}^\infty P(A_n). \] Events are elements of \(\mathcal F\). A statement holds almost surely if the set on which it fails has probability zero.

Proposition. If \(A_n\uparrow A\), meaning \(A_1\subset A_2\subset\cdots\) and \(A=\bigcup_n A_n\), then \(P(A_n)\uparrow P(A)\). If \(A_n\downarrow A\), meaning \(A_1\supset A_2\supset\cdots\) and \(A=\bigcap_n A_n\), then \(P(A_n)\downarrow P(A)\).

Proof. For the increasing case, set \(B_1=A_1\) and \(B_n=A_n\setminus A_{n-1}\) for \(n\ge2\). The sets \(B_n\) are disjoint, \(A_n=\bigcup_{k=1}^nB_k\), and \(A=\bigcup_{k=1}^\infty B_k\). Hence \[ P(A_n)=\sum_{k=1}^nP(B_k),\qquad P(A)=\sum_{k=1}^\infty P(B_k), \] so \(P(A_n)\to P(A)\). For the decreasing case, \(A_n^c\uparrow A^c\). By the increasing case, \(P(A_n^c)\uparrow P(A^c)\), and therefore \(P(A_n)=1-P(A_n^c)\downarrow 1-P(A^c)=P(A)\).

Measurability before probabilities

The \(\sigma\)-field is the domain of \(P\). Before writing \(P(X\in B)\), one must know that \(X^{-1}(B)\in\mathcal F\).

1. Espaces probabilisés

{% include probability-note-nav.html back_label="Retour" next_label="Page suivante" next_url="/notes/probability-theory/measures-extension-monotone-classes/" %}

Définition. Un espace probabilisé est un triplet \((\Omega,\mathcal F,P)\). Ici \(\mathcal F\) est une \(\sigma\)-algèbre sur \(\Omega\) : \(\Omega\in\mathcal F\), \(A\in\mathcal F\) entraîne \(A^c\in\mathcal F\), et \((A_n)_{n\ge1}\subset\mathcal F\) entraîne \(\bigcup_{n=1}^\infty A_n\in\mathcal F\). L’application \(P:\mathcal F\to[0,1]\) vérifie \(P(\Omega)=1\) et l’additivité dénombrable : \[ A_i\cap A_j=\varnothing\ (i\ne j) \quad\Longrightarrow\quad P\left(\bigcup_{n=1}^\infty A_n\right)=\sum_{n=1}^\infty P(A_n). \] Les événements sont les éléments de \(\mathcal F\). Une propriété est vraie presque sûrement si l’ensemble où elle échoue est de probabilité nulle.

Proposition. Si \(A_n\uparrow A\), c’est-à-dire \(A_1\subset A_2\subset\cdots\) et \(A=\bigcup_n A_n\), alors \(P(A_n)\uparrow P(A)\). Si \(A_n\downarrow A\), c’est-à-dire \(A_1\supset A_2\supset\cdots\) et \(A=\bigcap_n A_n\), alors \(P(A_n)\downarrow P(A)\).

Preuve. Dans le cas croissant, posons \(B_1=A_1\) et \(B_n=A_n\setminus A_{n-1}\) pour \(n\ge2\). Les ensembles \(B_n\) sont disjoints, \(A_n=\bigcup_{k=1}^nB_k\), et \(A=\bigcup_{k=1}^\infty B_k\). Donc \[ P(A_n)=\sum_{k=1}^nP(B_k),\qquad P(A)=\sum_{k=1}^\infty P(B_k), \] d’où \(P(A_n)\to P(A)\). Dans le cas décroissant, \(A_n^c\uparrow A^c\). Le cas croissant donne \(P(A_n^c)\uparrow P(A^c)\), donc \(P(A_n)=1-P(A_n^c)\downarrow 1-P(A^c)=P(A)\).

Mesurabilité avant probabilités

La \(\sigma\)-algèbre est le domaine de \(P\). Avant d’écrire \(P(X\in B)\), il faut savoir que \(X^{-1}(B)\in\mathcal F\).

1. 概率空间

{% include probability-note-nav.html back_label="返回" next_label="下一页" next_url="/notes/probability-theory/measures-extension-monotone-classes/" %}

定义. 概率空间是三元组 \((\Omega,\mathcal F,P)\)。其中 \(\mathcal F\) 是 \(\Omega\) 上的 \(\sigma\)-代数：\(\Omega\in\mathcal F\)，若 \(A\in\mathcal F\) 则 \(A^c\in\mathcal F\)，并且若 \((A_n)_{n\ge1}\subset\mathcal F\)，则 \(\bigcup_{n=1}^\infty A_n\in\mathcal F\)。映射 \(P:\mathcal F\to[0,1]\) 满足 \(P(\Omega)=1\)，并满足可列可加性： \[ A_i\cap A_j=\varnothing\ (i\ne j) \quad\Longrightarrow\quad P\left(\bigcup_{n=1}^\infty A_n\right)=\sum_{n=1}^\infty P(A_n). \] 事件是 \(\mathcal F\) 的元素。若一个命题失败的集合概率为零，则称它几乎处处成立。

命题. 若 \(A_n\uparrow A\)，即 \(A_1\subset A_2\subset\cdots\) 且 \(A=\bigcup_n A_n\)，则 \(P(A_n)\uparrow P(A)\)。若 \(A_n\downarrow A\)，即 \(A_1\supset A_2\supset\cdots\) 且 \(A=\bigcap_n A_n\)，则 \(P(A_n)\downarrow P(A)\)。

证明. 先看递增情形。令 \(B_1=A_1\)，且对 \(n\ge2\) 令 \(B_n=A_n\setminus A_{n-1}\)。这些 \(B_n\) 两两不交，且 \(A_n=\bigcup_{k=1}^nB_k\)、\(A=\bigcup_{k=1}^\infty B_k\)。因此 \[ P(A_n)=\sum_{k=1}^nP(B_k),\qquad P(A)=\sum_{k=1}^\infty P(B_k), \] 所以 \(P(A_n)\to P(A)\)。递减情形由补集得到：\(A_n^c\uparrow A^c\)。由递增情形 \(P(A_n^c)\uparrow P(A^c)\)，从而 \(P(A_n)=1-P(A_n^c)\downarrow 1-P(A^c)=P(A)\)。

概率之前的可测性

\(\sigma\)-代数是 \(P\) 的定义域。写下 \(P(X\in B)\) 之前，必须知道 \(X^{-1}(B)\in\mathcal F\)。
