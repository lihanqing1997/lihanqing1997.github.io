2. Cosets and quotient groups

{% include note-topic-nav.html back_label="Back" back_url="/notes/abstract-algebra/" prev_label="Last page" prev_url="/notes/abstract-algebra/groups-subgroups-and-homomorphisms/" next_label="Next page" next_url="/notes/abstract-algebra/isomorphism-theorems/" %}

Cosets and quotient groups records the objects, main results, proof mechanisms, and boundary cases needed for the abstract algebra note.

Definition

Definition. Given a subgroup \(H\le G\), left cosets are sets \(gH\). If \(N\triangleleft G\) is normal, cosets multiply by \((gN)(hN)=ghN\), forming the quotient group \(G/N\).

Proposition

Proposition. Cosets partition \(G\), and if \(G\) is finite then \(|G|=|H|[G:H]\). Quotient multiplication is well-defined exactly because normality controls conjugation.

Proof

Proof. Equivalence by \(g\sim h\) when \(g^{-1}h\in H\) gives the partition. For quotients, changing representatives introduces elements of \(N\), which normality moves across products.

Remark

Remark. Quotients are how algebra turns an equivalence relation into a new object of the same kind. This is the basic construction behind modding out relations.
