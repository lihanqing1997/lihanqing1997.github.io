4. Naive set theory and paradoxes

{% include note-topic-nav.html back_label="Back" back_url="/notes/foundations-of-mathematics/" prev_label="Last page" prev_url="/notes/foundations-of-mathematics/dedekind-cantor-and-arithmetization/" next_label="Next page" next_url="/notes/foundations-of-mathematics/zermelo-fraenkel-set-theory-and-choice/" %}

Naive set theory gave mathematics a universal language, but unrestricted set formation produced contradictions. This forced axiomatization.

Naive comprehension

Definition. Unrestricted comprehension would allow a set \(\{x:P(x)\}\) for every property \(P\). This is too strong because it permits self-reference.

Russell paradox

Theorem. Let \(R=\{x:x\notin x\}\). Then \(R\in R\) iff \(R\notin R\), a contradiction. Thus unrestricted comprehension is inconsistent.

Proper classes

Remark. Some collections, such as the collection of all sets, are too large to be sets. Modern foundations restrict formation rules or distinguish sets from classes.

Effect on analysis

Remark. Analysis uses sets of functions, null sets, product spaces, and measures. Axiomatic set theory gives controlled rules for forming them.
