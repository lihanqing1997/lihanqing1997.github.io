9. Modern foundations and structural viewpoints

{% include note-topic-nav.html back_label="Back" back_url="/notes/foundations-of-mathematics/" prev_label="Last page" prev_url="/notes/foundations-of-mathematics/continuum-problem-and-independence/" %}

Modern mathematics uses several foundational languages. Set theory remains standard for analysis, while category theory and type theory clarify structure and formal proof.

Structural viewpoint

Remark. All complete ordered fields are isomorphic, so the real line is characterized by structure rather than by one privileged construction.

Category theory

Definition. A category consists of objects, morphisms, identity morphisms, and associative composition. It emphasizes maps and universal properties.

Type theory

Remark. Type theory supports machine-checked mathematics in proof assistants. It gives a constructive and syntactic foundation different from classical set theory.

Convention for these notes

Proposition. The notes use ordinary classical mathematics in ZFC unless stated otherwise. This is the standard background for real analysis, measure theory, and functional analysis.
