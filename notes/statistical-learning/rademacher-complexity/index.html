7. Rademacher complexity

{% include note-topic-nav.html back_label="Back" back_url="/notes/statistical-learning/" prev_label="Last page" prev_url="/notes/statistical-learning/vc-dimension/" next_label="Next page" next_url="/notes/statistical-learning/covering-numbers-and-entropy/" %}

Rademacher complexity records the objects, main results, proof mechanisms, and boundary cases needed for the statistical learning note.

Definition

Definition. Rademacher complexity measures how well a function class correlates with random signs on the observed sample.

Proposition

Proposition. Expected uniform deviations of empirical means are bounded by symmetrized Rademacher complexity, up to constants.

Proof

Proof. Introduce an independent ghost sample, symmetrize the difference between empirical and expected means, and condition on data to obtain random-sign averages.

Remark

Remark. Rademacher complexity adapts to the geometry of real-valued classes more flexibly than pure counting dimensions.
