1. Supervised learning setup

{% include note-topic-nav.html back_label="Back" back_url="/notes/statistical-learning/" next_label="Next page" next_url="/notes/statistical-learning/bayes-rules-and-excess-risk/" %}

Supervised learning setup records the objects, main results, proof mechanisms, and boundary cases needed for the statistical learning note.

Definition

Definition. Supervised learning observes training data \((X_i,Y_i)\) and chooses a prediction rule \(f\) from inputs to outputs. A loss \(\ell(f(X),Y)\) measures prediction error.

Proposition

Proposition. The population risk \(R(f)=E[\ell(f(X),Y)]\) is the target, while empirical risk \(R_n(f)=n^{-1}\sum_i\ell(f(X_i),Y_i)\) is observable. Learning is the problem of controlling the gap.

Proof

Proof. Decompose excess risk into estimation error, approximation error, and optimization error. Each component is controlled by different assumptions and methods.

Remark

Remark. Statistical learning is not only model fitting; it is risk control under finite data and model restrictions.
