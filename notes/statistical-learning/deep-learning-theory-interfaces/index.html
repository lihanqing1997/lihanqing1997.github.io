18. Deep learning theory interfaces

{% include note-topic-nav.html back_label="Back" back_url="/notes/statistical-learning/" prev_label="Last page" prev_url="/notes/statistical-learning/boosting/" next_label="Next page" next_url="/notes/statistical-learning/distribution-shift-and-domain-adaptation/" %}

Deep learning theory is not a single theorem. It is a collection of interfaces between approximation, optimization, generalization, architecture, and data geometry.

Approximation by depth

Depth can represent certain compositional functions more efficiently than shallow representations.

Definition. A depth-\(L\) neural network composes affine maps and nonlinear activations: \[ f(x)=W_L\sigma(W_{L-1}\sigma(\cdots\sigma(W_1x+b_1)\cdots)+b_{L-1})+b_L. \]

Proposition. For piecewise-linear activations such as ReLU, the realized function is piecewise affine on a polyhedral partition of input space.

Proof. On any region where the sign pattern of all pre-activations is fixed, each ReLU is either the identity or zero. The network then reduces to a composition of affine maps, hence is affine on that region. The sign constraints define polyhedral regions.

Norm-based generalization

Parameter count alone is too crude; norm and margin information are more informative.

Proposition. For a network class with controlled product of layer norms and Lipschitz activation, Rademacher complexity bounds scale with the norm product and the input radius, rather than only with the number of parameters.

Proof. Apply the linear-class Rademacher bound to the first layer, then use contraction through each Lipschitz activation and linear map. Each layer contributes its operator-norm or appropriate matrix-norm factor. The resulting bound tracks the product of norms.

Optimization landscape

Nonconvexity does not by itself imply failure of gradient methods.

Remark. Overparameterized networks can have many global minimizers and connected low-loss regions. Theory often studies simplified regimes such as neural tangent kernels, mean-field limits, matrix factorization analogues, or interpolation with stability.

Interpolation and double descent

Modern predictors can fit training data exactly while still having controlled test risk.

Definition. Interpolation means \(P_n\ell_{\hat f}=0\) or nearly zero. Double descent describes test risk that decreases, peaks near an interpolation threshold, and then decreases again as overparameterization grows.

Remark. The theoretical issue is to identify the implicit bias of the algorithm among interpolating solutions. Minimum-norm interpolation, margin maximization, and stability of gradient flow are three common mathematical descriptions.
