6. VC dimension

{% include note-topic-nav.html back_label="Back" back_url="/notes/statistical-learning/" prev_label="Last page" prev_url="/notes/statistical-learning/bias-variance-and-approximation/" next_label="Next page" next_url="/notes/statistical-learning/rademacher-complexity/" %}

VC dimension is the fundamental combinatorial measure for binary classification classes. It controls how many labelings the class can realize and therefore how severe overfitting can be.

Shattering and growth

A class is complex if it can realize all dichotomies on large finite sets.

Definition. For a class \(\mathcal C\) of subsets of \(\mathcal X\), the growth function is \[ \Pi_{\mathcal C}(n)=\max_{x_1,\ldots,x_n}\left|\{(1_C(x_1),\ldots,1_C(x_n)):C\in\mathcal C\}\right|. \] The VC dimension is the largest \(d\) such that some \(d\)-point set is shattered, meaning \(\Pi_{\mathcal C}(d)=2^d\).

Example. Intervals on \(\mathbb R\) have VC dimension \(2\). Two points can be shattered; three ordered points cannot realize the labeling \(1,0,1\).

Sauer-Shelah lemma

Finite VC dimension forces polynomial growth of the number of labelings.

Theorem. If \(\operatorname{VC}(\mathcal C)=d<\infty\), then for \(n\ge d\), \[ \Pi_{\mathcal C}(n)\le \sum_{j=0}^d {n\choose j}\le \left(\frac{en}{d}\right)^d. \]

Proof. The Sauer-Shelah induction counts traces on \(n\) points by separating sets according to whether their trace changes when the last point is removed. The changing part corresponds to subsets shattered by the class on the first \(n-1\) points, so the recursion is bounded by the binomial recursion \(\sum_{j\le d}{n-1\choose j}+\sum_{j\le d-1}{n-1\choose j}=\sum_{j\le d}{n\choose j}\). The final inequality is the standard binomial bound.

VC inequality

VC classes satisfy uniform convergence for binary losses.

Theorem. For a VC class \(\mathcal C\) of dimension \(d\), there is a universal constant \(K\) such that with probability at least \(1-\delta\), \[ \sup_{C\in\mathcal C}|P_nC-PC| \le K\sqrt{\frac{d\log(en/d)+\log(1/\delta)}{n}} \] for \(n\ge d\).

Proof. Symmetrization reduces the expected supremum to random signs on the sample. Conditional on the sample, only \(\Pi_{\mathcal C}(n)\) distinct label vectors occur. A finite-class maximal inequality and Sauer-Shelah bound the expected supremum by the displayed complexity scale. A bounded-differences concentration step converts expectation to a high-probability statement.

Examples and invariances

VC dimension is stable under many geometric transformations but sensitive to algebraic richness.

Proposition. Halfspaces in \(\mathbb R^p\) have VC dimension \(p+1\).

Proof. Any \(p+1\) affinely independent points can be shattered by separating selected vertices of a simplex from the remaining vertices. Conversely, Radon's theorem says any \(p+2\) points can be partitioned into two subsets with intersecting convex hulls, so no hyperplane can separate those two labels. Hence the VC dimension is \(p+1\).
