Approximate dynamic programming studies how errors in Bellman backups, projections, simulations, and greedy steps propagate through control algorithms.

Approximate value iteration

Each Bellman backup may be followed by approximation error.

Definition. Approximate value iteration has the form \[ v_{k+1}=T_*v_k+\varepsilon_k. \]

Proposition. If \(\|\varepsilon_k\|_\infty\le\varepsilon\), then \[ \limsup_{k\to\infty}\|v_k-v_*\|_\infty\le\frac{\varepsilon}{1-\gamma}. \]

Proof. Use contraction: \[ \|v_{k+1}-v_*\|_\infty\le\gamma\|v_k-v_*\|_\infty+\varepsilon. \] Iterating this scalar recursion and letting \(k\to\infty\) gives the bound.

Approximate greedy policies

Control loss depends on both value approximation and greedy-step error.

Definition. A policy \(\pi\) is \(\eta\)-greedy with respect to \(v\) if \[ T_\pi v\ge T_*v-\eta\mathbf 1. \]

Theorem. If \(\pi\) is \(\eta\)-greedy with respect to \(v\), then \[ \|v_*-v_\pi\|_\infty \le \frac{2\gamma}{1-\gamma}\|v-v_*\|_\infty+\frac{\eta}{1-\gamma}. \]

Proof. Compare \(T_*v_*\) and \(T_\pi v_\pi\), adding and subtracting \(T_*v\) and \(T_\pi v\). The approximation terms are bounded by \(\gamma\|v-v_*\|_\infty\), the greedy error contributes \(\eta\), and the remaining \(T_\pi(v-v_\pi)\) term is absorbed by \(1-\gamma\).

Simulation error

Sampled Bellman backups introduce statistical error in addition to approximation error.

Remark. Fitted value iteration and fitted Q-iteration replace conditional expectations by regression estimates. Their analysis combines supervised-learning generalization error with Bellman error propagation.

Error norms

The norm in which approximation is small matters for control.

Remark. Sup-norm error gives clean worst-case control guarantees. \(L^2\)-type errors are often easier statistically but require concentrability coefficients to translate behavior-distribution accuracy into target-policy performance.
