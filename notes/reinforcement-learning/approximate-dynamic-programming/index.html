Approximate dynamic programming records the objects, main results, proof mechanisms, and boundary cases needed for the reinforcement learning note.

Definition

Definition. Approximate dynamic programming studies Bellman updates combined with approximation, sampling, or restricted policy classes.

Proposition

Proposition. Performance bounds often control policy loss by Bellman residuals, approximation errors, and concentrability coefficients comparing state distributions.

Proof

Proof. Iterate one-step error inequalities through the discounted dynamics. Distribution mismatch appears when errors are measured under one state distribution but performance under another.

Remark

Remark. Approximate dynamic programming is where statistical learning, optimization, and dynamic programming fully interact.
