Off-policy evaluation records the objects, main results, proof mechanisms, and boundary cases needed for the reinforcement learning note.

Definition

Definition. Off-policy evaluation estimates the value of a target policy using data generated by another behavior policy.

Proposition

Proposition. Importance sampling identifies target-policy returns when the target trajectory distribution is absolutely continuous with respect to the behavior distribution, but variance can grow rapidly with horizon.

Proof

Proof. Write the target trajectory expectation as a behavior expectation weighted by the likelihood ratio of target to behavior action probabilities. Products over time create variance.

Remark

Remark. Off-policy evaluation is a causal inference problem with sequential treatments and known or estimated propensities.
