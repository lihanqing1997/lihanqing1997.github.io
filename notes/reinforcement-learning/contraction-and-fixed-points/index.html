Discounted dynamic programming works because Bellman operators are contractions. This fixed-point geometry supplies existence, uniqueness, convergence of iteration, and perturbation bounds.

Banach fixed-point theorem

The contraction theorem is the basic analytic tool behind value iteration.

Theorem. Let \((X,d)\) be complete and \(T:X\to X\) satisfy \(d(Tx,Ty)\le\gamma d(x,y)\) for some \(0\le\gamma<1\). Then \(T\) has a unique fixed point \(x^*\), and \(T^nx\to x^*\) for every \(x\).

Proof. The iterates satisfy \(d(x_{n+1},x_n)\le\gamma^nd(x_1,x_0)\), so \((x_n)\) is Cauchy and converges to some \(x^*\). Continuity of \(T\) gives \(Tx^*=x^*\). If \(y^*\) is another fixed point, \(d(x^*,y^*)\le\gamma d(x^*,y^*)\), hence \(x^*=y^*\).

Bellman contractions

Bellman operators inherit contraction from discounting and Markov averaging.

Proposition. For bounded value functions, \[ \|T_\pi u-T_\pi v\|_\infty\le\gamma\|u-v\|_\infty, \qquad \|T_*u-T_*v\|_\infty\le\gamma\|u-v\|_\infty. \]

Proof. For \(T_\pi\), Markov averaging cannot increase the sup norm and the discount contributes \(\gamma\). For \(T_*\), use the inequality \(|\sup_a f_a-\sup_a g_a|\le\sup_a|f_a-g_a|\), then apply the same Markov averaging bound.

Residual bounds

A small Bellman residual implies closeness to the fixed point.

Proposition. If \(T\) is a \(\gamma\)-contraction with fixed point \(v^*\), then \[ \|v-v^*\|\le\frac{1}{1-\gamma}\|v-Tv\|. \]

Proof. Use \[ \|v-v^*\|\le\|v-Tv\|+\|Tv-Tv^*\| \le\|v-Tv\|+\gamma\|v-v^*\|. \] Move the last term to the left.

Perturbed fixed points

Approximate dynamic programming repeatedly applies operators with error; perturbation bounds quantify error accumulation.

Corollary. If \(Tv^*=v^*\), \(\tilde T\tilde v=\tilde v\), and \(\|Tv-\tilde Tv\|\le\varepsilon\) for all \(v\), then \[ \|v^*-\tilde v\|\le\frac{\varepsilon}{1-\gamma}. \]

Proof. Apply the residual bound for \(T\) at \(v=\tilde v\). Since \(T\tilde v-\tilde v=T\tilde v-\tilde T\tilde v\), the residual is at most \(\varepsilon\).
