Eligibility traces interpolate between one-step TD and Monte Carlo returns. They assign credit backward through recently visited states or features.

Lambda returns

The \(\lambda\)-return averages \(n\)-step returns geometrically.

Definition. The \(n\)-step return is \[ G_t^{(n)}=R_{t+1}+\gamma R_{t+2}+\cdots+\gamma^{n-1}R_{t+n}+\gamma^nv(S_{t+n}). \] The forward-view \(\lambda\)-return is \[ G_t^\lambda=(1-\lambda)\sum_{n=1}^\infty\lambda^{n-1}G_t^{(n)},\qquad 0\le\lambda<1. \]

Proposition. When \(\lambda=0\), \(G_t^\lambda\) is the one-step TD target. As \(\lambda\uparrow1\), it approaches the Monte Carlo return when the discounted return is finite.

Proof. At \(\lambda=0\), only \(n=1\) has positive weight. As \(\lambda\uparrow1\), the geometric weighting shifts mass to longer horizons; with bounded discounted returns, Abelian convergence gives the full return limit.

Backward eligibility traces

The backward view computes the same credit assignment online.

Definition. For tabular TD, the accumulating trace is \[ e_t(s)=\gamma\lambda e_{t-1}(s)+\mathbf 1\{S_t=s\}. \] The update is \[ v_{t+1}(s)=v_t(s)+\alpha_t\delta_t e_t(s). \]

Remark. The trace marks which previous states are eligible to receive the current TD error. Larger \(\lambda\) makes credit assignment more persistent.

Forward-backward equivalence

For fixed value estimates within an episode, the forward and backward views agree.

Theorem. In the episodic tabular case with offline updates, summing TD errors with eligibility traces gives the same total update as using forward-view \(\lambda\)-returns.

Proof. Expand the backward update as a double sum over times and traces: \[ \sum_t\delta_te_t(s)=\sum_{k:S_k=s}\sum_{t\ge k}(\gamma\lambda)^{t-k}\delta_t. \] The inner discounted sum telescopes into \(G_k^\lambda-v(S_k)\), which is precisely the forward-view error for visit \(k\).

Bias-variance continuum

The trace parameter tunes between bootstrapping and full returns.

Remark. Small \(\lambda\) uses more bootstrapping and often lower variance; large \(\lambda\) uses longer returns and often lower bootstrap bias. The best value depends on mixing, reward noise, and approximation error.
