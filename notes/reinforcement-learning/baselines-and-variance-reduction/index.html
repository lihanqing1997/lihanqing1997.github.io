13. Baselines and variance reduction

{% include note-topic-nav.html back_label="Back" back_url="/notes/reinforcement-learning/" prev_label="Last page" prev_url="/notes/reinforcement-learning/policy-gradient-theorem/" next_label="Next page" next_url="/notes/reinforcement-learning/actor-critic-methods/" %}

Baselines and variance reduction records the objects, main results, proof mechanisms, and boundary cases needed for the reinforcement learning note.

Definition

Definition. A baseline is a quantity subtracted from the return or action value in policy-gradient estimation without changing the expected gradient.

Proposition

Proposition. Any state-dependent baseline \(b(S)\) independent of the sampled action has zero contribution: \(E[\nabla\log\pi(A\mid S)b(S)\mid S]=0\).

Proof

Proof. Condition on the state and pull out \(b(S)\). The remaining expectation is the gradient of \( \sum_a\pi(a\mid S)\), which is the gradient of one.

Remark

Remark. Baselines do not change the target gradient, but they can dramatically change estimator variance.
