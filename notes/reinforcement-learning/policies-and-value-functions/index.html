2. Policies and value functions

{% include note-topic-nav.html back_label="Back" back_url="/notes/reinforcement-learning/" prev_label="Last page" prev_url="/notes/reinforcement-learning/markov-decision-processes/" next_label="Next page" next_url="/notes/reinforcement-learning/bellman-expectation-equations/" %}

Policies and value functions records the objects, main results, proof mechanisms, and boundary cases needed for the reinforcement learning note.

Definition

Definition. A policy maps states or histories to action distributions. The value function \(V^\pi\) is expected return from a state under policy \(\pi\), and \(Q^\pi\) is expected return after a state-action pair.

Proposition

Proposition. The state-value and action-value functions determine each other through the policy and transition kernel. Greedy improvement is defined using \(Q^\pi\).

Proof

Proof. Write the first-step decomposition for \(Q^\pi\), then average over actions to obtain \(V^\pi\). Conversely, combine immediate reward and next-state value to obtain \(Q^\pi\).

Remark

Remark. Value functions are summaries of long-run consequence. They turn sequential decision making into fixed-point equations.
