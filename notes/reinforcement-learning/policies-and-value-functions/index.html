2. Policies and value functions

{% include note-topic-nav.html back_label="Back" back_url="/notes/reinforcement-learning/" prev_label="Last page" prev_url="/notes/reinforcement-learning/markov-decision-processes/" next_label="Next page" next_url="/notes/reinforcement-learning/bellman-expectation-equations/" %}

Policies define how actions are chosen, and value functions summarize the long-run consequences of those choices. The whole Bellman theory is a calculus for these functions.

State and action values

State values average over the policy's first action; action values keep the first action fixed.

Definition. For a policy \(\pi\), \[ v_\pi(s)=\mathbb E_\pi(G_0\mid S_0=s), \qquad q_\pi(s,a)=\mathbb E_\pi(G_0\mid S_0=s,A_0=a). \] The optimal values are \(v_*(s)=\sup_\pi v_\pi(s)\) and \(q_*(s,a)=\sup_\pi q_\pi(s,a)\).

Proposition. The state and action values are related by \[ v_\pi(s)=\int q_\pi(s,a)\pi(\mathrm{d}a\mid s), \quad q_\pi(s,a)=r(s,a)+\gamma\int v_\pi(s')P(\mathrm{d}s'\mid s,a). \]

Proof. The first identity conditions on \(A_0\sim\pi(\cdot\mid s)\). The second identity splits the return into \(R_1+\gamma G_1\) and conditions on \(S_1\) after fixing \((S_0,A_0)=(s,a)\).

Stationary deterministic policies

For discounted finite MDPs, optimizing over all history-dependent randomized policies is no better than optimizing over stationary deterministic policies.

Theorem. In a finite discounted MDP, there exists an optimal stationary deterministic policy \(\pi_*\) satisfying \[ \pi_*(s)\in\operatorname*{argmax}_{a\in\mathcal A}q_*(s,a). \]

Proof. The Bellman optimality operator has a unique fixed point \(v_*\). For each state choose an action attaining the finite maximum in the Bellman equation. The greedy policy \(\pi_*\) has Bellman expectation operator agreeing with the optimal operator at \(v_*\), so \(T_{\pi_*}v_*=T_*v_*=v_*\). By uniqueness of the fixed point of \(T_{\pi_*}\), \(v_{\pi_*}=v_*\).

Occupancy measures

A policy can also be represented by the discounted distribution of states and actions it visits.

Definition. The discounted state occupancy measure of \(\pi\) from \(\mu\) is \[ d_{\mu,\pi}(B)=(1-\gamma)\sum_{t=0}^\infty \gamma^tP_\pi(S_t\in B\mid S_0\sim\mu). \] The state-action occupancy is \(d_{\mu,\pi}(\mathrm{d}s,\mathrm{d}a)=d_{\mu,\pi}(\mathrm{d}s)\pi(\mathrm{d}a\mid s)\).

Proposition. For bounded \(g\), \[ \mathbb E_{\mu,\pi}\sum_{t=0}^\infty\gamma^tg(S_t,A_t) =\frac{1}{1-\gamma}\int g(s,a)d_{\mu,\pi}(\mathrm{d}s,\mathrm{d}a). \]

Proof. Substitute the definition of \(d_{\mu,\pi}\), integrate \(g\) against the distribution of \((S_t,A_t)\) at each time, and interchange the bounded sum and integral.

Value as a sufficient statistic for control

The value function compresses all future consequences of a state under a policy.

Remark. A value function is not a model of the environment. It is a prediction of return. Dynamic programming is possible because that prediction recursively supplies the continuation value after one transition.
