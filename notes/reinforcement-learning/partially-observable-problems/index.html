21. Partially observable problems

{% include note-topic-nav.html back_label="Back" back_url="/notes/reinforcement-learning/" prev_label="Last page" prev_url="/notes/reinforcement-learning/entropy-regularized-reinforcement-learning/" %}

Partially observable problems records the objects, main results, proof mechanisms, and boundary cases needed for the reinforcement learning note.

Definition

Definition. In a partially observable Markov decision process, the agent observes signals rather than the full Markov state. The belief state is the posterior distribution over hidden states.

Proposition

Proposition. The belief process is Markov under Bayesian filtering, so a POMDP can be transformed into a fully observed MDP on the belief space.

Proof

Proof. Update beliefs by Bayes rule after each action and observation. The conditional distribution of the next belief depends only on the current belief and action.

Remark

Remark. Partial observability shifts complexity from hidden state dynamics to infinite-dimensional belief states.
