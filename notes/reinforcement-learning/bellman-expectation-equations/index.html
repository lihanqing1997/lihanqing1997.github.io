The Bellman expectation equation is the fixed-point equation for the value of a fixed policy. It is linear in the value function once the policy is fixed.

Bellman operator for a policy

The one-step decomposition of return defines an operator whose fixed point is the policy value.

Definition. For a policy \(\pi\), define \[ (T_\pi v)(s)=r_\pi(s)+\gamma\int v(s')P_\pi(\mathrm{d}s'\mid s), \] where \(r_\pi(s)=\int r(s,a)\pi(\mathrm{d}a\mid s)\).

Theorem. The value function \(v_\pi\) is the unique bounded solution of \[ v=T_\pi v. \]

Proof. The return decomposition \(G_0=R_1+\gamma G_1\) gives \(v_\pi=T_\pi v_\pi\). If \(u\) and \(v\) both solve the equation, then \[ \|u-v\|_\infty=\gamma\|P_\pi(u-v)\|_\infty\le\gamma\|u-v\|_\infty. \] Since \(\gamma<1\), \(\|u-v\|_\infty=0\).

Matrix form

In finite spaces, policy evaluation is a linear system.

Proposition. For a finite MDP, \[ v_\pi=r_\pi+\gamma P_\pi v_\pi, \qquad v_\pi=(I-\gamma P_\pi)^{-1}r_\pi. \]

Proof. Rearrange the Bellman equation to \((I-\gamma P_\pi)v_\pi=r_\pi\). Since the spectral radius of \(P_\pi\) is at most \(1\), the Neumann series \(\sum_{k\ge0}\gamma^kP_\pi^k\) converges and equals \((I-\gamma P_\pi)^{-1}\).

Action-value expectation equation

The same fixed-point idea applies to \(q_\pi\).

Definition. The action-value Bellman operator is \[ (T_\pi^Q q)(s,a)=r(s,a)+\gamma\int\int q(s',a')\pi(\mathrm{d}a'\mid s')P(\mathrm{d}s'\mid s,a). \]

Theorem. The action-value function \(q_\pi\) is the unique bounded fixed point of \(T_\pi^Q\).

Proof. The return split after the first fixed action gives \(q_\pi=T_\pi^Qq_\pi\). The operator is a \(\gamma\)-contraction in sup norm because the transition-policy kernel is Markov. Banach's fixed-point theorem gives uniqueness.

Evaluation versus improvement

Expectation equations answer prediction questions for one policy; optimality equations answer control questions.

Remark. Many algorithms alternate these two roles: estimate \(v_\pi\) or \(q_\pi\), then choose actions greedily or approximately greedily with respect to that estimate.
