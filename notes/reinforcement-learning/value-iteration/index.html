Value iteration applies the Bellman optimality operator directly. It computes the optimal value as a fixed point and obtains policies by greedy extraction.

Optimal value iteration

The algorithm is the simplest fixed-point iteration for \(T_*\).

Definition. Starting from a bounded \(v_0\), value iteration sets \[ v_{k+1}=T_*v_k. \]

Theorem. The iterates satisfy \[ \|v_k-v_*\|_\infty\le\gamma^k\|v_0-v_*\|_\infty. \]

Proof. Use \(v_*=T_*v_*\) and the contraction property: \[ \|v_{k+1}-v_*\|_\infty\le\gamma\|v_k-v_*\|_\infty. \] Iterate the inequality.

Stopping by residual

Bellman residuals give computable stopping rules.

Proposition. If \(\|v_k-T_*v_k\|_\infty\le\varepsilon\), then \[ \|v_k-v_*\|_\infty\le\frac{\varepsilon}{1-\gamma}. \]

Proof. This is the residual bound for a contraction applied to \(T_*\).

Greedy policy loss

A nearly optimal value gives a nearly optimal greedy policy.

Theorem. If \(\pi\) is greedy with respect to \(v\), then \[ \|v_*-v_\pi\|_\infty\le \frac{2\gamma}{1-\gamma}\|v-v_*\|_\infty. \]

Proof. Greediness gives \(T_\pi v=T_*v\). Compare \[ v_*-v_\pi=T_*v_*-T_\pi v_\pi \] by adding and subtracting \(T_*v\) and \(T_\pi v\). The two approximation terms are each bounded by \(\gamma\|v-v_*\|_\infty\) or \(\gamma\|v-v_\pi\|_\infty\). Rearranging yields the standard bound.

Asynchronous updates

Tabular value iteration need not update all states at once.

Remark. Asynchronous value iteration updates selected coordinates using Bellman backups. Under discounted contraction and sufficient repeated updates of every state, the tabular iterates still converge to \(v_*\).
