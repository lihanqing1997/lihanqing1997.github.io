8. Value iteration

{% include note-topic-nav.html back_label="Back" back_url="/notes/reinforcement-learning/" prev_label="Last page" prev_url="/notes/reinforcement-learning/policy-iteration/" next_label="Next page" next_url="/notes/reinforcement-learning/temporal-difference-learning/" %}

Value iteration records the objects, main results, proof mechanisms, and boundary cases needed for the reinforcement learning note.

Definition

Definition. Value iteration repeatedly applies the optimal Bellman operator to approximate \(V^*\).

Proposition

Proposition. For discounted finite MDPs, value iteration converges geometrically in sup norm to \(V^*\), and greedy policies from accurate values are near-optimal.

Proof

Proof. Contraction gives value convergence. Performance loss of the greedy policy is bounded by translating Bellman residual or value error through the contraction inequality.

Remark

Remark. Value iteration is simple and robust, but high-dimensional state spaces require approximation.
