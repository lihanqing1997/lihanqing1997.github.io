4. Bellman optimality equations

{% include note-topic-nav.html back_label="Back" back_url="/notes/reinforcement-learning/" prev_label="Last page" prev_url="/notes/reinforcement-learning/bellman-expectation-equations/" next_label="Next page" next_url="/notes/reinforcement-learning/contraction-and-fixed-points/" %}

Bellman optimality equations records the objects, main results, proof mechanisms, and boundary cases needed for the reinforcement learning note.

Definition

Definition. The Bellman optimality equation characterizes optimal value by choosing the best first action and then behaving optimally.

Proposition

Proposition. For discounted MDPs, \(V^*=T^*V^*\), where \((T^*V)(s)=\max_a E[R_{t+1}+\gamma V(S_{t+1})\mid S_t=s,A_t=a]\).

Proof

Proof. Split any policy's return by its first action to show \(V^*\le T^*V^*\), and construct greedy policies to obtain the reverse inequality.

Remark

Remark. Optimality equations are nonlinear because of maximization. They are still contractions in discounted settings.
