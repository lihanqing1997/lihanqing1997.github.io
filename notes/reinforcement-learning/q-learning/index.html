10. Q-learning

{% include note-topic-nav.html back_label="Back" back_url="/notes/reinforcement-learning/" prev_label="Last page" prev_url="/notes/reinforcement-learning/temporal-difference-learning/" next_label="Next page" next_url="/notes/reinforcement-learning/eligibility-traces-and-td-lambda/" %}

Q-learning is an off-policy stochastic approximation method for the optimal action-value function. It learns greedy control values without following the greedy policy during data collection.

Tabular update

The update samples the Bellman optimality operator for action values.

Definition. For a transition \((S_t,A_t,R_{t+1},S_{t+1})\), tabular Q-learning updates \[ Q_{t+1}(S_t,A_t)=Q_t(S_t,A_t)+\alpha_t\{R_{t+1}+\gamma\max_aQ_t(S_{t+1},a)-Q_t(S_t,A_t)\}. \]

Proposition. Conditionally on \((S_t,A_t)=(s,a)\), the expected update target equals \((T_*^QQ_t)(s,a)\).

Proof. Condition on \((s,a)\). The reward and next-state expectation of \(R_{t+1}+\gamma\max_{a'}Q_t(S_{t+1},a')\) is \(r(s,a)+\gamma\int\max_{a'}Q_t(s',a')P(\mathrm{d}s'\mid s,a)\), which is the optimal \(Q\)-Bellman operator.

Convergence in the tabular case

The classical convergence theorem requires enough exploration and diminishing stepsizes.

Theorem. In a finite discounted MDP, if every state-action pair is visited infinitely often, rewards have bounded variance, and the stepsizes for each pair satisfy Robbins-Monro conditions, tabular Q-learning converges to \(q_*\) almost surely.

Proof. The update is asynchronous stochastic approximation to the contraction \(T_*^Q\). The conditional noise is a martingale difference, the stepsizes remove persistent noise, and infinite visits ensure every coordinate is updated. Standard asynchronous stochastic approximation for contractions gives almost-sure convergence to the unique fixed point \(q_*\).

Off-policy nature

The behavior policy generates data; the target policy is greedy with respect to \(Q\).

Remark. Q-learning can learn the greedy optimal value while taking exploratory actions, because the backup uses \(\max_aQ(S_{t+1},a)\) rather than averaging over the behavior policy's next action.

Function approximation warning

The convergence guarantee is tabular; nonlinear approximation changes the problem.

Remark. With function approximation, bootstrapping, and off-policy data, the update may diverge. Stabilizing devices such as target networks and replay buffers are algorithmic remedies, not replacements for the tabular contraction proof.
