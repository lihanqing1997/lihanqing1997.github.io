6. Policy evaluation

{% include note-topic-nav.html back_label="Back" back_url="/notes/reinforcement-learning/" prev_label="Last page" prev_url="/notes/reinforcement-learning/contraction-and-fixed-points/" next_label="Next page" next_url="/notes/reinforcement-learning/policy-iteration/" %}

Policy evaluation records the objects, main results, proof mechanisms, and boundary cases needed for the reinforcement learning note.

Definition

Definition. Policy evaluation estimates \(V^\pi\) or \(Q^\pi\) for a fixed policy from a known model, sampled trajectories, or both.

Proposition

Proposition. Monte Carlo evaluation is unbiased for episodic returns under suitable sampling, while temporal-difference evaluation bootstraps using current value estimates.

Proof

Proof. Monte Carlo averages complete sampled returns. TD methods replace the unknown future return by one-step reward plus estimated next value, creating a stochastic approximation to the Bellman fixed point.

Remark

Remark. Evaluation is conceptually simpler than control, but off-policy sampling and function approximation make it subtle.
