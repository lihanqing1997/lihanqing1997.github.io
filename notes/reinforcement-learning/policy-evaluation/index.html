6. Policy evaluation

{% include note-topic-nav.html back_label="Back" back_url="/notes/reinforcement-learning/" prev_label="Last page" prev_url="/notes/reinforcement-learning/contraction-and-fixed-points/" next_label="Next page" next_url="/notes/reinforcement-learning/policy-iteration/" %}

Policy evaluation estimates the value of a fixed policy. It is the prediction problem inside dynamic programming, TD learning, actor-critic methods, and off-policy evaluation.

Iterative policy evaluation

Repeated Bellman updates converge to the policy value in the tabular discounted case.

Definition. Starting from \(v_0\), iterative policy evaluation applies \[ v_{k+1}=T_\pi v_k. \]

Theorem. For bounded rewards and \(0\le\gamma<1\), \[ \|v_k-v_\pi\|_\infty\le\gamma^k\|v_0-v_\pi\|_\infty. \]

Proof. Since \(v_\pi=T_\pi v_\pi\) and \(T_\pi\) is a \(\gamma\)-contraction, \[ \|v_{k+1}-v_\pi\|_\infty\le\gamma\|v_k-v_\pi\|_\infty. \] Iterating gives the result.

Monte Carlo evaluation

Monte Carlo evaluation estimates values by sample returns without bootstrapping.

Definition. For visits to state \(s\), Monte Carlo evaluation averages realized returns: \[ \hat v_N(s)=\frac1{N_s}\sum_{i:S_i=s}G_i. \]

Proposition. If returns from state \(s\) are sampled i.i.d. with finite mean, then \(\hat v_N(s)\to v_\pi(s)\) almost surely.

Proof. Conditional on starting from \(s\), each return has expectation \(v_\pi(s)\). The strong law of large numbers applies to the return samples.

Linear system solution

In finite MDPs, exact policy evaluation can be solved by matrix inversion or iterative linear solvers.

Proposition. The finite policy value solves \((I-\gamma P_\pi)v=r_\pi\), and the Neumann representation is \[ v_\pi=\sum_{t=0}^\infty\gamma^tP_\pi^tr_\pi. \]

Proof. The Bellman equation gives the linear system. The Neumann series converges because \(\gamma P_\pi\) has spectral radius at most \(\gamma<1\), and multiplying \((I-\gamma P_\pi)\) by the series gives the identity.

Prediction error

Evaluation error can be measured by value norm, Bellman residual, or downstream policy-improvement loss.

Remark. A small value-function error does not always imply a small control error unless action gaps or approximate-greedy conditions are also controlled.
