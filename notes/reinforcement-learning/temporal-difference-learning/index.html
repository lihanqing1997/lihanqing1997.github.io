Temporal-difference learning evaluates a policy by bootstrapping: it uses observed rewards and the current estimate of the next state's value in the same update.

TD error

The TD error is a sample Bellman residual.

Definition. For policy evaluation, the one-step TD error is \[ \delta_t=R_{t+1}+\gamma v(S_{t+1})-v(S_t). \] Tabular TD(0) updates \[ v_{t+1}(S_t)=v_t(S_t)+\alpha_t\delta_t, \] leaving other states unchanged.

Proposition. Conditionally on \(S_t=s\), the expected TD target equals \((T_\pi v)(s)\).

Proof. Take conditional expectation of \(R_{t+1}+\gamma v(S_{t+1})\) given \(S_t=s\) under policy \(\pi\). The result is exactly \(r_\pi(s)+\gamma P_\pi v(s)=(T_\pi v)(s)\).

Stochastic approximation view

TD(0) is noisy fixed-point iteration for \(T_\pi\).

Theorem. In a finite on-policy discounted MDP with all states visited sufficiently often and stepsizes satisfying \(\sum_t\alpha_t=\infty\), \(\sum_t\alpha_t^2<\infty\), tabular TD(0) converges to \(v_\pi\) under standard irreducibility assumptions.

Proof. The conditional mean update is a contraction drift toward the Bellman fixed point, and the noise is a martingale difference with bounded second moment. Robbins-Monro stochastic approximation for contractions implies convergence to the unique fixed point \(v_\pi\).

Bias and variance

TD trades Monte Carlo unbiasedness for bootstrapping bias and lower variance.

Remark. The TD target uses the current value estimate, so it is biased unless \(v=v_\pi\). It often has lower variance than the full return because it does not include the entire future reward sequence.

Mean squared projected Bellman error

With function approximation, TD converges to a projected Bellman fixed point rather than the exact value.

Definition. For a linear value class \(\{v_\theta=\Phi\theta\}\) and weighting distribution \(d\), the projected Bellman equation is \[ \Phi\theta=\Pi_d T_\pi(\Phi\theta), \] where \(\Pi_d\) is \(L^2(d)\)-projection onto the feature span.

Remark. This projection is the source of both the usefulness and the pathology of TD with function approximation, especially off-policy.
