Temporal-difference learning records the objects, main results, proof mechanisms, and boundary cases needed for the reinforcement learning note.

Definition

Definition. Temporal-difference learning updates value estimates using the TD error \(\delta_t=R_{t+1}+\gamma V(S_{t+1})-V(S_t)\).

Proposition

Proposition. Under standard finite-state on-policy assumptions and appropriate step sizes, tabular TD(0) converges to the true value function.

Proof

Proof. The update is a stochastic approximation to the projected Bellman equation. Martingale noise is controlled by step-size conditions, and contraction supplies stability.

Remark

Remark. TD learning is the core idea of learning from incomplete returns while bootstrapping from current estimates.
