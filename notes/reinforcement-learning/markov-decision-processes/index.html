A Markov decision process is the mathematical object behind fully observed reinforcement learning. It separates the environment dynamics from the agent's policy and makes sequential optimization a problem about controlled Markov chains.

Controlled Markov models

The Markov property says that the present state and action contain all information needed to predict the next transition.

Definition. A discounted Markov decision process is a tuple \((\mathcal S,\mathcal A,P,r,\gamma,\mu)\), where \(\mathcal S\) is the state space, \(\mathcal A\) is the action space, \(P(\cdot\mid s,a)\) is a Markov kernel, \(r(s,a)=\mathbb E(R_{t+1}\mid S_t=s,A_t=a)\), \(0\le\gamma<1\), and \(\mu\) is an initial distribution. A policy \(\pi(\cdot\mid s)\) is a Markov kernel from states to actions.

Proposition. For a fixed policy \(\pi\), the state process is a Markov chain with transition kernel \[ P_\pi(B\mid s)=\int_{\mathcal A}P(B\mid s,a)\pi(\mathrm{d}a\mid s). \]

Proof. Condition on the present state \(S_t=s\). The action is sampled from \(\pi(\cdot\mid s)\), and the next state is sampled from \(P(\cdot\mid s,A_t)\). Integrating over the action gives the displayed kernel. No earlier state or action enters the conditional law once \(S_t\) is fixed.

Discounted return

Discounting makes infinite-horizon value finite and gives Bellman operators contraction geometry.

Definition. The discounted return from time \(t\) is \[ G_t=\sum_{k=0}^\infty \gamma^k R_{t+k+1}. \] When rewards are bounded by \(M\), \(|G_t|\le M/(1-\gamma)\).

Proposition. If \(|r(s,a)|\le M\), then every policy value satisfies \(\|v_\pi\|_\infty\le M/(1-\gamma)\).

Proof. Take absolute values inside the discounted sum: \[ |v_\pi(s)|=\left|\mathbb E_\pi\sum_{k\ge0}\gamma^kR_{k+1}\mid S_0=s\right| \le \sum_{k\ge0}\gamma^kM=\frac{M}{1-\gamma}. \]

Finite and measurable cases

Finite MDPs can be written in matrices; general spaces require kernels and measurability.

Remark. In finite state-action spaces, \(P_\pi\) is a stochastic matrix and value functions are vectors. In Borel spaces, existence of optimal stationary policies requires compactness, continuity, or measurable-selection hypotheses; the formal Bellman equations remain the organizing principle, but measurability is not automatic.

Model-free and model-based viewpoints

Reinforcement learning algorithms differ in whether they estimate the transition model or estimate value and policy objects directly.

Definition. A model-based method estimates or uses \(P\) and \(r\). A model-free method estimates \(v_\pi\), \(q_\pi\), \(v_*\), \(q_*\), or policy gradients without explicitly constructing a transition model.

Remark. The distinction is algorithmic, not mathematical: both approaches are approximating consequences of the same MDP law.
