18. Function approximation

{% include note-topic-nav.html back_label="Back" back_url="/notes/reinforcement-learning/" prev_label="Last page" prev_url="/notes/reinforcement-learning/off-policy-evaluation/" next_label="Next page" next_url="/notes/reinforcement-learning/approximate-dynamic-programming/" %}

Function approximation is necessary in large state spaces, but it changes exact dynamic programming into projected or approximate fixed-point problems.

Approximate value classes

A value function is represented by parameters rather than a table.

Definition. A parametric value approximation has the form \(v_\theta(s)\). In the linear case, \(v_\theta(s)=\phi(s)^\top\theta\), where \(\phi(s)\in\mathbb R^d\).

Remark. Approximation error is unavoidable when \(v_\pi\) is not in the chosen class. Estimation and bootstrapping errors then sit on top of that approximation error.

Projected Bellman equation

With linear approximation, TD seeks a projected fixed point.

Definition. Let \(\Phi\) be the feature matrix and \(D\) a state weighting distribution. The \(D\)-orthogonal projection onto the feature span is \(\Pi_D\). The projected Bellman equation is \[ \Phi\theta=\Pi_DT_\pi(\Phi\theta). \]

Theorem. If \(\Phi^\top D(I-\gamma P_\pi)\Phi\) is nonsingular, the projected Bellman equation has a unique solution satisfying \[ \Phi^\top D\{r_\pi+\gamma P_\pi\Phi\theta-\Phi\theta\}=0. \]

Proof. The projection residual is orthogonal to the feature span in the \(D\)-inner product. Thus \(\Phi\theta=\Pi_DT_\pi\Phi\theta\) is equivalent to \(\Phi^\top D(T_\pi\Phi\theta-\Phi\theta)=0\). Expanding \(T_\pi\) gives the displayed linear system. Nonsingularity gives uniqueness.

Deadly triad

Function approximation, bootstrapping, and off-policy learning can break contraction.

Remark. In the tabular on-policy discounted case, Bellman operators are contractions in sup norm. With projection and off-policy weighting, the composed operator \(\Pi_DT_\pi\) need not be a contraction in the relevant norm, so iterative methods may diverge.

Gradient versus semi-gradient

Many practical updates are not gradients of a true objective.

Definition. A semi-gradient TD update differentiates only the current estimate \(v_\theta(S_t)\), treating the bootstrap target \(R_{t+1}+\gamma v_\theta(S_{t+1})\) as fixed.

Remark. This shortcut is computationally convenient but changes the optimization geometry. Gradient-TD methods were designed to optimize well-defined projected Bellman-error objectives.
