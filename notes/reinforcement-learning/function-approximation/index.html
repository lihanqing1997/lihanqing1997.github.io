Function approximation records the objects, main results, proof mechanisms, and boundary cases needed for the reinforcement learning note.

Definition

Definition. Function approximation represents values, policies, or models in parameterized classes rather than tabular arrays. It is necessary for large or continuous state spaces.

Proposition

Proposition. With linear value approximation, projected Bellman equations replace exact Bellman equations because the Bellman update may leave the approximation space.

Proof

Proof. Project the Bellman fixed-point equation onto the feature span under a chosen sampling distribution. Orthogonality of residuals yields the normal equations for the projected problem.

Remark

Remark. Approximation changes the fixed point. Understanding the projection is as important as running the algorithm.
