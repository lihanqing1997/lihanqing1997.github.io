4. Gronwall inequalities

{% include note-topic-nav.html back_label="Back" back_url="/notes/differential-equations/" prev_label="Last page" prev_url="/notes/differential-equations/maximal-solutions-and-blow-up/" next_label="Next page" next_url="/notes/differential-equations/flows-and-phase-portraits/" %}

Gronwall inequalities records the objects, main results, proof mechanisms, and boundary cases needed for the differential equations note.

Definition

Definition. Gronwall inequalities bound a nonnegative function that is controlled by its own integral. They are the basic estimates for uniqueness and continuous dependence.

Proposition

Proposition. If \(u(t)\le a+\int_{t_0}^t b(s)u(s)\,\mathrm{d}s\), then \(u(t)\le a\exp(\int_{t_0}^t b(s)\,\mathrm{d}s)\).

Proof

Proof. Set \(v(t)=a+\int_{t_0}^t b(s)u(s)\,\mathrm{d}s\) so that \(u\le v\) and \(v'\le bv\). Multiply by the integrating factor \(\exp(-\int_{t_0}^t b(s)\,\mathrm{d}s)\) and integrate.

Remark

Remark. Gronwall is the estimate that turns local Lipschitz control into uniqueness, stability, and parameter dependence.
