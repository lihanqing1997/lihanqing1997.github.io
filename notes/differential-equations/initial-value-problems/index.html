Initial value problems records the objects, main results, proof mechanisms, and boundary cases needed for the differential equations note.

Definition

Definition. An initial value problem specifies a differential equation and the value of the unknown at an initial time, for example \(\mathrm{d}x/\mathrm{d}t=f(t,x)\), \(x(t_0)=x_0\).

Peano and Picard-Lindelöf theorems

Theorem. If \(f\) is continuous, local solutions exist. If \(f\) is locally Lipschitz in the state variable, the solution is locally unique.

Proof

Proof. Existence follows from compactness of approximate solutions or Schauder-type fixed points. Uniqueness follows by estimating two solutions and applying Gronwall inequality.

Remark

Remark. ODE theory separates existence from uniqueness. Continuity alone can create solutions but not determine them.
