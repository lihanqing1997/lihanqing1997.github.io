6. Cyclical monotonicity

{% include note-topic-nav.html back_label="Back" back_url="/notes/optimal-transport/" prev_label="Last page" prev_url="/notes/optimal-transport/c-transforms-and-c-convexity/" next_label="Next page" next_url="/notes/optimal-transport/wasserstein-distances/" %}

Cyclical monotonicity records the objects, main results, proof mechanisms, and boundary cases needed for the optimal transport note.

Definition

Definition. A set \(\Gamma\subset X\times Y\) is \(c\)-cyclically monotone if every finite cycle satisfies \(\sum_i c(x_i,y_i)\le\sum_i c(x_i,y_{\sigma(i)})\).

Proposition

Proposition. An optimal plan is concentrated on a \(c\)-cyclically monotone set. Conversely, under appropriate conditions, concentration on such a set implies optimality.

Proof

Proof. Optimality prevents any finite rearrangement of assigned destinations from lowering cost. The converse constructs a potential whose contact set contains the support.

Remark

Remark. Cyclical monotonicity is the local finite-dimensional shadow of global optimality.
