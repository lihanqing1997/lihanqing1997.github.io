Do-calculus records the objects, main results, proof mechanisms, and boundary cases needed for the causal inference note.

Definition

Definition. Do-calculus is a set of graphical rules for transforming interventional distributions into observational or simpler interventional distributions.

Proposition

Proposition. If repeated application of do-calculus removes all intervention operators from a target causal query, then the query is identifiable from the observational distribution.

Proof

Proof. Each rule is justified by d-separation in a graph modified by deleting or adding intervention arrows. Soundness follows from the Markov factorization of intervention distributions.

Remark

Remark. Do-calculus is an identification calculus. It does not require parametric modeling, but it does require the causal graph.
