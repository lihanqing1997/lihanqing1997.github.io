Back-door adjustment records the objects, main results, proof mechanisms, and boundary cases needed for the causal inference note.

Definition

Definition. A set \(Z\) satisfies the back-door criterion for treatment \(A\) and outcome \(Y\) if it blocks all noncausal paths from \(A\) to \(Y\) and contains no descendants of \(A\).

Back-door formula

Theorem. If \(Z\) satisfies the back-door criterion, then \(P(Y\in B\mid \operatorname{do}(A=a))=\int P(Y\in B\mid A=a,Z=z)\,\mathrm{d}P_Z(z)\).

Proof

Proof. In the mutilated graph for intervention, back-door blocking gives conditional exchangeability of potential outcomes or interventional outcomes given \(Z\). Then apply the adjustment formula.

Remark

Remark. Back-door adjustment is graphical confounding control. It tells which variables to condition on, not how to estimate after conditioning.
