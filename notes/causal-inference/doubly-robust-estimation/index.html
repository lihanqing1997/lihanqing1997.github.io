Doubly robust estimation records the objects, main results, proof mechanisms, and boundary cases needed for the causal inference note.

Definition

Definition. A doubly robust score combines outcome regression and propensity weighting so that the estimand remains correctly centered if either nuisance component is correct.

Proposition

Proposition. The augmented inverse probability score for \(E[Y(1)]\), \(m_1(X)+A(Y-m_1(X))/e(X)\), has expectation \(E[Y(1)]\) if either \(m_1\) or \(e\) is correctly specified.

Proof

Proof. Condition on \(X\). If the outcome regression is correct, the residual term has mean zero. If the propensity is correct, the weighted residual restores the true conditional mean.

Remark

Remark. Double robustness is an estimating-equation property, not protection against violations of causal identification assumptions.
