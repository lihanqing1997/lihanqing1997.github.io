20. Transportability and external validity

{% include note-topic-nav.html back_label="Back" back_url="/notes/causal-inference/" prev_label="Last page" prev_url="/notes/causal-inference/sensitivity-analysis/" %}

Transportability and external validity records the objects, main results, proof mechanisms, and boundary cases needed for the causal inference note.

Definition

Definition. Transportability asks whether a causal effect learned in one population or environment identifies the effect in another. External validity is the corresponding scientific concern.

Proposition

Proposition. Transport formulas require assumptions about which mechanisms remain invariant across environments and which distributions change. Selection diagrams encode these invariance assumptions graphically.

Proof

Proof. Augment the causal graph with selection variables marking changed mechanisms. Use do-calculus and ordinary probability transport across stable mechanisms to express the target effect using source experiments and target observations.

Remark

Remark. A causal estimate is not automatically portable. Transportability makes the extrapolation assumptions explicit.
