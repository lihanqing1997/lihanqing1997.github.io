19. Sensitivity analysis

{% include note-topic-nav.html back_label="Back" back_url="/notes/causal-inference/" prev_label="Last page" prev_url="/notes/causal-inference/longitudinal-treatment-regimes/" next_label="Next page" next_url="/notes/causal-inference/transportability-and-external-validity/" %}

Sensitivity analysis records the objects, main results, proof mechanisms, and boundary cases needed for the causal inference note.

Definition

Definition. Sensitivity analysis quantifies how conclusions change when identification assumptions such as no unmeasured confounding are relaxed.

Proposition

Proposition. Bounds or bias formulas can be derived by parameterizing departures from exchangeability, such as odds-ratio bounds, selection functions, or negative-control restrictions.

Proof

Proof. Write the target causal estimand as the identified functional plus a bias term involving the sensitivity parameter. Vary the parameter over a scientifically meaningful range.

Remark

Remark. Sensitivity analysis does not eliminate uncertainty about assumptions; it makes that uncertainty explicit and quantitative.
