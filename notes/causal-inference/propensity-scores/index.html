Propensity scores records the objects, main results, proof mechanisms, and boundary cases needed for the causal inference note.

Definition

Definition. The propensity score is \(e(X)=P(A=1\mid X)\). It is a balancing score: conditional on \(e(X)\), the distribution of covariates is the same across treatment groups under treatment assignment.

Proposition

Proposition. If treatment is exchangeable given \(X\), then it is exchangeable given the propensity score \(e(X)\), provided positivity holds.

Proof

Proof. Show first that \(A\perp X\mid e(X)\) by Bayes rule. Then combine this balancing property with exchangeability given \(X\) to pass to exchangeability given \(e(X)\).

Remark

Remark. The propensity score reduces adjustment dimension for treatment assignment, but it does not by itself solve lack of overlap or hidden confounding.
