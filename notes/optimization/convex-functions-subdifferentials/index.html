3. Convex functions and subdifferentials

{% include note-topic-nav.html back_label="Back" back_url="/notes/optimization/" prev_label="Last page" prev_url="/notes/optimization/convex-sets-separation/" next_label="Next page" next_url="/notes/optimization/smoothness-strong-convexity-condition-numbers/" %}

This page records the definitions, propositions, and proof mechanisms for Convex functions and subdifferentials.

Convex function

Definition. A function \(f:C\to\overline{\mathbb R}\) is convex if \(f(tx+(1-t)y)\le tf(x)+(1-t)f(y)\). Its epigraph \(\{(x,r):f(x)\le r\}\) is then a convex set.

Subgradient

Definition. For a convex \(f\), a vector \(g\) is a subgradient at \(x\) if \(f(y)\ge f(x)+\langle g,y-x\rangle\) for all \(y\). The set of all such vectors is \(\partial f(x)\).

Global optimality

Theorem. For proper convex \(f\), a point \(x^\star\) minimizes \(f\) if and only if \(0\in\partial f(x^\star)\). If \(f\) is differentiable this reduces to \(\nabla f(x^\star)=0\).

Nonsmooth calculus

Remark. Subdifferentials let absolute values, maxima, norms, indicators, and penalties enter the same optimality language as smooth functions.
