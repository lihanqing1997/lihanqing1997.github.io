This page records the definitions, propositions, and proof mechanisms for Smoothness, strong convexity, and condition numbers.

Smoothness

Definition. A differentiable function is \(L\)-smooth if \(\|\nabla f(x)-\nabla f(y)\|\le L\|x-y\|\). Equivalently for twice differentiable \(f\), \(\nabla^2f(x)\preceq LI\).

Strong convexity

Definition. A function is \(\mu\)-strongly convex if \(f(y)\ge f(x)+\langle g,y-x\rangle+\frac{\mu}{2}\|y-x\|^2\) for \(g\in\partial f(x)\). For smooth \(f\), this is \(\nabla^2f(x)\succeq\mu I\).

Descent lemma

Lemma. If \(f\) is \(L\)-smooth, then \(f(y)\le f(x)+\langle\nabla f(x),y-x\rangle+\frac L2\|y-x\|^2\). Taking \(y=x-L^{-1}\nabla f(x)\) gives guaranteed descent.

Condition number

Remark. The ratio \(\kappa=L/\mu\) governs many convergence rates. Large \(\kappa\) means level sets are elongated, so first-order methods zigzag unless geometry or preconditioning is improved.
