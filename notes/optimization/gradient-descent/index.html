11. Gradient descent

{% include note-topic-nav.html back_label="Back" back_url="/notes/optimization/" prev_label="Last page" prev_url="/notes/optimization/saddle-points-minimax-formulations/" next_label="Next page" next_url="/notes/optimization/accelerated-gradient-methods/" %}

This page records the definitions, propositions, and proof mechanisms for Gradient descent.

Gradient step

Definition. Gradient descent iterates \(x_{k+1}=x_k-\eta_k\nabla f(x_k)\). The direction \(-\nabla f(x_k)\) is the steepest descent direction for the Euclidean norm.

Descent for smooth functions

Theorem. If \(f\) is \(L\)-smooth and \(0<\eta\le1/L\), then \(f(x_{k+1})\le f(x_k)-\frac{\eta}{2}\|\nabla f(x_k)\|^2\). Hence the gradient norm must become small along bounded-below descent sequences.

Convex rate

Theorem. If \(f\) is convex and \(L\)-smooth, fixed step \(1/L\) gives \(f(x_k)-f^\star\le L\|x_0-x^\star\|^2/(2k)\). If \(f\) is also \(\mu\)-strongly convex, the rate is linear with factor at most \(1-\mu/L\).

Geometry

Remark. Gradient descent depends on the chosen inner product. Changing coordinates without changing the metric changes the algorithm; natural gradient fixes this for statistical models.
