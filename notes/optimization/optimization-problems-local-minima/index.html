Optimization studies the structure and computation of points where an objective is small, subject to constraints when present.

Optimization problem

Definition. An optimization problem is specified by a set \(C\) and an objective \(f:C\to\overline{\mathbb R}\). A point \(x^\star\in C\) is a global minimizer if \(f(x^\star)\le f(x)\) for all \(x\in C\); it is a local minimizer if the same inequality holds on \(C\cap U\) for some neighborhood \(U\) of \(x^\star\).

First-order necessary condition

Proposition. If \(C\subset\mathbb R^d\) is open, \(f\) is differentiable, and \(x^\star\) is a local minimizer, then \(\nabla f(x^\star)=0\). More generally, every feasible direction has nonnegative directional derivative.

Second-order condition

Proposition. If \(f\) is twice continuously differentiable on an open set and \(x^\star\) is a local minimizer, then \(\nabla f(x^\star)=0\) and \(\nabla^2 f(x^\star)\) is positive semidefinite. If \(\nabla f(x^\star)=0\) and the Hessian is positive definite, then \(x^\star\) is a strict local minimizer.

Geometry of nonconvexity

Remark. Local conditions identify candidates, not global solutions. Convexity is the property that turns local optimality into global optimality; without it, algorithms also need initialization, landscape structure, or statistical arguments.
