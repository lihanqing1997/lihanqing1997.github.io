This page records the definitions, propositions, and proof mechanisms for Optimization error and statistical error.

Error decomposition

Definition. For an estimator computed by optimization, the excess risk can be decomposed into statistical error from replacing \(P\) by \(P_n\), approximation error from model restriction, and optimization error from not solving the empirical problem exactly.

Optimization tolerance

Proposition. If the statistical target fluctuates at scale \(r_n\), there is usually no inferential benefit in solving the empirical problem to accuracy much smaller than \(r_n\), unless the optimization error affects bias or constraints differently.

Stability and generalization

Theorem. Algorithmic stability can convert properties of the optimization map into generalization bounds: if replacing one observation changes the output little, empirical and population risks are close.

Practical principle

Remark. Statistical computation asks for the right mathematical tolerance, not exact arithmetic minimization. The algorithm should be accurate enough for inference but not divorced from sampling uncertainty.
