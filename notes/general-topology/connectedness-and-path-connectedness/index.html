8. Connectedness and path connectedness

{% include note-topic-nav.html back_label="Back" back_url="/notes/general-topology/" prev_label="Last page" prev_url="/notes/general-topology/compactness/" next_label="Next page" next_url="/notes/general-topology/local-compactness-and-one-point-compactification/" %}

Connectedness and path connectedness records the objects, main results, proof mechanisms, and boundary cases needed for the general topology note.

Definition

Definition. A space is connected if it cannot be split into two disjoint nonempty open sets. It is path connected if any two points can be joined by a continuous path.

Proposition

Proposition. Path connectedness implies connectedness, but the converse can fail. Continuous images of connected spaces are connected.

Proof

Proof. The image of a connected space under a continuous map cannot be separated, because a separation would pull back to a separation of the domain. A path image is connected, and unions of connected sets with common point are connected.

Remark

Remark. Connectedness records indivisibility; path connectedness records explicit motion. They coincide in many manifolds but not in arbitrary spaces.
