13. Urysohn lemma

{% include note-topic-nav.html back_label="Back" back_url="/notes/general-topology/" prev_label="Last page" prev_url="/notes/general-topology/tychonoff-theorem/" next_label="Next page" next_url="/notes/general-topology/tietze-extension-theorem/" %}

Urysohn's lemma is the moment normality turns into continuous real-valued functions. It is the technical heart behind metrization and extension theorems.

Normal separation by functions

Normality separates closed sets by open neighborhoods; Urysohn upgrades that separation to a continuous function.

Theorem. If \(X\) is normal and \(A,B\subseteq X\) are disjoint closed sets, then there exists a continuous function \(f:X\to[0,1]\) such that \(f(A)=0\) and \(f(B)=1\).

Proof. The construction in the next two sections proves the theorem. Normality first gives nested open sets \(U_r\) indexed by dyadic rationals with \(\overline{U_r}\subseteq U_s\) for \(ra\}\).

Remark. The conclusion is false in general Hausdorff spaces. Normality is exactly the open-set supply needed to build a continuous scale between two closed sets.

Nested open sets

The proof builds open sets indexed by dyadic rationals, with closures nested in the correct order.

Lemma. If \(X\) is normal, \(F\subseteq U\), \(F\) is closed, and \(U\) is open, then there is an open set \(V\) with \(F\subseteq V\subseteq\overline V\subseteq U\).

The Urysohn function

The function is the first dyadic level whose open set contains the point.

Definition. Define \(f(x)=\inf\{r\in\mathbb D\cap[0,1]:x\in U_r\}\), where \(\mathbb D\) denotes the dyadic rationals.

Proposition. The function \(f\) is continuous, equals \(0\) on \(A\), and equals \(1\) on \(B\).

Proof. The endpoint values follow from \(A\subseteq U_0\) and \(B\cap U_r=\varnothing\) for all \(r<1\). For continuity, it suffices to show \(\{fa\}\) are open. The first is the union of \(U_r\) over dyadic \(ra\), using the nesting relation to compare levels. Both are open.

Consequences

Urysohn functions separate topology by scalar-valued observables.

Corollary. Every normal \(T_1\) space is completely regular: points and closed sets can be separated by continuous functions into \([0,1]\).

Proof. Apply Urysohn's lemma to the disjoint closed sets \(\{x\}\) and \(F\).

Remark. This function-separation principle is the entrance to Tychonoff embeddings and Stone-Cech compactification.
