11. Nets and filters

{% include note-topic-nav.html back_label="Back" back_url="/notes/general-topology/" prev_label="Last page" prev_url="/notes/general-topology/countability-axioms/" next_label="Next page" next_url="/notes/general-topology/tychonoff-theorem/" %}

Nets and filters records the objects, main results, proof mechanisms, and boundary cases needed for the general topology note.

Definition

Definition. A net is a function from a directed set into a space. Nets generalize sequences so that arbitrary topological convergence can be characterized.

Proposition

Proposition. A point lies in the closure of a set exactly when some net from that set converges to the point. A space is compact exactly when every net has a convergent subnet.

Proof

Proof. Use neighborhoods of the closure point as a directed set to build a convergent net. For compactness, open covers and subnet failure can be translated into finite-intersection contradictions.

Remark

Remark. Nets are what sequences become when spaces are not first countable. They are essential for weak topologies and product compactness.
