14. Tietze extension theorem

{% include note-topic-nav.html back_label="Back" back_url="/notes/general-topology/" prev_label="Last page" prev_url="/notes/general-topology/urysohn-lemma/" next_label="Next page" next_url="/notes/general-topology/metrization-theorems/" %}

Tietze extension theorem records the objects, main results, proof mechanisms, and boundary cases needed for the general topology note.

Definition

Definition. Tietze extension extends continuous real-valued functions from closed subsets of normal spaces to the whole space.

Tietze extension theorem

Theorem. If \(A\) is closed in a normal space \(X\) and \(f:A\to[-1,1]\) is continuous, then \(f\) extends to a continuous \(F:X\to[-1,1]\).

Proof

Proof. Approximate the function by successive continuous functions built using Urysohn lemma, each reducing the remaining error geometrically. Uniform convergence gives the extension.

Remark

Remark. Extension theorems show that normality is not merely separation by neighborhoods; it is separation by scalar functions.
