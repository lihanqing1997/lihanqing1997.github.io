4. Cox proportional hazards model

{% include note-topic-nav.html back_label="Back" back_url="/notes/survival-analysis/" prev_label="Last page" prev_url="/notes/survival-analysis/counting-processes-martingales/" next_label="Next page" next_url="/notes/survival-analysis/partial-likelihood-score-information/" %}

Model

The Cox model separates regression effects from the unknown baseline hazard. This is why it is semiparametric: the parameter of interest is finite-dimensional, but the nuisance parameter is a function.

Definition. Given covariates \(Z_i\in\mathbb R^p\), the Cox proportional hazards model assumes \[ \lambda_i(t\mid Z_i)=Y_i(t)\lambda_0(t)\exp(Z_i^\top\beta), \] where \(\lambda_0\) is an unspecified baseline hazard and \(\beta\in\mathbb R^p\).

Proposition. For two subjects with covariates \(z_1,z_2\), the hazard ratio is \[ {\lambda(t\mid z_1)\over\lambda(t\mid z_2)}=\exp\{(z_1-z_2)^\top\beta\}, \] independent of \(t\).

Proof. The baseline hazard cancels in the ratio. The proportional hazards interpretation is exactly this cancellation.

Why Cox is not a GLM

Cox regression resembles logistic regression through convex log-likelihood geometry, but the denominator of the partial likelihood uses the random risk set. Hence observations do not decouple into independent fixed-design likelihood terms.

Remark. The unknown \(\lambda_0\) is not estimated before \(\beta\). The partial likelihood removes it by conditioning on which member of the risk set failed at each event time.

Identifiability and normalization

Proposition. If the risk-set distribution of \(Z\) is not concentrated on an affine hyperplane over a time set with positive event intensity, then \(\beta\) is identifiable from the conditional failure probabilities inside risk sets.

Proof. The conditional probability that subject \(i\) fails at time \(t\) given one failure in the risk set is proportional to \(\exp(Z_i^\top\beta)\). If two parameter values give the same probabilities for all risk sets, then \((\beta-\beta')^\top Z\) must be constant on those risk sets. Nondegenerate risk-set variation forces \(\beta=\beta'\).

Stratification and nonproportionality

Definition. A stratified Cox model permits different baseline hazards \(\lambda_{0g}\) across strata \(g\) while keeping a common coefficient \(\beta\): \[ \lambda_i(t\mid Z_i,g_i)=Y_i(t)\lambda_{0g_i}(t)\exp(Z_i^\top\beta). \]

Remark. Stratification protects the regression coefficient against baseline heterogeneity. It does not repair time-varying regression effects; those require interactions such as \(Z(t)^\top\beta(t)\) or explicit time-dependent covariates.
