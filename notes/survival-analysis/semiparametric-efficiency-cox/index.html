10. Semiparametric efficiency in Cox models

{% include note-topic-nav.html back_label="Back" back_url="/notes/survival-analysis/" prev_label="Last page" prev_url="/notes/survival-analysis/frailty-random-effects/" next_label="Next page" next_url="/notes/survival-analysis/high-dimensional-cox-regression/" %}

Nuisance tangent space

The Cox model is semiparametric because \(\beta\) is finite-dimensional and \(\Lambda_0\) is infinite-dimensional. Efficiency asks for the best possible asymptotic variance for estimating \(\beta\) after removing nuisance directions.

Definition. A nuisance tangent direction for the baseline hazard is a smooth path \(\lambda_{0,\eta}(t)\) through \(\lambda_0\). Its score is a martingale integral of a predictable scalar function against \(\mathrm{d}M_i(t)\).

Theorem. In the Cox model, the efficient score for \(\beta\) is the centered risk-set score \[ \ell_\beta^*=\sum_i\int_0^\tau \{Z_i-\bar Z(t,\beta_0)\}\,\mathrm{d}M_i(t), \] and its variance is the efficient information.

Proof. The raw regression score contains \(Z_i\,\mathrm{d}M_i(t)\). Projecting it off the nuisance tangent space removes predictable functions of time that can be generated by changes in the baseline hazard. The least-squares projection within each risk set is the weighted risk-set mean \(\bar Z(t,\beta_0)\), leaving \(Z_i-\bar Z(t,\beta_0)\).

Remark. This projection view is the clean mathematical reason the Cox partial likelihood has the right information for \(\beta\) without estimating \(\lambda_0\) parametrically.

Efficient influence function

Theorem. The efficient influence function for \(\beta\) in the Cox model is \[ \varphi_\beta(O)=\mathcal I_{\mathrm{eff}}^{-1} \int_0^\tau\{Z-\bar z(t,\beta_0)\}\,\mathrm{d}M(t), \] with \(\mathcal I_{\mathrm{eff}}=E\int_0^\tau v(t,\beta_0)Y(t)\lambda_0(t)e^{Z^\top\beta_0}\,\mathrm{d}t\).

Proof. The regression score is projected orthogonally off the nuisance tangent space generated by baseline-hazard perturbations. The projection subtracts the risk-set conditional mean \(\bar z(t,\beta_0)\). Multiplication by the inverse efficient information normalizes the score into an influence function.

Efficiency of partial likelihood

Corollary. The Cox partial likelihood estimator is semiparametrically efficient for \(\beta\) under the correctly specified model and regular censoring assumptions.
