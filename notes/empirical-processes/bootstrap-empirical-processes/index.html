19. Bootstrap empirical processes

{% include note-topic-nav.html back_label="Back" back_url="/notes/empirical-processes/" prev_label="Last page" prev_url="/notes/empirical-processes/multiplier-empirical-processes/" next_label="Next page" next_url="/notes/empirical-processes/sequential-empirical-processes/" %}

Bootstrap empirical processes replace \(P\) by \(P_n\) and ask whether the resampled fluctuation conditionally reproduces the original fluctuation.

Definition. If \(P_n^*\) is the empirical measure of a bootstrap sample from \(P_n\), then \(\mathbb G_n^*=\sqrt n(P_n^*-P_n)\). Bootstrap consistency means \(\mathbb G_n^*\) converges conditionally to the same limit as \(\mathbb G_n\).

Conditional empirical-process convergence

  • Donsker classes are the natural setting for nonparametric bootstrap empirical-process consistency.
  • Exchangeable bootstrap weights generalize multinomial resampling.
  • Functional delta methods transfer bootstrap process convergence to bootstrap distributional consistency.

Delta method after bootstrap

This page connects the empirical-process note to the inference-level bootstrap page: here the object is process convergence; there the object is confidence and testing.

Bootstrap process criterion

Theorem. Bootstrap consistency for a statistic \(T_n=\phi(P_n)\) often follows in two steps: first show \(\sqrt n(P_n^*-P_n)\) conditionally mimics \(\sqrt n(P_n-P)\); then apply a conditional functional delta method to \(\phi\).

Hadamard differentiable and nonregular functionals

If a statistic is a non-smooth functional of the empirical process, the ordinary bootstrap can fail even when the empirical process itself is well behaved. Always check the continuity or differentiability of the final map.

19. Processus empiriques bootstrap

{% include note-topic-nav.html back_label="Retour" back_url="/notes/empirical-processes/" prev_label="Page précédente" prev_url="/notes/empirical-processes/multiplier-empirical-processes/" next_label="Page suivante" next_url="/notes/empirical-processes/sequential-empirical-processes/" %}

Les processus bootstrap remplacent \(P\) par \(P_n\) et demandent si la fluctuation rééchantillonnée reproduit conditionnellement la fluctuation originale.

Définition. Si \(P_n^*\) est la mesure empirique d'un échantillon bootstrap issu de \(P_n\), alors \(\mathbb G_n^*=\sqrt n(P_n^*-P_n)\). La consistance bootstrap signifie que \(\mathbb G_n^*\) converge conditionnellement vers la même limite que \(\mathbb G_n\).

Convergence conditionnelle du processus empirique

  • Les classes de Donsker sont le cadre naturel de la consistance bootstrap non paramétrique.
  • Les poids bootstrap échangeables généralisent le rééchantillonnage multinomial.
  • La méthode delta fonctionnelle transfère la convergence du processus bootstrap aux fonctionnelles.

Méthode delta après bootstrap

Cette page relie la théorie empirique à la page bootstrap d'inférence : ici l'objet est la convergence de processus; là-bas les intervalles et tests.

Critère du processus bootstrap

Théorème. La consistance bootstrap de \(T_n=\phi(P_n)\) suit souvent en deux étapes : montrer que \(\sqrt n(P_n^*-P_n)\) imite conditionnellement \(\sqrt n(P_n-P)\), puis appliquer la méthode delta fonctionnelle conditionnelle.

Fonctionnelles différentiables et non régulières

Si la statistique est une fonctionnelle non lisse du processus empirique, le bootstrap ordinaire peut échouer. Il faut vérifier la continuité ou différentiabilité de la transformation finale.

19. Bootstrap 经验过程

{% include note-topic-nav.html back_label="返回" back_url="/notes/empirical-processes/" prev_label="上一页" prev_url="/notes/empirical-processes/multiplier-empirical-processes/" next_label="下一页" next_url="/notes/empirical-processes/sequential-empirical-processes/" %}

Bootstrap 经验过程用 \(P_n\) 替代 \(P\)，并询问重抽样波动是否能在条件意义下再现原始波动。

定义. 若 \(P_n^*\) 是从 \(P_n\) 重抽样得到的经验测度，则 \(\mathbb G_n^*=\sqrt n(P_n^*-P_n)\)。Bootstrap 一致性指 \(\mathbb G_n^*\) 条件收敛到与 \(\mathbb G_n\) 相同的极限。

条件经验过程收敛

  • Donsker 类是非参数 bootstrap 经验过程一致性的自然场景。
  • Exchangeable bootstrap 权重推广了多项式重抽样。
  • 泛函 delta method 把 bootstrap 过程收敛传递到统计量分布一致性。

Bootstrap 后的 delta method

这一页连接经验过程笔记和统计推断中的 bootstrap：这里研究过程收敛，那里研究置信区间和检验。

Bootstrap 过程准则

定理. 统计量 \(T_n=\phi(P_n)\) 的 bootstrap 一致性常分两步：先证明 \(\sqrt n(P_n^*-P_n)\) 条件模拟 \(\sqrt n(P_n-P)\)，再对 \(\phi\) 使用条件泛函 delta method。

Hadamard 可微与非正则泛函

若统计量是经验过程的非光滑泛函，即使经验过程本身良好，普通 bootstrap 也可能失败。因此必须检查最后映射的连续性或可微性。
