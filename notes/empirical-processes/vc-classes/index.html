12. VC classes

{% include note-topic-nav.html back_label="Back" back_url="/notes/empirical-processes/" prev_label="Last page" prev_url="/notes/empirical-processes/symmetrization-and-rademacher-processes/" next_label="Next page" next_url="/notes/empirical-processes/entropy-of-smooth-classes/" %}

VC theory gives robust entropy bounds for combinatorial classes of sets and subgraphs. It is the quickest route from structure to empirical-process control.

Definition. A class of sets \(\mathcal C\) shatters \(\{x_1,\ldots,x_m\}\) if every subset is realized as \(\{x_i:x_i\in C\}\) for some \(C\in\mathcal C\). The VC dimension is the largest shattered \(m\).

Core facts

  • Finite VC dimension implies polynomial growth of the number of labelings on finite samples.
  • VC-subgraph classes extend the idea from sets to real-valued functions.
  • Permanence rules cover finite unions, intersections, monotone transformations, and many parametric threshold classes.

Use

When a class is defined by inequalities, thresholds, rectangles, halfspaces, or finite-dimensional parameters, check VC structure before estimating entropy from scratch.

12. Classes VC

{% include note-topic-nav.html back_label="Retour" back_url="/notes/empirical-processes/" prev_label="Page précédente" prev_url="/notes/empirical-processes/symmetrization-and-rademacher-processes/" next_label="Page suivante" next_url="/notes/empirical-processes/entropy-of-smooth-classes/" %}

La théorie VC donne des bornes d'entropie robustes pour les classes combinatoires d'ensembles et de sous-graphes.

Définition. Une classe \(\mathcal C\) pulvérise \(\{x_1,\ldots,x_m\}\) si chaque sous-ensemble est réalisé par \(\{x_i:x_i\in C\}\) pour un \(C\in\mathcal C\). La dimension VC est le plus grand \(m\).

Faits clés

  • Une dimension VC finie implique une croissance polynomiale du nombre d'étiquetages possibles.
  • Les classes VC-subgraph étendent l'idée des ensembles aux fonctions réelles.
  • Les règles de permanence couvrent unions, intersections, transformations monotones et classes paramétriques à seuils.

Usage

Pour des classes définies par inégalités, seuils, rectangles, demi-espaces ou paramètres de dimension finie, on vérifie d'abord la structure VC.

12. VC 类

{% include note-topic-nav.html back_label="返回" back_url="/notes/empirical-processes/" prev_label="上一页" prev_url="/notes/empirical-processes/symmetrization-and-rademacher-processes/" next_label="下一页" next_url="/notes/empirical-processes/entropy-of-smooth-classes/" %}

VC 理论为集合类和 subgraph 类给出稳健熵界，是从结构走向经验过程控制的最快路径。

定义. 若 \(\mathcal C\) 能在 \(\{x_1,\ldots,x_m\}\) 上实现所有子集，则称其 shatter 该点集。VC 维是可被 shatter 的最大 \(m\)。

核心事实

  • 有限 VC 维意味着有限样本上可实现标记数只按多项式增长。
  • VC-subgraph 类把集合类概念推广到实值函数类。
  • 永久性规则处理有限并、交、单调变换以及许多参数阈值类。

用途

遇到由不等式、阈值、矩形、半空间或有限维参数定义的类时，先检查 VC 结构，而不是从零估计熵。
