1. Overview and roadmap

{% include note-topic-nav.html back_label="Back" back_url="/notes/empirical-processes/" next_label="Next page" next_url="/notes/empirical-processes/outer-probability-and-measurability/" %}

Empirical-process theory studies the uniform fluctuation of empirical averages over a class of functions. It is the language behind uniform laws, Donsker theorems, bootstrap consistency, and the stochastic part of modern nonparametric inference.

Definition. For i.i.d. observations \(X_1,\ldots,X_n\sim P\), set \(P_n=n^{-1}\sum_{i=1}^n\delta_{X_i}\). The empirical process indexed by \(\mathcal F\) is \(\mathbb G_n f=\sqrt n(P_n-P)f\), \(f\in\mathcal F\).

Empirical-process decomposition

  • Uniform LLNs study \(\|P_n-P\|_{\mathcal F}\); uniform CLTs study weak convergence of \(\mathbb G_n\).
  • Complexity enters through entropy, VC dimension, bracketing, chaining, and concentration.
  • The note follows a pedagogical path from empirical measures to statistical applications, while keeping probability-limit theory and inference-level conclusions in their own notes.

Reading route

Read the first two parts for the minimal toolkit, the complexity part for proving conditions, and the last two parts for bootstrap, goodness-of-fit, M-estimation, and semiparametric interfaces.

Basic decomposition

Proposition. A useful decomposition is \[ P_nf-Pf=(P_n-P)(f-f_0)+(P_n-P)f_0+P(f-f_0), \] where \(f_0\) is a nearby reference function. The first term is controlled by equicontinuity, the second by an ordinary CLT, and the last by deterministic approximation.

How to read later arguments

When reading later pages, keep asking three questions: what is the index class \(\mathcal F\), what metric makes nearby functions statistically close, and what complexity bound makes the supremum small? Most applications are just this template with different answers.

1. Vue d'ensemble et feuille de route

{% include note-topic-nav.html back_label="Retour" back_url="/notes/empirical-processes/" next_label="Page suivante" next_url="/notes/empirical-processes/outer-probability-and-measurability/" %}

La théorie des processus empiriques étudie les fluctuations uniformes des moyennes empiriques sur une classe de fonctions. Elle organise les lois uniformes, les théorèmes de Donsker, le bootstrap et la partie aléatoire de l'inférence non paramétrique.

Définition. Pour des observations i.i.d. \(X_1,\ldots,X_n\sim P\), on pose \(P_n=n^{-1}\sum_{i=1}^n\delta_{X_i}\). Le processus empirique indexé par \(\mathcal F\) est \(\mathbb G_n f=\sqrt n(P_n-P)f\).

Décomposition du processus empirique

  • Les lois uniformes des grands nombres contrôlent \(\|P_n-P\|_{\mathcal F}\); les TCL uniformes contrôlent la convergence faible de \(\mathbb G_n\).
  • La complexité est mesurée par entropie, dimension VC, bracketing, chaînage et concentration.
  • La note va des mesures empiriques aux applications statistiques, en gardant les limites probabilistes générales et l'inférence de base dans leurs notes propres.

Parcours de lecture

Lire les deux premières parties pour les outils minimaux, la partie complexité pour vérifier les conditions, puis les dernières parties pour le bootstrap, les tests d'adéquation, les M-estimateurs et les interfaces semiparamétriques.

Décomposition de base

Proposition. Une décomposition utile est \[ P_nf-Pf=(P_n-P)(f-f_0)+(P_n-P)f_0+P(f-f_0). \] Le premier terme vient de l'équicontinuité, le second d'un TCL ordinaire, et le dernier de l'approximation déterministe.

Comment lire les arguments suivants

En lisant la suite, poser toujours trois questions : quelle est la classe \(\mathcal F\), quelle métrique rend deux fonctions statistiquement proches, et quelle borne de complexité contrôle le supremum?

1. 总览与路线图

{% include note-topic-nav.html back_label="返回" back_url="/notes/empirical-processes/" next_label="下一页" next_url="/notes/empirical-processes/outer-probability-and-measurability/" %}

经验过程理论研究经验平均在整个函数类上的一致波动。它是统一大数定律、唐斯克定理、自助法一致性以及现代非参数推断中随机项控制的共同语言。

定义. 设 \(X_1,\ldots,X_n\sim P\) 独立同分布，\(P_n=n^{-1}\sum_{i=1}^n\delta_{X_i}\)。由 \(\mathcal F\) 索引的经验过程为 \(\mathbb G_n f=\sqrt n(P_n-P)f\)。

经验过程分解

  • 统一大数定律控制 \(\|P_n-P\|_{\mathcal F}\)；统一中心极限定理控制 \(\mathbb G_n\) 的弱收敛。
  • 复杂度通过熵、VC 维、括号化、链式法与 concentration 进入。
  • 本笔记从经验测度走向统计应用，同时把一般概率极限理论和基础统计推断保留在各自笔记中。

阅读路线

前两部分给出最小工具箱；复杂度部分用于验证条件；最后两部分连接自助法、拟合优度、M-估计和半参数接口。

基本分解

命题. 一个常用分解是 \[ P_nf-Pf=(P_n-P)(f-f_0)+(P_n-P)f_0+P(f-f_0). \] 第一项由等连续性控制，第二项由普通 CLT 控制，最后一项是确定性逼近误差。

后续论证的读法

读后面的页面时始终问三件事：索引类 \(\mathcal F\) 是什么，什么距离让函数“统计上接近”，以及什么复杂度界能让 supremum 变小。多数应用只是这个模板的不同版本。
