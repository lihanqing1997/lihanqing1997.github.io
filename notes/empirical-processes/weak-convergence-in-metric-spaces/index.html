3. Weak convergence in metric spaces

{% include note-topic-nav.html back_label="Back" back_url="/notes/empirical-processes/" prev_label="Last page" prev_url="/notes/empirical-processes/outer-probability-and-measurability/" next_label="Next page" next_url="/notes/empirical-processes/bounded-function-spaces/" %}

Empirical processes live as random elements of function spaces. The needed convergence is weak convergence in metric spaces, not just finite-dimensional convergence.

Definition. Random elements \(X_n\) converge weakly to \(X\), written \(X_n\rightsquigarrow X\), if \(E^*h(X_n)\to Eh(X)\) for every bounded continuous \(h\) on the state space.

Core facts

  • Finite-dimensional convergence identifies candidate limits; tightness prevents uncontrolled oscillation.
  • The continuous mapping theorem carries weak convergence through continuous functionals such as suprema, integrals, and argmax maps when the map is continuous at the limit.
  • In empirical-process applications, the metric often comes from \(L_2(P)\), but the sample paths are viewed in \(\ell^\infty(\mathcal F)\).

Use

This page supplies the convergence language used later for Donsker classes, bootstrap empirical processes, and functional delta methods.

3. Convergence faible dans les espaces métriques

{% include note-topic-nav.html back_label="Retour" back_url="/notes/empirical-processes/" prev_label="Page précédente" prev_url="/notes/empirical-processes/outer-probability-and-measurability/" next_label="Page suivante" next_url="/notes/empirical-processes/bounded-function-spaces/" %}

Les processus empiriques sont des éléments aléatoires d'espaces de fonctions. La convergence pertinente est la convergence faible dans un espace métrique, et pas seulement la convergence fini-dimensionnelle.

Définition. On écrit \(X_n\rightsquigarrow X\) si \(E^*h(X_n)\to Eh(X)\) pour toute fonction bornée continue \(h\) sur l'espace d'état.

Faits clés

  • La convergence fini-dimensionnelle identifie la limite possible; la tension empêche les oscillations incontrôlées.
  • Le théorème de l'application continue transporte la convergence vers les fonctionnelles continues aux points pertinents.
  • Dans les applications, la métrique vient souvent de \(L_2(P)\), alors que les trajectoires vivent dans \(\ell^\infty(\mathcal F)\).

Usage

Cette page donne le langage de convergence utilisé ensuite pour les classes de Donsker, le bootstrap empirique et la méthode delta fonctionnelle.

3. 度量空间中的弱收敛

{% include note-topic-nav.html back_label="返回" back_url="/notes/empirical-processes/" prev_label="上一页" prev_url="/notes/empirical-processes/outer-probability-and-measurability/" next_label="下一页" next_url="/notes/empirical-processes/bounded-function-spaces/" %}

经验过程是函数空间中的随机元素。所需的收敛是度量空间中的弱收敛，而不仅是有限维分布收敛。

定义. 若对状态空间上一切有界连续函数 \(h\)，都有 \(E^*h(X_n)\to Eh(X)\)，则写作 \(X_n\rightsquigarrow X\)。

核心事实

  • 有限维收敛确定候选极限；紧性防止过程出现不可控振荡。
  • 连续映射定理把弱收敛传递到在极限点连续的 supremum、积分、argmax 等泛函。
  • 经验过程中的距离常来自 \(L_2(P)\)，但样本路径通常放在 \(\ell^\infty(\mathcal F)\) 中。

用途

这一页提供后续 Donsker 类、bootstrap 经验过程和泛函 delta method 所需的收敛语言。
