27. Sieve and penalized estimators

{% include note-topic-nav.html back_label="Back" back_url="/notes/empirical-processes/" prev_label="Last page" prev_url="/notes/empirical-processes/rates-of-m-estimators/" next_label="Next page" next_url="/notes/empirical-processes/semiparametric-and-random-empirical-processes/" %}

Sieve and penalized estimators control infinite-dimensional problems by restricting or penalizing complexity.

Definition. A sieve estimator optimizes over \(\Theta_n\subset\Theta\). A penalized estimator optimizes \(P_nm_\theta-\lambda_nJ(\theta)\), where \(J\) measures complexity.

Core facts

  • Approximation error decreases as \(\Theta_n\) grows; stochastic error increases with entropy.
  • Penalty terms localize the estimator to sublevel sets with smaller effective entropy.
  • Oracle inequalities compare the selected estimator with the best available tradeoff in the class.

Use

This page is the interface to density estimation, nonparametric regression, penalized least squares, classification, support vector machines, and lasso-type methods.

27. Estimateurs par tamis et pénalisés

{% include note-topic-nav.html back_label="Retour" back_url="/notes/empirical-processes/" prev_label="Page précédente" prev_url="/notes/empirical-processes/rates-of-m-estimators/" next_label="Page suivante" next_url="/notes/empirical-processes/semiparametric-and-random-empirical-processes/" %}

Les estimateurs par tamis et pénalisés traitent les problèmes de dimension infinie en restreignant ou pénalisant la complexité.

Définition. Un estimateur par tamis optimise sur \(\Theta_n\subset\Theta\). Un estimateur pénalisé optimise \(P_nm_\theta-\lambda_nJ(\theta)\), où \(J\) mesure la complexité.

Faits clés

  • L'erreur d'approximation diminue quand \(\Theta_n\) grandit; l'erreur stochastique augmente avec l'entropie.
  • Les pénalités localisent l'estimateur dans des sous-niveaux d'entropie effective plus petite.
  • Les inégalités oracle comparent l'estimateur au meilleur compromis disponible.

Usage

Cette page sert d'interface vers estimation de densité, régression non paramétrique, moindres carrés pénalisés, classification, SVM et méthodes de type lasso.

27. Sieve 与惩罚估计量

{% include note-topic-nav.html back_label="返回" back_url="/notes/empirical-processes/" prev_label="上一页" prev_url="/notes/empirical-processes/rates-of-m-estimators/" next_label="下一页" next_url="/notes/empirical-processes/semiparametric-and-random-empirical-processes/" %}

Sieve 和惩罚估计通过限制或惩罚复杂度来处理无限维问题。

定义. Sieve 估计量在 \(\Theta_n\subset\Theta\) 上优化。惩罚估计量优化 \(P_nm_\theta-\lambda_nJ(\theta)\)，其中 \(J\) 衡量复杂度。

核心事实

  • 随着 \(\Theta_n\) 增大，逼近误差下降，但随机误差随熵增加。
  • 惩罚项把估计量局部化到有效熵更小的 sublevel sets。
  • Oracle inequalities 将所选估计量与类中最佳权衡进行比较。

用途

这一页连接密度估计、非参数回归、惩罚最小二乘、分类、SVM 和 lasso 型方法。
