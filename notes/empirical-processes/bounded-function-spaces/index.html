4. Bounded function spaces

{% include note-topic-nav.html back_label="Back" back_url="/notes/empirical-processes/" prev_label="Last page" prev_url="/notes/empirical-processes/weak-convergence-in-metric-spaces/" next_label="Next page" next_url="/notes/empirical-processes/empirical-measures-and-processes/" %}

The canonical state space for a process indexed by \(\mathcal F\) is \(\ell^\infty(\mathcal F)\), the space of all bounded real functions on \(\mathcal F\).

Definition. \(\ell^\infty(T)=\{z:T\to\mathbb R:\|z\|_T=\sup_{t\in T}|z(t)|<\infty\}\), equipped with the uniform norm \(\|\cdot\|_T\).

Core facts

  • A sample path \(f\mapsto \mathbb G_n f\) is bounded when the indexed supremum is finite.
  • Weak convergence in \(\ell^\infty(\mathcal F)\) is stronger than convergence of every finite vector \((\mathbb G_n f_1,\ldots,\mathbb G_n f_k)\).
  • Nonseparability is common, which is why outer probability and asymptotic equicontinuity are part of the basic toolkit.

Use

Whenever later pages write \(\mathbb G_n\rightsquigarrow\mathbb G\), the default meaning is convergence as bounded functions on the relevant index class.

Criterion to remember

Guide. The uniform norm is strong: convergence in \(\ell^\infty(\mathcal F)\) controls all coordinates simultaneously. This is why it is suitable for confidence bands and sup-norm test statistics, not merely for a fixed finite list of functions.

Example and proof guide

Think of an element of \(\ell^\infty(\mathcal F)\) as a table with one entry for each \(f\). Finite-dimensional convergence checks finitely many entries; tightness says the whole table is not hiding wild oscillations between nearby columns.

4. Espaces de fonctions bornées

{% include note-topic-nav.html back_label="Retour" back_url="/notes/empirical-processes/" prev_label="Page précédente" prev_url="/notes/empirical-processes/weak-convergence-in-metric-spaces/" next_label="Page suivante" next_url="/notes/empirical-processes/empirical-measures-and-processes/" %}

L'espace d'état canonique d'un processus indexé par \(\mathcal F\) est \(\ell^\infty(\mathcal F)\), l'espace des fonctions réelles bornées sur \(\mathcal F\).

Définition. \(\ell^\infty(T)=\{z:T\to\mathbb R:\|z\|_T=\sup_{t\in T}|z(t)|<\infty\}\), muni de la norme uniforme \(\|\cdot\|_T\).

Faits clés

  • Une trajectoire \(f\mapsto \mathbb G_n f\) est bornée lorsque le supremum indexé est fini.
  • La convergence faible dans \(\ell^\infty(\mathcal F)\) est plus forte que la convergence de chaque vecteur fini.
  • La non-séparabilité est fréquente; d'où le rôle de la probabilité extérieure et de l'équicontinuité asymptotique.

Usage

Quand les pages suivantes écrivent \(\mathbb G_n\rightsquigarrow\mathbb G\), il s'agit par défaut de convergence comme fonctions bornées sur la classe d'indices.

Critère à retenir

Guide. La norme uniforme est forte : la convergence dans \(\ell^\infty(\mathcal F)\) contrôle toutes les coordonnées simultanément. Elle convient donc aux bandes de confiance et aux statistiques en norme supremum.

Exemple et guide de preuve

On peut voir un élément de \(\ell^\infty(\mathcal F)\) comme un tableau indexé par \(f\). La convergence fini-dimensionnelle vérifie quelques colonnes; la tension contrôle les oscillations entre colonnes proches.

4. 有界函数空间

{% include note-topic-nav.html back_label="返回" back_url="/notes/empirical-processes/" prev_label="上一页" prev_url="/notes/empirical-processes/weak-convergence-in-metric-spaces/" next_label="下一页" next_url="/notes/empirical-processes/empirical-measures-and-processes/" %}

由 \(\mathcal F\) 索引的过程通常取值于 \(\ell^\infty(\mathcal F)\)，即 \(\mathcal F\) 上所有有界实函数组成的空间。

定义. \(\ell^\infty(T)=\{z:T\to\mathbb R:\|z\|_T=\sup_{t\in T}|z(t)|<\infty\}\)，配备一致范数 \(\|\cdot\|_T\)。

核心事实

  • 当索引 supremum 有限时，\(f\mapsto \mathbb G_n f\) 是一条有界样本路径。
  • \(\ell^\infty(\mathcal F)\) 中的弱收敛强于任意有限向量的分布收敛。
  • 非可分性很常见，因此外概率与渐近等连续性是基础工具。

用途

后续写 \(\mathbb G_n\rightsquigarrow\mathbb G\) 时，默认指它们作为相关索引类上的有界函数收敛。

需要记住的准则

指南. 一致范数很强：\(\ell^\infty(\mathcal F)\) 中的收敛同时控制所有坐标。因此它适合置信带和 sup-norm 检验统计量，而不只是固定有限个函数。

例子与证明思路

可以把 \(\ell^\infty(\mathcal F)\) 的元素看成以 \(f\) 为列的表。有限维收敛只检查有限列；紧性说明相邻列之间没有隐藏的剧烈振荡。
