The canonical state space for a process indexed by \(\mathcal F\) is \(\ell^\infty(\mathcal F)\), the space of all bounded real functions on \(\mathcal F\).

Definition. \(\ell^\infty(T)=\{z:T\to\mathbb R:\|z\|_T=\sup_{t\in T}|z(t)|<\infty\}\), equipped with the uniform norm \(\|\cdot\|_T\).

Uniform norm

  • A sample path \(f\mapsto \mathbb G_n f\) is bounded when the indexed supremum is finite.
  • Weak convergence in \(\ell^\infty(\mathcal F)\) is stronger than convergence of every finite vector \((\mathbb G_n f_1,\ldots,\mathbb G_n f_k)\).
  • Nonseparability is common, which is why outer probability and asymptotic equicontinuity are part of the basic toolkit.

Separation and tightness

Whenever later pages write \(\mathbb G_n\rightsquigarrow\mathbb G\), the default meaning is convergence as bounded functions on the relevant index class.

Uniform convergence in ell-infinity

Proposition. The uniform norm is strong: convergence in \(\ell^\infty(\mathcal F)\) controls all coordinates simultaneously. This is why it is suitable for confidence bands and sup-norm test statistics, not merely for a fixed finite list of functions.

Finite coordinates versus full process

Think of an element of \(\ell^\infty(\mathcal F)\) as a table with one entry for each \(f\). Finite-dimensional convergence checks finitely many entries; tightness says the whole table is not hiding wild oscillations between nearby columns.
