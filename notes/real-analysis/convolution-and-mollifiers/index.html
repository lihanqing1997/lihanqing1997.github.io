45. Convolution and mollifiers

{% include note-topic-nav.html back_label="Back" back_url="/notes/real-analysis/" prev_label="Last page" prev_url="/notes/real-analysis/approximation-by-smooth-and-simple-functions/" next_label="Next page" next_url="/notes/real-analysis/fourier-series-and-orthogonality/" %}

Convolution averages a function by translation. Mollifiers use this operation to create smooth approximations to rough functions.

Convolution

Definition. For functions on \(\mathbb R^d\), \((f*g)(x)=\int f(x-y)g(y)\,\mathrm{d}y\) when the integral exists. Convolution is commutative under standard integrability assumptions.

Boundedness on Lp

Theorem. If \(g\in L^1\), then \(\|f*g\|_p\le\|f\|_p\|g\|_1\) for \(1\le p\le\infty\). Thus integrable kernels act as bounded smoothing operators.

Mollifiers

Definition. A mollifier is a nonnegative \(\rho\in C_c^\infty\) with \(\int \rho\,\mathrm{d}x=1\). Set \(\rho_\varepsilon(x)=\varepsilon^{-d}\rho(x/\varepsilon)\).

Approximate identity

Theorem. If \(f\in L^p(\mathbb R^d)\) with \(1\le p<\infty\), then \(\rho_\varepsilon*f\to f\) in \(L^p\) as \(\varepsilon\downarrow0\).

Kernel smoothing

Remark. Kernel density estimators are empirical convolutions. Deterministic mollifier theory explains bias and smoothness, while probability controls stochastic fluctuation.
