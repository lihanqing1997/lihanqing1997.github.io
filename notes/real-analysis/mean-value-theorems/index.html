7. Mean value theorems

{% include note-topic-nav.html back_label="Back" back_url="/notes/real-analysis/" prev_label="Last page" prev_url="/notes/real-analysis/differentiability-and-taylor-expansion/" next_label="Next page" next_url="/notes/real-analysis/riemann-integration/" %}

Mean value theorems convert local derivative information into global statements on intervals. They are the bridge between infinitesimal slope and finite change.

Rolle theorem

Theorem. If \(f\) is continuous on \([a,b]\), differentiable on \((a,b)\), and \(f(a)=f(b)\), then there exists \(c\in(a,b)\) with \(f'(c)=0\). The proof uses the extreme value theorem: an interior maximum or minimum has derivative zero unless the function is constant.

Lagrange mean value theorem

Theorem. If \(f\) is continuous on \([a,b]\) and differentiable on \((a,b)\), then there exists \(c\in(a,b)\) such that \[ f'(c)=\frac{f(b)-f(a)}{b-a}. \] Apply Rolle's theorem to \(f(x)-\frac{f(b)-f(a)}{b-a}(x-a)\).

Monotonicity and derivative sign

Corollary. If \(f'\ge0\) on an interval, then \(f\) is nondecreasing. If \(f'=0\) everywhere on an interval, then \(f\) is constant. If \(f'\le g'\) and \(f(a)\le g(a)\), then \(f\le g\) to the right of \(a\).

Cauchy mean value theorem and lHopital rule

Theorem. If \(f,g\) are continuous on \([a,b]\), differentiable on \((a,b)\), and \(g'\) never vanishes, then for some \(c\), \[ \frac{f(b)-f(a)}{g(b)-g(a)}=\frac{f'(c)}{g'(c)}. \] This theorem is the structural source of l'Hopital-type rules, after adding limiting hypotheses.

Convexity through slopes

Proposition. For differentiable \(f\) on an interval, convexity is equivalent to the monotonicity of \(f'\), and also to \(f(y)\ge f(x)+f'(x)(y-x)\) for all \(x,y\). Thus the tangent line is a global supporting line for a differentiable convex function.

Why intervals matter

Counterexample. If a domain is disconnected, derivative information on each component need not compare values across components. A function with derivative zero on \((0,1)\cup(2,3)\) is constant on each component but can take different constants on the two components. The interval hypothesis is essential.
