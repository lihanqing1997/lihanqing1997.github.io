20. Bounded variation and absolute continuity

{% include note-topic-nav.html back_label="Back" back_url="/notes/real-analysis/" prev_label="Last page" prev_url="/notes/real-analysis/riemann-stieltjes-integration/" next_label="Next page" next_url="/notes/real-analysis/fundamental-theorem-of-calculus-and-change-of-variables/" %}

Bounded variation and absolute continuity separate functions with finite accumulated movement from functions that are genuine integrals of their derivatives.

Total variation

Definition. For \(f:[a,b]\to\mathbb R\), the total variation is the supremum of \(\sum |f(x_i)-f(x_{i-1})|\) over partitions. Finite total variation means the graph has finite accumulated vertical movement.

Jordan decomposition of functions

Theorem. A real function on \([a,b]\) has bounded variation if and only if it is the difference of two increasing functions. This is the function analogue of Jordan decomposition for signed measures.

Absolute continuity

Definition. A function is absolutely continuous if small total interval length forces small total oscillation over every finite disjoint collection of intervals. This is stronger than uniform continuity and weaker than continuous differentiability.

Fundamental theorem

Theorem. The function \(f\) is absolutely continuous if and only if \(f(x)=f(a)+\int_a^x g(t)\,\mathrm{d}t\) for some \(g\in L^1\). Then \(f^\prime=g\) almost everywhere.

Cantor function

Counterexample. The Cantor function is continuous, increasing, and has derivative zero almost everywhere, but it is not constant. It has bounded variation but is not absolutely continuous.
