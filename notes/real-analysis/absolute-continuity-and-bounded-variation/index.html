23. Absolute continuity and bounded variation

{% include note-topic-nav.html back_label="Back" back_url="/notes/real-analysis/" prev_label="Last page" prev_url="/notes/real-analysis/approximation-by-smooth-and-simple-functions/" next_label="Next page" next_url="/notes/real-analysis/lebesgue-differentiation-theorem/" %}

Absolute continuity and bounded variation refine ordinary continuity by measuring how much a function can move across many intervals. They are the correct hypotheses for the Lebesgue version of the fundamental theorem of calculus.

Bounded variation

Definition. For \(f:[a,b]\to\mathbb R\), the total variation is \[ V_a^b(f)=\sup_P\sum_{i=1}^n |f(x_i)-f(x_{i-1})|, \] where the supremum is over partitions \(P\). The function has bounded variation if this quantity is finite.

Jordan decomposition for BV functions

Theorem. A real function on \([a,b]\) has bounded variation if and only if it can be written as the difference of two increasing functions. The variation function \(v(x)=V_a^x(f)\) gives one decomposition: \(f=(v+f)/2-(v-f)/2\), after checking monotonicity.

Absolute continuity

Definition. A function \(f:[a,b]\to\mathbb R\) is absolutely continuous if for every \(\varepsilon>0\) there is \(\delta>0\) such that for every finite disjoint family of intervals \((a_k,b_k)\), \(\sum_k(b_k-a_k)<\delta\) implies \(\sum_k|f(b_k)-f(a_k)|<\varepsilon\).

Relations among regularity classes

Proposition. Absolute continuity implies uniform continuity and bounded variation. Lipschitz functions are absolutely continuous. Bounded variation functions may have jumps, and continuous bounded-variation functions need not be absolutely continuous, as the Cantor function shows.

Fundamental theorem for Lebesgue integrals

Theorem. A function \(f:[a,b]\to\mathbb R\) is absolutely continuous if and only if there is \(g\in L^1([a,b])\) such that \(f(x)=f(a)+\int_a^x g(t)\,\mathrm{d}t\). In that case \(f'=g\) almost everywhere.

Cantor function

Example. The Cantor function is continuous, nondecreasing, and maps \([0,1]\) onto \([0,1]\), but its derivative is \(0\) almost everywhere. It has bounded variation but is not absolutely continuous, showing that continuity plus almost-everywhere derivative information is not enough.
