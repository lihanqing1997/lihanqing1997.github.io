14. Lebesgue integral

{% include note-topic-nav.html back_label="Back" back_url="/notes/real-analysis/" prev_label="Last page" prev_url="/notes/real-analysis/lebesgue-measurable-functions/" next_label="Next page" next_url="/notes/real-analysis/convergence-theorems/" %}

The Lebesgue integral is built by integrating simple functions and then passing to monotone limits. It is designed so limiting operations are controlled by measure-theoretic convergence theorems.

Integral of simple functions

Definition. For a nonnegative simple function \(s=\sum_{k=1}^m a_k\mathbf 1_{A_k}\), define \(\int s\,\mathrm{d}\mu=\sum_{k=1}^m a_k\mu(A_k)\), using a representation with disjoint measurable sets. This definition is independent of the chosen disjoint representation.

Nonnegative measurable functions

Definition. For \(f\ge0\) measurable, define \[ \int f\,\mathrm{d}\mu=\sup\left\{\int s\,\mathrm{d}\mu:0\le s\le f,\ s\text{ simple}\right\}. \] The value may be \(\infty\). This definition is monotone by construction.

Signed and complex integrable functions

Definition. For real \(f\), write \(f=f^+-f^-\), where \(f^+=\max(f,0)\) and \(f^-=\max(-f,0)\). The function is integrable if \(\int f^+\,\mathrm{d}\mu\) and \(\int f^-\,\mathrm{d}\mu\) are finite, equivalently \(\int |f|\,\mathrm{d}\mu<\infty\). Complex functions are integrable when their absolute value is integrable.

Linearity and monotonicity

Proposition. The Lebesgue integral is linear on integrable functions, monotone on nonnegative functions, and satisfies \(|\int f\,\mathrm{d}\mu|\le\int |f|\,\mathrm{d}\mu\). If \(f=0\) almost everywhere, then \(\int f\,\mathrm{d}\mu=0\); conversely, if \(f\ge0\) and \(\int f\,\mathrm{d}\mu=0\), then \(f=0\) almost everywhere.

Layer-cake formula

Theorem. For \(f\ge0\), \[ \int f\,\mathrm{d}\mu=\int_0^\infty \mu(\{x:f(x)>t\})\,\mathrm{d}t, \] whenever the right side is understood as a Lebesgue integral on \([0,\infty)\). This formula converts integrals of nonnegative functions into integrals of tail measures.

Comparison with Riemann integration

Remark. Every Riemann integrable function on a compact interval is Lebesgue integrable, and the two integrals agree. Lebesgue integration is more flexible because it integrates by measuring level sets rather than by partitioning the domain into intervals.
