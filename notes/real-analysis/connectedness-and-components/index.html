10. Connectedness, intervals, and components

{% include note-topic-nav.html back_label="Back" back_url="/notes/real-analysis/" prev_label="Last page" prev_url="/notes/real-analysis/complete-metric-spaces-and-baire-category/" next_label="Next page" next_url="/notes/real-analysis/continuity-and-uniform-continuity/" %}

Connectedness expresses the impossibility of splitting a space into two separated open pieces. On the real line, connected sets are exactly intervals.

Connected spaces

Definition. A space is connected if it cannot be written as a union of two disjoint nonempty open sets. Equivalently, the only clopen subsets are the empty set and the whole space.

Intervals are connected

Theorem. A subset of \(\mathbb R\) is connected if and only if it is an interval. Missing an intermediate point separates the set into left and right parts; conversely, a separation would create a forbidden boundary cut inside an interval.

Intermediate value theorem

Corollary. A continuous image of a connected set is connected. Therefore a continuous real function on an interval takes every value between any two values it attains.

Path-connectedness

Definition. A space is path-connected if any two points can be joined by a continuous map from \([0,1]\). Path-connectedness implies connectedness; convex subsets of \(\mathbb R^d\) are path-connected by line segments.

Parameter-space interpretation

Remark. Connected and convex parameter spaces allow sign, root, and deformation arguments. Disconnected parameter spaces often produce separate local analyses rather than one global argument.
