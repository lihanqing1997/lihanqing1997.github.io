10. Connectedness, intervals, and components

{% include note-topic-nav.html back_label="Back" back_url="/notes/real-analysis/" prev_label="Last page" prev_url="/notes/real-analysis/complete-metric-spaces-and-baire-category/" next_label="Next page" next_url="/notes/real-analysis/continuity-and-uniform-continuity/" %}

Connectedness formalizes the idea that a space has no separation into two open pieces. On the real line this recovers intervals and the intermediate value theorem.

Separations and connected spaces

A separation is a genuine topological split.

Definition. A topological space \(X\) is connected if it cannot be written as \(X=U\cup V\), where \(U,V\) are disjoint nonempty sets that are open in the relative topology of \(X\).

Proposition. A space \(X\) is connected if and only if the only subsets that are both open and closed are \(\varnothing\) and \(X\).

Proof. If \(A\) is nonempty, proper, open, and closed, then \(A\) and \(X\setminus A\) form a separation. Conversely, a separation \(X=U\cup V\) makes \(U\) both open and closed, nonempty and proper.

Intervals are the connected subsets of the line

Order and connectedness coincide in \(\mathbb R\).

Theorem. A subset \(E\subset\mathbb R\) is connected if and only if it is an interval, meaning that \(x,z\in E\) and \(x\lt y\lt z\) imply \(y\in E\).

Proof. If \(E\) misses a point \(y\) between \(x,z\in E\), then \(E\cap(-\infty,y)\) and \(E\cap(y,\infty)\) separate \(E\). Conversely, if an interval \(E\) is separated by \(U,V\), choose \(u\in U\), \(v\in V\) with \(u\lt v\), and let \(s=\sup(U\cap[u,v])\). Interval order and relative openness force \(s\) to belong to neither side or to allow crossing past \(s\), contradiction.

Continuous images and intermediate values

Continuous functions cannot tear connected sets apart.

Theorem. The continuous image of a connected set is connected. Hence if \(f:[a,b]\to\mathbb R\) is continuous and \(f(a)\lt y\lt f(b)\), then \(f(c)=y\) for some \(c\in(a,b)\).

Proof. If \(f(X)\) had a separation \(U\cup V\), then \(f^{-1}(U)\) and \(f^{-1}(V)\) would separate \(X\). The image \(f([a,b])\) is therefore a connected subset of \(\mathbb R\), hence an interval containing \(f(a)\) and \(f(b)\), so it contains \(y\).

Components

Components are maximal connected pieces.

Definition. The connected component of \(x\in X\) is the union of all connected subsets of \(X\) containing \(x\).

Proposition. Components are connected, closed, pairwise disjoint, and partition the space.

Proof. The union of connected sets with a common point is connected, giving connectedness and maximality. Distinct components cannot meet, because their union would then be connected. The closure of a connected set is connected, so maximality implies each component contains its closure.

Path connectedness

Path connectedness is stronger than connectedness, though the two agree for intervals and many Euclidean domains.

Definition. A space is path connected if for every \(x,y\) there is a continuous map \(\gamma:[0,1]\to X\) with \(\gamma(0)=x\) and \(\gamma(1)=y\).

Proposition. Every path connected space is connected.

Proof. If \(X=U\cup V\) were a separation and \(x\in U,y\in V\), a path from \(x\) to \(y\) would have connected image in \(X\). That image would be split by \(U\) and \(V\), contradicting connectedness of \([0,1]\).
