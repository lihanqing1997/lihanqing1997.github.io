21. Egorov, Lusin, and approximation theorems

{% include note-topic-nav.html back_label="Back" back_url="/notes/real-analysis/" prev_label="Last page" prev_url="/notes/real-analysis/modes-of-convergence/" next_label="Next page" next_url="/notes/real-analysis/approximation-by-smooth-and-simple-functions/" %}

Egorov and Lusin are regularity theorems: they say that measurable convergence and measurable functions become almost uniform and almost continuous after discarding a small set.

Egorov theorem

Theorem. If \(\mu(X)<\infty\) and \(f_n\to f\) almost everywhere, then for every \(\varepsilon>0\) there is a measurable set \(E\) with \(\mu(E)<\varepsilon\) such that \(f_n\to f\) uniformly on \(X\setminus E\). Almost everywhere convergence becomes almost uniform convergence on finite measure spaces.

Proof idea for Egorov

Proof. For each \(k,m\), define \(E_{k,m}=\bigcup_{n\ge m}\{|f_n-f|>1/k\}\). For fixed \(k\), these sets decrease to a null set as \(m\to\infty\). Choose \(m_k\) so that \(\mu(E_{k,m_k})<\varepsilon2^{-k}\). Outside \(\bigcup_k E_{k,m_k}\), convergence is uniform.

Lusin theorem

Theorem. If \(f\) is finite-valued and measurable on a finite-measure subset of \(\mathbb R^d\), then for every \(\varepsilon>0\) there is a compact set \(K\) with \(\mu(X\setminus K)<\varepsilon\) such that \(f|_K\) is continuous. Measurable functions are nearly continuous on large compact sets.

Simple-function approximation

Proposition. Every nonnegative measurable function is the pointwise increasing limit of simple functions. If \(f\) is integrable, simple functions can approximate \(f\) in \(L^1\). This is the basic approximation layer underneath Lebesgue integration.

Approximation by continuous functions

Proposition. On \(\mathbb R^d\), if \(f\in L^p\) for \(1\le p<\infty\), then continuous compactly supported functions approximate \(f\) in \(L^p\). One first approximates by simple functions on finite-measure sets, then approximates measurable sets from inside and outside by compact and open sets.

Finite measure is essential

Remark. Egorov's theorem can fail on infinite measure spaces. The functions \(f_n=\mathbf 1_{[n,\infty)}\) on \(\mathbb R\) converge pointwise to \(0\), but no finite-measure exceptional set makes convergence uniform on its complement.
