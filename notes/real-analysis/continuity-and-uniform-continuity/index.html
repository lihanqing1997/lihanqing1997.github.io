5. Continuity and uniform continuity

{% include note-topic-nav.html back_label="Back" back_url="/notes/real-analysis/" prev_label="Last page" prev_url="/notes/real-analysis/metric-spaces-and-compactness/" next_label="Next page" next_url="/notes/real-analysis/differentiability-and-taylor-expansion/" %}

Continuity says small input changes produce small output changes near a point. Uniform continuity asks for one input scale that works everywhere on the domain.

Pointwise continuity

Definition. A function \(f:X\to Y\) between metric spaces is continuous at \(x\) if for every \(\varepsilon>0\) there is \(\delta>0\) such that \(d_X(x,y)<\delta\) implies \(d_Y(f(x),f(y))<\varepsilon\). It is continuous if it is continuous at every point.

Sequential and topological criteria

Proposition. For metric spaces, \(f\) is continuous at \(x\) if and only if \(x_n\to x\) implies \(f(x_n)\to f(x)\). Equivalently, \(f^{-1}(U)\) is open for every open \(U\) in the target. The sequential criterion is often best for examples; the open-set criterion is best for structure.

Uniform continuity

Definition. A function \(f:X\to Y\) is uniformly continuous if for every \(\varepsilon>0\) there is \(\delta>0\) such that \(d_X(x,y)<\delta\) implies \(d_Y(f(x),f(y))<\varepsilon\) for all \(x,y\in X\). The same \(\delta\) must work across the whole domain.

Heine-Cantor theorem

Theorem. Every continuous function from a compact metric space to a metric space is uniformly continuous. If uniform continuity failed, one could find \(x_n,y_n\) with \(d(x_n,y_n)\to0\) but \(d(f(x_n),f(y_n))\ge\varepsilon_0\). Compactness gives a convergent subsequence, and continuity gives the contradiction.

Compactness consequences

Corollary. A continuous real-valued function on a compact set is bounded and attains its extrema. If the target is \(\mathbb R^m\), the image is compact. These results explain why compact domains are the natural setting for existence problems.

Continuity without uniform continuity

Counterexample. The function \(f(x)=x^2\) is continuous on \(\mathbb R\) but not uniformly continuous. Given any \(\delta>0\), choose large \(x\) and \(y=x+\delta/2\); then \(|x-y|<\delta\) while \(|x^2-y^2|\) can be made arbitrarily large.
