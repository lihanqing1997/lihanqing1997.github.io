Measurable functions are functions compatible with sigma-algebras. They are the objects that can be integrated, approximated by simple functions, and compared almost everywhere.

Measurability criteria

For real-valued functions it is enough to test half-lines.

Theorem. A function \(f:X\to\mathbb R\) is measurable if and only if \(\{x:f(x)>a\}\in\mathcal A\) for every \(a\in\mathbb R\). Equivalently one may use \(\{f\le a\}\), \(\{f\lt a\}\), or \(\{f\ge a\}\).

Proof. The inverse images of intervals \((a,\infty)\) generate the Borel sigma-algebra on \(\mathbb R\), so measurability implies the condition. Conversely, if inverse images of these generators are measurable, the class of Borel sets with measurable inverse image is a sigma-algebra containing the generators.

Simple approximation

Nonnegative measurable functions are monotone limits of simple functions.

Theorem. If \(f\ge0\) is measurable, there exist nonnegative simple functions \(s_n\uparrow f\) pointwise.

Proof. Define \(s_n(x)=2^{-n}\lfloor 2^n\min(f(x),n)\rfloor\). Each \(s_n\) takes finitely many values and is measurable because its level sets are described by inverse images of intervals under \(f\). The construction increases to \(f\) pointwise.

Almost-everywhere equality

Measure theory identifies functions that differ only on null sets.

Definition. Functions \(f,g\) are equal almost everywhere if \(\mu(\{f\ne g\})=0\). A property holds almost everywhere if it fails only on a null set.

Proposition. If \(f=g\) almost everywhere and \(f\) is measurable on a complete measure space, then \(g\) is measurable.

Proof. For every \(a\), the symmetric difference between \(\{g>a\}\) and \(\{f>a\}\) is contained in \(\{f\ne g\}\). Completeness makes the exceptional subset measurable, so \(\{g>a\}\) is measurable.

Convergence in measure

Convergence in measure ignores small exceptional sets whose measure vanishes.

Definition. \(f_n\to f\) in measure if for every \(\varepsilon>0\), \(\mu(|f_n-f|>\varepsilon)\to0\).

Theorem. If \(f_n\to f\) almost everywhere and \(\mu(X)\lt \infty\), then \(f_n\to f\) in measure.

Proof. For fixed \(\varepsilon>0\), the sets \(A_n=\bigcup_{k\ge n}\{|f_k-f|>\varepsilon\}\) decrease to a null set by almost-everywhere convergence. Since \(\mu(X)\lt \infty\), continuity from above gives \(\mu(A_n)\to0\). Then \(\mu(|f_n-f|>\varepsilon)\le\mu(A_n)\).

Subsequence principle

Convergence in measure contains almost-everywhere convergence along subsequences.

Theorem. If \(f_n\to f\) in measure, then some subsequence converges to \(f\) almost everywhere.

Proof. Choose \(n_k\) so that \(\mu(|f_{n_k}-f|>2^{-k})\lt 2^{-k}\). The sum of these measures is finite, so Borel-Cantelli for measures implies that almost every \(x\) belongs to only finitely many exceptional sets. Hence \(|f_{n_k}(x)-f(x)|\le2^{-k}\) eventually for almost every \(x\).
