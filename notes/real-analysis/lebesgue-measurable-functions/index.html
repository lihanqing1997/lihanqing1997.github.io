13. Lebesgue measurable functions

{% include note-topic-nav.html back_label="Back" back_url="/notes/real-analysis/" prev_label="Last page" prev_url="/notes/real-analysis/outer-measures-and-lebesgue-measure/" next_label="Next page" next_url="/notes/real-analysis/lebesgue-integral/" %}

Measurable functions are the functions compatible with measurable sets. They are the correct class for integration because they are stable under limits and can be approximated by simple functions.

Measurable functions

Definition. A function \(f:(X,\mathcal A)\to\mathbb R\) is measurable if \(f^{-1}(B)\in\mathcal A\) for every Borel set \(B\subset\mathbb R\). It is enough to check the threshold sets \(\{x:f(x)>a\}\), or \(\{x:f(x)\le a\}\), for all \(a\in\mathbb R\).

Closure properties

Proposition. If \(f,g\) are measurable, then \(af+bg\), \(fg\), \(\max(f,g)\), \(\min(f,g)\), \(|f|\), and \(f^+\), \(f^-\) are measurable whenever the expressions are defined. These facts follow by reducing algebraic inequalities to countable combinations of measurable threshold sets.

Limsup and pointwise limits

Theorem. If \(f_n\) are measurable, then \(\sup_n f_n\), \(\inf_n f_n\), \(\limsup_n f_n\), \(\liminf_n f_n\), and every pointwise limit are measurable, with extended real values allowed. Countable operations are exactly why \(\sigma\)-algebras are used.

Simple functions

Definition. A simple function has finite range and can be written \(s=\sum_{k=1}^m a_k\mathbf 1_{A_k}\) with measurable \(A_k\). Simple functions are measurable and serve as the finite-dimensional approximations from which the Lebesgue integral is built.

Simple approximation theorem

Theorem. If \(f\ge0\) is measurable, then there exists an increasing sequence of nonnegative simple functions \(s_n\) with \(s_n(x)\uparrow f(x)\) for every \(x\). One construction partitions the range into dyadic intervals and truncates values above \(n\).

Almost everywhere equality

Remark. If \(f=g\) outside a null set and \(f\) is measurable, then \(g\) is measurable after modifying it on a measurable null set. In measure theory, functions are often identified up to almost everywhere equality because integration cannot see null-set changes.
