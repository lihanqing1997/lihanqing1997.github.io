1. Natural numbers, integers, rationals, and induction

{% include note-topic-nav.html back_label="Back" back_url="/notes/real-analysis/" next_label="Next page" next_url="/notes/real-analysis/ordered-fields-and-completeness/" %}

Real analysis starts before the continuum: induction, order, and rational arithmetic provide the discrete skeleton that later has to be completed. The goal is to see exactly which limiting operations \(\mathbb Q\) cannot support.

Peano induction and recursion

Definition. The natural numbers are governed by a successor operation and the induction principle: if a property holds at \(1\) and passes from \(n\) to \(n+1\), it holds for every \(n \in \mathbb N\). Recursive definitions of sums, products, powers, and finite sequences all rest on this principle.

Formal differences and quotients

Construction. The integers can be built from formal differences \((a,b)\) modulo \((a,b)\sim(c,d)\) when \(a+d=b+c\). The rationals are then formal quotients \((m,n)\) with \(n \ne 0\), modulo cross multiplication. This creates the ordered field \(\mathbb Q\).

The square-root gap

Counterexample. The set \(\{q \in \mathbb Q:q>0, q^2<2\}\) is bounded above in \(\mathbb Q\) but has no rational least upper bound. If a rational supremum existed, it would square to \(2\), impossible by the parity argument.

Why this matters for analysis

Remark. Limits, infinite series, nested intervals, and Cauchy sequences can point outside \(\mathbb Q\). Real analysis begins by adding exactly enough numbers so these limiting procedures no longer leave the system.
