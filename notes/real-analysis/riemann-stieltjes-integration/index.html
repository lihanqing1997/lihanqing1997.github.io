19. Riemann-Stieltjes integration

{% include note-topic-nav.html back_label="Back" back_url="/notes/real-analysis/" prev_label="Last page" prev_url="/notes/real-analysis/riemann-integration/" next_label="Next page" next_url="/notes/real-analysis/absolute-continuity-and-bounded-variation/" %}

Riemann-Stieltjes integration replaces interval length by increments of another function. It is the classical doorway to distribution functions and Stieltjes measures.

Stieltjes sums

Definition. Given \(f\) and increasing \(\alpha\) on \([a,b]\), sums have the form \(\sum f(t_i)(\alpha(x_i)-\alpha(x_{i-1}))\). Their limiting value, when it exists, is written \(\int_a^b f\,\mathrm{d}\alpha\).

Continuous integrands

Theorem. If \(f\) is continuous and \(\alpha\) is increasing, then the Riemann-Stieltjes integral exists. Uniform continuity of \(f\) controls oscillation, while monotonicity gives finite total mass.

Jumps as atoms

Example. A jump \(\Delta \alpha(c)\) contributes \(f(c)\Delta \alpha(c)\) to the integral. Thus point masses already appear in classical Stieltjes integration.

Integration by parts

Theorem. Under standard hypotheses, \(\int f\,\mathrm{d}g+\int g\,\mathrm{d}f=f(b)g(b)-f(a)g(a)\), with endpoint conventions adjusted for jumps.

Lebesgue-Stieltjes viewpoint

Remark. A right-continuous increasing function \(F\) defines a measure \(\mu_F\) with \(\mu_F((a,b])=F(b)-F(a)\). This measure-theoretic formulation handles limits more cleanly than partitions.
