20. Modes of convergence

{% include note-topic-nav.html back_label="Back" back_url="/notes/real-analysis/" prev_label="Last page" prev_url="/notes/real-analysis/lp-spaces/" next_label="Next page" next_url="/notes/real-analysis/egorov-lusin-and-approximation-theorems/" %}

A sequence of functions can converge in many ways. Each mode controls different behavior, and most mistakes in analysis come from silently replacing one mode by a stronger one.

Almost everywhere and pointwise convergence

Definition. A sequence \(f_n\) converges to \(f\) almost everywhere if the set of \(x\) where \(f_n(x)\) fails to converge to \(f(x)\) has measure zero. This is pointwise convergence after ignoring a null exceptional set.

Convergence in measure

Definition. A sequence \(f_n\) converges to \(f\) in measure if for every \(\varepsilon>0\), \(\mu(\{|f_n-f|>\varepsilon\})\to0\). This allows large errors on sets whose measure tends to zero.

\(L^p\) and uniform convergence

Definition. Convergence in \(L^p\) means \(\|f_n-f\|_p\to0\). Uniform convergence means \(\sup_x|f_n(x)-f(x)|\to0\). Uniform convergence controls pointwise behavior everywhere; \(L^p\) convergence controls average size with respect to the measure.

Implication diagram

Proposition. On a finite measure space, almost everywhere convergence implies convergence in measure, and \(L^p\) convergence implies convergence in measure. Uniform convergence implies pointwise convergence and, on finite measure spaces with bounded measure, implies \(L^p\) convergence. The converses generally fail.

Subsequence principle

Theorem. If \(f_n\to f\) in measure, then some subsequence converges to \(f\) almost everywhere. Moreover, \(f_n\to f\) in measure if and only if every subsequence has a further subsequence converging to \(f\) almost everywhere.

Moving spike example

Counterexample. Let \(f_n=n\mathbf 1_{(0,1/n)}\) on \((0,1)\). Then \(f_n\to0\) almost everywhere and in measure, but not in \(L^1\), since \(\|f_n\|_1=1\). Let \(g_n=\mathbf 1_{(0,1/n)}\); then \(g_n\to0\) in every \(L^p\), but not uniformly.
