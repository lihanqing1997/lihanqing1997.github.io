38. Modes of convergence for functions

{% include note-topic-nav.html back_label="Back" back_url="/notes/real-analysis/" prev_label="Last page" prev_url="/notes/real-analysis/differentiation-of-measures-and-density-theorems/" next_label="Next page" next_url="/notes/real-analysis/uniform-integrability-and-vitali-theorem/" %}

Different modes of convergence encode different kinds of control: pointwise, almost everywhere, in measure, in norm, and uniformly. Knowing the implications prevents false limit interchanges.

Almost everywhere and in measure

Almost-everywhere convergence is pointwise outside a null set; convergence in measure controls exceptional sets quantitatively.

Definition. \(f_n\to f\) almost everywhere if \(\mu(\{x:f_n(x)\not\to f(x)\})=0\). It converges in measure if \(\mu(|f_n-f|>\varepsilon)\to0\) for every \(\varepsilon>0\).

Theorem. On a finite measure space, almost-everywhere convergence implies convergence in measure.

Proof. For fixed \(\varepsilon\), define \(A_n=\bigcup_{k\ge n}\{|f_k-f|>\varepsilon\}\). Then \(A_n\downarrow A\), where \(A\) is contained in the null set of nonconvergence. Since \(\mu(X)\lt \infty\), \(\mu(A_n)\downarrow0\), and \(\mu(|f_n-f|>\varepsilon)\le\mu(A_n)\).

Lp convergence

Norm convergence controls average size of errors.

Definition. For \(1\le p\lt \infty\), \(f_n\to f\) in \(L^p\) if \(\int |f_n-f|^p\,\mathrm{d}\mu\to0\). For \(p=\infty\), use essential sup norm.

Proposition. If \(f_n\to f\) in \(L^p\) for \(1\le p\lt \infty\), then \(f_n\to f\) in measure.

Proof. Markov's inequality gives \(\mu(|f_n-f|>\varepsilon)\le\varepsilon^{-p}\int|f_n-f|^p\,\mathrm{d}\mu\), which tends to zero.

Subsequence principle

Convergence in measure is almost-everywhere convergence after subselection.

Theorem. If \(f_n\to f\) in measure, then some subsequence converges to \(f\) almost everywhere.

Proof. Choose \(n_k\) with \(\mu(|f_{n_k}-f|>2^{-k})\lt 2^{-k}\). The sum of the exceptional measures is finite. By Borel-Cantelli, almost every point belongs to only finitely many exceptional sets, so \(f_{n_k}(x)\to f(x)\) almost everywhere.

Uniform convergence

Uniform convergence is stronger than pointwise convergence and preserves continuity.

Proposition. If \(f_n\to f\) uniformly and each \(f_n\) is continuous on a metric space, then \(f\) is continuous.

Proof. Fix \(x\) and \(\varepsilon>0\). Choose \(n\) with \(\|f_n-f\|_\infty\lt \varepsilon/3\), then choose a neighborhood of \(x\) on which \(f_n\) varies by less than \(\varepsilon/3\). The triangle inequality gives continuity of \(f\).

Implication map

No single mode dominates all others without hypotheses.

Remark. On finite measure spaces, \(L^p\) convergence implies convergence in measure, and almost-everywhere convergence implies convergence in measure. Conversely, convergence in measure gives almost-everywhere convergence only along subsequences. Uniform convergence implies pointwise convergence but is usually much stronger than measure convergence.
