14. Equicontinuity and Arzela-Ascoli theorem

{% include note-topic-nav.html back_label="Back" back_url="/notes/real-analysis/" prev_label="Last page" prev_url="/notes/real-analysis/sequences-of-functions/" next_label="Next page" next_url="/notes/real-analysis/differentiability-and-taylor-expansion/" %}

Arzela-Ascoli is compactness for families of functions. It says boundedness plus a common continuity modulus can replace finite-dimensional boundedness.

Equicontinuity

Definition. A family \(\mathcal F\) is equicontinuous if every point has neighborhoods on which all functions in the family oscillate by less than a prescribed tolerance. Uniform equicontinuity uses one neighborhood size for all points.

Pointwise relative compactness

Definition. The family is pointwise relatively compact if \(\{f(x):f\in \mathcal F\}\) has compact closure for each \(x\). For scalar-valued functions this usually means pointwise boundedness.

Arzela-Ascoli theorem

Theorem. For compact metric \(K\), a subset of \(C(K)\) has compact closure in the sup norm exactly when it is bounded and equicontinuous. The theorem turns local oscillation control into uniform subsequential convergence.

Diagonal proof mechanism

Proof. Choose a countable dense subset of \(K\), extract subsequences converging at those points by diagonalization, and then use equicontinuity to extend convergence uniformly from the dense set to all of \(K\).

Statistical bridge

Remark. Stochastic equicontinuity is a probabilistic version of this idea: control a process on a finite net, then control oscillations between nearby points.
