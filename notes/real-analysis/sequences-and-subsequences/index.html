6. Sequences, subsequences, and asymptotic notation

{% include note-topic-nav.html back_label="Back" back_url="/notes/real-analysis/" prev_label="Last page" prev_url="/notes/real-analysis/completeness-principles/" next_label="Next page" next_url="/notes/real-analysis/series-and-absolute-convergence/" %}

Sequential language is the working grammar of real analysis. Limits, subsequences, limsup, and asymptotic notation are the tools used to control approximation and extract structure from infinite objects.

Epsilon definition of convergence

Convergence means all sufficiently late terms lie in every prescribed neighborhood of the limit.

Definition. A sequence \(x_n\) converges to \(x\), written \(x_n\to x\), if for every \(\varepsilon>0\) there is \(N\) such that \(n\ge N\Rightarrow |x_n-x|\lt \varepsilon\).

Proposition. Limits are unique and every convergent sequence is bounded.

Proof. If \(x_n\to x\) and \(x_n\to y\), then \(|x-y|\le |x-x_n|+|x_n-y|\), and the right side can be made arbitrarily small, so \(x=y\). Boundedness follows by bounding the tail in a unit neighborhood of the limit and taking the maximum of finitely many initial absolute values.

Subsequences and cluster points

Subsequences reveal the possible limiting behaviors hidden inside a sequence.

Definition. A subsequence has the form \(x_{n_k}\) with \(n_1\lt n_2\lt \cdots\). A point \(x\) is a cluster point if some subsequence converges to \(x\).

Theorem. A sequence \(x_n\) converges to \(x\) if and only if every subsequence has a further subsequence converging to \(x\).

Proof. If \(x_n\to x\), every subsequence also converges to \(x\). Conversely, if \(x_n\) does not converge to \(x\), then for some \(\varepsilon>0\) there are infinitely many \(n\) with \(|x_n-x|\ge\varepsilon\). Those terms form a subsequence with no further subsequence converging to \(x\).

Limsup and liminf

Limsup and liminf record the eventual upper and lower envelopes.

Definition. For a real sequence, \(\limsup x_n=\inf_N\sup_{n\ge N}x_n\) and \(\liminf x_n=\sup_N\inf_{n\ge N}x_n\), with extended real values allowed.

Proposition. A bounded sequence converges if and only if \(\limsup x_n=\liminf x_n\), and the common value is the limit.

Proof. If \(x_n\to x\), then every tail eventually lies in \((x-\varepsilon,x+\varepsilon)\), so both envelopes lie within \(\varepsilon\) of \(x\). Conversely, if the two envelopes equal \(x\), then for every \(\varepsilon>0\) the tail supremum is \(\lt x+\varepsilon\) and the tail infimum is \(>x-\varepsilon\), forcing all sufficiently late terms into that interval.

Cauchy criterion

Completeness allows convergence to be checked internally.

Theorem. A real sequence converges if and only if for every \(\varepsilon>0\) there is \(N\) such that \(m,n\ge N\Rightarrow |x_m-x_n|\lt \varepsilon\).

Proof. The forward implication follows from the triangle inequality. For the reverse implication, Cauchy sequences are bounded and therefore have convergent subsequences. If \(x_{n_k}\to x\), the Cauchy property gives \(|x_n-x|\le |x_n-x_{n_k}|+|x_{n_k}-x|\) for large \(n,k\).

Asymptotic notation

Asymptotic notation packages comparison estimates without naming constants at every line.

Definition. Write \(a_n=O(b_n)\) if \(|a_n|\le C|b_n|\) eventually for some \(C\lt \infty\), and \(a_n=o(b_n)\) if \(a_n/b_n\to0\) along indices where \(b_n\ne0\).

Remark. In proofs, \(O(\cdot)\) gives boundedness after scaling, while \(o(\cdot)\) gives a vanishing error term. Keeping the quantifiers visible prevents mistakes in estimates with parameters.
