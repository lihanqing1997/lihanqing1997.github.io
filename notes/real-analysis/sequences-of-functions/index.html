13. Sequences of functions and uniform convergence

{% include note-topic-nav.html back_label="Back" back_url="/notes/real-analysis/" prev_label="Last page" prev_url="/notes/real-analysis/semicontinuity-and-extreme-values/" next_label="Next page" next_url="/notes/real-analysis/equicontinuity-and-arzela-ascoli/" %}

Function limits depend on topology. Pointwise, uniform, compact-uniform, measure, norm, and weak convergence preserve different operations.

Pointwise and uniform convergence

Definition. Functions \(f_n:X\to Y\) converge pointwise to \(f\) if \(f_n(x)\to f(x)\) for each \(x\). They converge uniformly if \(\sup_{x\in X}d(f_n(x),f(x))\to0\).

Uniform limit theorem

Theorem. A uniform limit of continuous functions is continuous. The proof adds and subtracts one fixed \(f_N\): uniform closeness handles the tails, and continuity of \(f_N\) handles the local change.

Integrals under uniform convergence

Proposition. If \(f_n\to f\) uniformly on \([a,b]\) and the functions are Riemann integrable, then \(\int_a^b f_n(x)\,\mathrm{d}x \to \int_a^b f(x)\,\mathrm{d}x\). The error is bounded by interval length times the sup norm.

Derivatives under uniform convergence

Theorem. If \(f_n(x_0)\) converges at one point and \(f_n^\prime\) converges uniformly to \(g\), then \(f_n\) converges uniformly to a differentiable \(f\) with \(f^\prime=g\). Differentiation requires stronger hypotheses than integration.

Weierstrass test

Proposition. If \(|f_n(x)|\le M_n\) for all \(x\) and \(\sum M_n<\infty\), then \(\sum f_n\) converges uniformly and absolutely. This is the basic domination principle for function series.
