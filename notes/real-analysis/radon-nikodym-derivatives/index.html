16. Radon-Nikodym derivatives

{% include note-topic-nav.html back_label="Back" back_url="/notes/real-analysis/" prev_label="Last page" prev_url="/notes/real-analysis/product-measures-and-fubini-theorem/" next_label="Next page" next_url="/notes/real-analysis/lp-spaces/" %}

Radon-Nikodym derivatives records the objects, main results, proof mechanisms, and boundary cases needed for the real analysis note.

Definition

Definition. A measure \(\nu\) is absolutely continuous with respect to \(\mu\), written \(\nu\ll\mu\), if \(\mu(A)=0\) implies \(\nu(A)=0\).

Radon-Nikodym theorem

Theorem. If \(\nu\ll\mu\) and both measures are \(\sigma\)-finite, then there exists a measurable \(f\ge0\) such that \(\nu(A)=\int_A f\,\mathrm{d}\mu\). The function \(f\) is unique up to \(\mu\)-null sets.

Proof

Proof. The proof uses a Hilbert-space argument for finite measures or a decomposition into level sets; uniqueness follows because two densities with equal integrals over all sets must agree almost everywhere.

Remark

Remark. Radon-Nikodym derivatives are densities in invariant form. Likelihood ratios in statistics are a direct application.
