18. Riemann and Darboux integration

{% include note-topic-nav.html back_label="Back" back_url="/notes/real-analysis/" prev_label="Last page" prev_url="/notes/real-analysis/convex-functions-and-monotone-functions/" next_label="Next page" next_url="/notes/real-analysis/riemann-stieltjes-integration/" %}

Riemann integration approximates area by partitions of the domain. Darboux upper and lower sums identify precisely when these approximations agree.

Upper and lower sums

Definition. For bounded \(f\) on \([a,b]\), a partition produces upper sums \(U(f,P)\) and lower sums \(L(f,P)\) from suprema and infima on subintervals. Riemann integrability means the best upper and lower sums coincide.

Darboux criterion

Theorem. A bounded function is Riemann integrable if and only if for every \(\varepsilon>0\) there is a partition \(P\) with \(U(f,P)-L(f,P)<\varepsilon\).

Lebesgue criterion

Theorem. A bounded function on a compact interval is Riemann integrable if and only if its discontinuity set has Lebesgue measure zero. This theorem exactly marks the boundary between Riemann and Lebesgue integration.

Fundamental theorem compatibility

Proposition. Continuous functions and monotone functions on compact intervals are Riemann integrable. If \(F^\prime=f\) and \(f\) is Riemann integrable, then \(\int_a^b f(x)\,\mathrm{d}x=F(b)-F(a)\).

Dirichlet function

Counterexample. The indicator of \(\mathbb Q\cap[0,1]\) has lower sums \(0\) and upper sums \(1\) for every partition, so it is not Riemann integrable. Lebesgue integration assigns it integral \(0\).
