8. Riemann integration

{% include note-topic-nav.html back_label="Back" back_url="/notes/real-analysis/" prev_label="Last page" prev_url="/notes/real-analysis/mean-value-theorems/" next_label="Next page" next_url="/notes/real-analysis/sequences-of-functions/" %}

Riemann integration approximates area by finite sums over intervals. It is concrete and geometrically intuitive, but its convergence theory is limited compared with Lebesgue integration.

Partitions and Darboux sums

Definition. For a bounded function \(f:[a,b]\to\mathbb R\) and partition \(P:a=x_0<\cdots

Riemann criterion

Theorem. A bounded function is Riemann integrable on \([a,b]\) if and only if for every \(\varepsilon>0\) there is a partition \(P\) such that \(U(f,P)-L(f,P)<\varepsilon\). Equivalently, sufficiently fine tagged sums become close to a single number independent of tags.

Continuous and monotone functions

Proposition. Every continuous function on \([a,b]\) is Riemann integrable, because compactness gives uniform continuity and therefore small oscillation on sufficiently short intervals. Every monotone function on \([a,b]\) is Riemann integrable because the sum of interval oscillations is controlled by the total variation \(f(b)-f(a)\).

Lebesgue criterion for Riemann integrability

Theorem. A bounded function on \([a,b]\) is Riemann integrable if and only if its set of discontinuities has Lebesgue measure zero. This theorem precisely explains which bounded functions Riemann integration can handle.

Dirichlet and Thomae examples

Example. The indicator of \(\mathbb Q\cap[0,1]\) is discontinuous everywhere and is not Riemann integrable. Thomae's function, defined by \(f(p/q)=1/q\) in lowest terms and \(f(x)=0\) for irrational \(x\), is discontinuous at rationals and continuous at irrationals, hence Riemann integrable with integral \(0\).

Fundamental theorem for Riemann integrals

Theorem. If \(f\) is continuous and \(F(x)=\int_a^x f(t)\,\mathrm{d}t\), then \(F'=f\). If \(F\) is continuously differentiable, then \(\int_a^b F'(t)\,\mathrm{d}t=F(b)-F(a)\). Later, Lebesgue theory will weaken these hypotheses using absolute continuity.
