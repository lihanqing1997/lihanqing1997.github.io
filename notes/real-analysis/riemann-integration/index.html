Riemann integration records the objects, main results, proof mechanisms, and boundary cases needed for the real analysis note.

Definition

Definition. The Riemann integral approximates area by sums over tagged partitions. Integrability means upper and lower sums can be made arbitrarily close.

Lebesgue criterion for Riemann integrability

Theorem. A bounded function on \([a,b]\) is Riemann integrable if its set of discontinuities has Lebesgue measure zero. Continuous and monotone functions are therefore Riemann integrable.

Proof

Proof. Oscillation on subintervals controls the gap between upper and lower sums. Cover the discontinuity set by intervals with small total length and use small oscillation elsewhere.

Remark

Remark. Riemann integration is concrete and useful, but measure theory gives the cleaner convergence theorems needed in modern analysis and probability.
