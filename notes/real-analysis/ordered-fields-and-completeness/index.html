1. Ordered fields and completeness

{% include note-topic-nav.html back_label="Back" back_url="/notes/real-analysis/" next_label="Next page" next_url="/notes/real-analysis/sequences-and-subsequences/" %}

Ordered fields and completeness records the objects, main results, proof mechanisms, and boundary cases needed for the real analysis note.

Definition

Definition. The real line is the complete ordered field: every nonempty subset bounded above has a least upper bound. Completeness is the axiom distinguishing \(\mathbb R\) from \(\mathbb Q\).

Completeness principles

Theorem. Least-upper-bound completeness, Cauchy completeness, nested interval completeness, and monotone convergence are equivalent formulations for the real line.

Proof

Proof. Use order to pass between suprema and monotone sequences; use intervals to locate limits; use Cauchy estimates to build nested intervals. Each formulation reconstructs the missing limit.

Remark

Remark. Real analysis is analysis of limiting processes under completeness. Without completeness, many intuitive limit arguments fail silently.
