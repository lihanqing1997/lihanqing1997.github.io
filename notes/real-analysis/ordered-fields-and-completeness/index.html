1. Ordered fields and completeness

{% include note-topic-nav.html back_label="Back" back_url="/notes/real-analysis/" next_label="Next page" next_url="/notes/real-analysis/sequences-and-subsequences/" %}

Real analysis begins with the fact that the real line is ordered and complete. Completeness is the principle that prevents Cauchy processes, nested intervals, and monotone bounded sequences from escaping the number system.

Ordered fields and order intervals

Definition. An ordered field is a field equipped with a total order compatible with addition and multiplication by positive elements. Intervals, upper bounds, lower bounds, suprema, and infima are order-theoretic notions before any metric is mentioned. Least upper bound property

Theorem. Every nonempty subset \(A\subset\mathbb R\) that is bounded above has a least upper bound \(\sup A\). Equivalently, every nonempty bounded-below subset has an infimum. This axiom is false in \(\mathbb Q\): the set \(\{q\in\mathbb Q:q^2<2\}\) is bounded above in \(\mathbb Q\) but has no rational supremum.

Equivalent completeness principles

Theorem. For the real line, the following are equivalent: the least-upper-bound property, monotone convergence for bounded monotone sequences, Cauchy completeness, nested compact interval completeness, and Bolzano-Weierstrass. Each form is used in a different proof style, but all express the same absence of gaps.

Nested interval argument

Proof. Assume least-upper-bound completeness and let \([a_n,b_n]\) be nested closed intervals with \(b_n-a_n\to0\). The set \(\{a_n:n\ge1\}\) is bounded above, so let \(x=\sup_n a_n\). Since \(a_n\le b_m\) whenever \(n\ge m\), one gets \(x\le b_m\) for every \(m\), and therefore \(x\in\bigcap_m[a_m,b_m]\). If another point \(y\) is in every interval, then \(|x-y|\le b_m-a_m\) for all \(m\), hence \(x=y\).

Completeness as a warning

Counterexample. The decimal approximations to \(\sqrt2\) form a Cauchy sequence in \(\mathbb Q\) but have no rational limit. Thus algebraic closure under arithmetic is not enough for analysis; one needs closure under limiting operations.
