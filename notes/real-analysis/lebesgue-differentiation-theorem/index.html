24. Lebesgue differentiation theorem

{% include note-topic-nav.html back_label="Back" back_url="/notes/real-analysis/" prev_label="Last page" prev_url="/notes/real-analysis/absolute-continuity-and-bounded-variation/" next_label="Next page" next_url="/notes/real-analysis/fourier-series-and-orthogonality/" %}

The Lebesgue differentiation theorem says that an integrable function can be recovered almost everywhere from averages over shrinking intervals or balls. It is the local converse to integration.

Averages over shrinking balls

Definition. For \(f\in L^1_{\mathrm{loc}}(\mathbb R^d)\), define the local average over a ball by \[ A_r f(x)=\frac{1}{|B(x,r)|}\int_{B(x,r)} f(y)\,\mathrm{d}y. \] The question is whether \(A_r f(x)\to f(x)\) as \(r\downarrow0\).

Hardy-Littlewood maximal function

Definition. The centered maximal function is \(Mf(x)=\sup_{r>0} |B(x,r)|^{-1}\int_{B(x,r)}|f(y)|\,\mathrm{d}y\). Maximal inequalities control the size of the set where averages are large and are the main tool in the differentiation theorem.

Weak type maximal inequality

Theorem. For \(f\in L^1(\mathbb R^d)\), there is a constant \(C_d\) such that \[ \left|\{x:Mf(x)>\lambda\}\right|\le \frac{C_d}{\lambda}\|f\|_1,\qquad \lambda>0. \] The proof uses a covering lemma to extract almost disjoint balls from a large-level set cover.

Lebesgue differentiation theorem

Theorem. If \(f\in L^1_{\mathrm{loc}}(\mathbb R^d)\), then \[ \lim_{r\downarrow0}\frac{1}{|B(x,r)|}\int_{B(x,r)} |f(y)-f(x)|\,\mathrm{d}y=0 \] for almost every \(x\). In particular, averages of \(f\) over shrinking balls converge to \(f(x)\) almost everywhere.

Lebesgue points

Definition. A point satisfying the conclusion of the differentiation theorem is called a Lebesgue point of \(f\). Almost every point is a Lebesgue point. At such points, \(f(x)\) is the genuine local value of \(f\), despite functions being defined only up to null sets in \(L^1\).

Connection with absolute continuity

Corollary. If \(F(x)=\int_a^x f(t)\,\mathrm{d}t\) with \(f\in L^1\), then \(F'(x)=f(x)\) at almost every Lebesgue point of \(f\). This is the analytic heart of the fundamental theorem for Lebesgue integrals.
