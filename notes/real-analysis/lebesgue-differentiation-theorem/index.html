36. Lebesgue differentiation theorem

{% include note-topic-nav.html back_label="Back" back_url="/notes/real-analysis/" prev_label="Last page" prev_url="/notes/real-analysis/distribution-functions-and-lebesgue-stieltjes-measures/" next_label="Next page" next_url="/notes/real-analysis/differentiation-of-measures-and-density-theorems/" %}

The Lebesgue differentiation theorem says that an integrable function is recovered almost everywhere from averages over shrinking neighborhoods.

Averages over balls

Differentiation asks whether local averages converge back to the pointwise value.

Definition. For \(f\in L^1_{\mathrm{loc}}(\mathbb R^d)\), define the ball average \[ A_rf(x)=\frac{1}{m(B(x,r))}\int_{B(x,r)}f(y)\,\mathrm{d}y. \]

Theorem. For \(f\in L^1_{\mathrm{loc}}(\mathbb R^d)\), \(A_rf(x)\to f(x)\) for Lebesgue-almost every \(x\) as \(r\downarrow0\).

Proof. Approximate \(f\) in \(L^1\) on a large ball by a continuous compactly supported function \(g\). The averages of \(g\) converge uniformly to \(g(x)\) by uniform continuity. The error is controlled by the Hardy-Littlewood maximal function applied to \(|f-g|\), whose weak \(L^1\) bound makes the exceptional set small. Let the approximation error tend to zero.

Hardy-Littlewood maximal function

The maximal function controls the worst local average.

Definition. The centered maximal function is \(Mf(x)=\sup_{r>0}m(B(x,r))^{-1}\int_{B(x,r)}|f(y)|\,\mathrm{d}y\).

Theorem. There is a constant \(C_d\) such that for \(f\in L^1(\mathbb R^d)\), \[ m(\{Mf>\lambda\})\le \frac{C_d}{\lambda}\|f\|_1. \]

Proof. For each \(x\) with \(Mf(x)>\lambda\), choose a ball on which the average exceeds \(\lambda\). A Vitali covering lemma selects disjoint balls whose controlled enlargements cover the level set. Summing \(\lambda m(B_i)\lt \int_{B_i}|f|\) over the disjoint balls gives the estimate.

Lebesgue points

Almost every point is a point of local \(L^1\) continuity.

Definition. A point \(x\) is a Lebesgue point of \(f\) if \[ \frac{1}{m(B(x,r))}\int_{B(x,r)}|f(y)-f(x)|\,\mathrm{d}y\to0. \]

Corollary. Almost every point is a Lebesgue point of every \(f\in L^1_{\mathrm{loc}}(\mathbb R^d)\).

Proof. Apply the differentiation theorem to \(|f-c|\) over sets where \(f(x)\) is approximated by constants from a countable dense set, or use the standard approximation by continuous functions plus maximal inequality directly for local oscillations.

Density theorem

Sets have density one at almost all of their points.

Theorem. If \(E\subset\mathbb R^d\) is measurable, then for almost every \(x\in E\), \[ \frac{m(E\cap B(x,r))}{m(B(x,r))}\to1. \]

Proof. Apply the Lebesgue differentiation theorem to \(f=\mathbf 1_E\). At almost every point, the averages of \(\mathbf 1_E\) converge to \(\mathbf 1_E(x)\), which is \(1\) for almost every \(x\in E\).

Recovering Radon-Nikodym densities

Differentiation turns measure derivatives into local mass ratios.

Remark. If \(\nu=f\,m\), then \(\nu(B(x,r))/m(B(x,r))\to f(x)\) at almost every \(x\). This is why Radon-Nikodym derivatives deserve to be called densities.
