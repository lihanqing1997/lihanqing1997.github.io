44. Approximation by simple, continuous, and smooth functions

{% include note-topic-nav.html back_label="Back" back_url="/notes/real-analysis/" prev_label="Last page" prev_url="/notes/real-analysis/egorov-lusin-and-approximation-theorems/" next_label="Next page" next_url="/notes/real-analysis/convolution-and-mollifiers/" %}

Dense nice subclasses let one prove theorems first for manageable functions and then pass to general functions by convergence.

Simple functions dense in Lp

Theorem. For \(1\le p<\infty\), simple functions are dense in \(L^p\). Truncation controls tails, and finite quantization controls bounded parts.

Continuous compact support

Theorem. In \(\mathbb R^d\), \(C_c(\mathbb R^d)\) is dense in \(L^p\) for \(1\le p<\infty\). The proof uses regularity of Lebesgue measure.

Smooth compact support

Theorem. \(C_c^\infty(\mathbb R^d)\) is dense in \(L^p(\mathbb R^d)\) for \(1\le p<\infty\). Convolution with mollifiers smooths continuous compactly supported approximants.

Indicator approximation

Example. If \(\lambda(\partial A)=0\), indicators of \(A\) can be approximated in \(L^1\) by continuous functions from inside and outside. Boundary measure controls the error.

Statistical approximation

Remark. Sieve estimators, histograms, basis expansions, and kernel smoothers are statistical versions of approximation by finite-dimensional or smooth subclasses.
