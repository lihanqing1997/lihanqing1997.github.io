12. Outer measures and Lebesgue measure

{% include note-topic-nav.html back_label="Back" back_url="/notes/real-analysis/" prev_label="Last page" prev_url="/notes/real-analysis/sigma-algebras-and-measures/" next_label="Next page" next_url="/notes/real-analysis/lebesgue-measurable-functions/" %}

Lebesgue measure is built by first assigning an outer size to every set, then selecting the sets for which this size interacts correctly with complements. This construction explains why measurability is a theorem, not just a label.

Outer measure

Definition. An outer measure on \(X\) is a function \(\mu^*:\mathcal P(X)\to[0,\infty]\) with \(\mu^*(\varnothing)=0\), monotonicity, and countable subadditivity: \(\mu^*(\bigcup_n E_n)\le\sum_n\mu^*(E_n)\). It is defined on all subsets, before measurability is imposed.

Lebesgue outer measure

Definition. For \(E\subset\mathbb R\), define \[ m^*(E)=\inf\left\{\sum_{n=1}^\infty |I_n|:E\subset\bigcup_{n=1}^\infty I_n,\ I_n\text{ open intervals}\right\}. \] This covers arbitrary sets by countably many intervals and asks for the cheapest total length.

Caratheodory measurability

Definition. A set \(E\) is \(\mu^*\)-measurable if for every \(A\subset X\), \(\mu^*(A)=\mu^*(A\cap E)+\mu^*(A\setminus E)\). Measurable sets are precisely those that split every test set without losing or creating outer measure.

Caratheodory extension theorem

Theorem. The \(\mu^*\)-measurable sets form a \(\sigma\)-algebra, and \(\mu^*\) restricted to this \(\sigma\)-algebra is a measure. For Lebesgue outer measure on \(\mathbb R\), all Borel sets are measurable, and the restriction is Lebesgue measure.

Intervals and translation invariance

Proposition. Lebesgue measure assigns \(m((a,b))=b-a\), is countably additive on Lebesgue measurable sets, and is translation invariant: \(m(E+t)=m(E)\). The interval normalization plus translation invariance are what make it the ordinary length measure.

Cantor set

Example. The middle-thirds Cantor set is closed, uncountable, and has Lebesgue measure zero. At stage \(n\), it is covered by \(2^n\) intervals of length \(3^{-n}\), so its outer measure is at most \((2/3)^n\) for every \(n\). This separates cardinality from measure size.
