Lebesgue measure is built by first assigning an outer size to every set, then selecting the sets whose interaction with outer size is consistent.

Outer measure

Outer measures are defined on all subsets, but only subadditively.

Definition. An outer measure on \(X\) is a function \(\mu^*:2^X\to[0,\infty]\) with \(\mu^*(\varnothing)=0\), monotonicity, and countable subadditivity.

Proposition. If \(\ell(I)\) is interval length, then \[ m^*(E)=\inf\left\{\sum_n\ell(I_n):E\subset\bigcup_n I_n\right\} \] defines an outer measure on \(\mathbb R\).

Proof. The empty set is covered with total length zero. Monotonicity follows because every cover of a larger set covers a smaller one. For subadditivity, cover each \(E_k\) within \(\varepsilon2^{-k}\) of its outer measure; the union of all covers covers \(\bigcup_kE_k\), and summing gives the inequality after \(\varepsilon\downarrow0\).

Caratheodory measurability

Measurable sets are those that split every set additively.

Definition. A set \(A\subset X\) is \(\mu^*\)-measurable if for every \(E\subset X\), \[ \mu^*(E)=\mu^*(E\cap A)+\mu^*(E\setminus A). \]

Theorem. The \(\mu^*\)-measurable sets form a sigma-algebra, and \(\mu^*\) restricted to it is a complete measure.

Proof. Complements are immediate from the criterion. Finite unions follow by splitting \(E\) successively and using subadditivity for the reverse inequality. Countable unions are obtained by applying finite additivity to disjoint partial unions and then passing to the limit. Subsets of outer-null sets satisfy the criterion by monotonicity.

Lebesgue measurable sets

Lebesgue measure is the Caratheodory measure induced by interval covers.

Theorem. All Borel subsets of \(\mathbb R\) are Lebesgue measurable, and the resulting measure agrees with interval length on intervals.

Proof. One first checks that intervals satisfy the Caratheodory criterion by separating covers across interval endpoints with arbitrarily small loss. Since measurable sets form a sigma-algebra containing intervals, they contain the Borel sigma-algebra. The equality with interval length follows from covering intervals and compactness arguments for the reverse inequality.

Translation invariance

Lebesgue measure respects the additive structure of the line.

Proposition. For every \(a\in\mathbb R\), \(m(E+a)=m(E)\) for Lebesgue measurable \(E\).

Proof. Translating interval covers preserves total length, so \(m^*(E+a)=m^*(E)\). Translation also preserves the Caratheodory criterion, hence measurable sets remain measurable and their measures agree.

Null sets and completion

Lebesgue measure contains more sets than the Borel sigma-algebra.

Remark. Every subset of the Cantor set is Lebesgue measurable because the Cantor set has measure zero, but not every subset of the Cantor set is Borel. This is why completion matters in analysis and probability.
