26. Outer measures and Lebesgue measure

{% include note-topic-nav.html back_label="Back" back_url="/notes/real-analysis/" prev_label="Last page" prev_url="/notes/real-analysis/sigma-algebras-and-measures/" next_label="Next page" next_url="/notes/real-analysis/borel-and-radon-measures/" %}

Lebesgue measure is built by assigning an outer size to every set, then selecting the sets that split all other sets additively.

Outer measure

Definition. An outer measure \(\mu^*\) is defined on all subsets, is monotone, vanishes at the empty set, and is countably subadditive. It need not be countably additive on all subsets.

Caratheodory measurability

Definition. A set \(E\) is measurable if \(\mu^*(A)=\mu^*(A\cap E)+\mu^*(A\setminus E)\) for every \(A\). Measurable sets split outer measure without loss.

Caratheodory theorem

Theorem. The measurable sets form a sigma-algebra, and the restriction of \(\mu^*\) to that sigma-algebra is a complete measure. This converts subadditive outer size into additive measure.

Lebesgue outer measure

Construction. For \(E\subset\mathbb R^d\), cover \(E\) by countably many boxes and take the infimum of the sums of their volumes. The resulting measure on Caratheodory sets is Lebesgue measure.

Translation and scaling

Proposition. Lebesgue measure satisfies \(\lambda(E+x)=\lambda(E)\) and \(\lambda(cE)=|c|^d\lambda(E)\). These properties follow first for boxes and then pass through the outer-measure construction.
