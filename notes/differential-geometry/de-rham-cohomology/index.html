9. De Rham cohomology

{% include note-topic-nav.html back_label="Back" back_url="/notes/differential-geometry/" prev_label="Last page" prev_url="/notes/differential-geometry/stokes-theorem/" next_label="Next page" next_url="/notes/differential-geometry/lie-derivative-and-cartan-formula/" %}

De Rham cohomology records the objects, main results, proof mechanisms, and boundary cases needed for the differential geometry note.

Definition

Definition. A form is closed if \(\mathrm{d}\omega=0\) and exact if \(\omega=\mathrm{d}\eta\). De Rham cohomology is \(H^k_{\mathrm{dR}}(M)=\ker \mathrm{d}/\operatorname{im}\mathrm{d}\).

Proposition

Proposition. De Rham cohomology is invariant under smooth homotopy equivalence. Contractible manifolds have the same de Rham cohomology as a point.

Proof

Proof. Construct chain homotopy operators from a smooth homotopy and use Cartan-type formulas to show pullbacks by homotopic maps induce the same map on cohomology.

Remark

Remark. Cohomology measures global obstruction to solving \(\omega=\mathrm{d}\eta\) for closed forms. It is topology seen by calculus.
