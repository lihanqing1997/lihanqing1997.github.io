4. Tensor fields

{% include note-topic-nav.html back_label="Back" back_url="/notes/differential-geometry/" prev_label="Last page" prev_url="/notes/differential-geometry/vector-fields-and-flows/" next_label="Next page" next_url="/notes/differential-geometry/submanifolds-and-immersions/" %}

Tensor fields records the objects, main results, proof mechanisms, and boundary cases needed for the differential geometry note.

Definition

Definition. A tensor field is a smooth section of a tensor bundle built from tangent and cotangent bundles. It assigns multilinear data to each point smoothly.

Proposition

Proposition. Tensorial constructions are coordinate invariant: if a multilinear formula is defined intrinsically on tangent and cotangent vectors, its coordinate components transform by the tensor transformation law.

Proof

Proof. Change bases in each tangent and cotangent slot and use multilinearity. The transformation rule follows from how vectors and covectors change coordinates.

Remark

Remark. Metrics, differential forms, curvature, and stress-energy objects are tensor fields. Tensor language prevents coordinate artifacts from being mistaken for geometry.
