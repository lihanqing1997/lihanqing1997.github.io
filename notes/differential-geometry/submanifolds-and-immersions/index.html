Submanifolds and immersions records the objects, main results, proof mechanisms, and boundary cases needed for the differential geometry note.

Definition

Definition. An immersion has injective differential; a submersion has surjective differential. Embedded submanifolds are subsets locally cut out as coordinate planes.

Regular level set theorem

Theorem. The regular level set theorem says that if \(F:M\to N\) is a submersion at every point of \(F^{-1}(q)\), then \(F^{-1}(q)\) is an embedded submanifold of codimension \(\dim N\).

Proof

Proof. Use the inverse function theorem in adapted coordinates so \(F\) becomes projection onto the last coordinates. The level set is then locally a coordinate plane.

Remark

Remark. Submanifold theorems are the manifold form of solving equations under full-rank hypotheses.
