3. Vector fields and flows

{% include note-topic-nav.html back_label="Back" back_url="/notes/differential-geometry/" prev_label="Last page" prev_url="/notes/differential-geometry/tangent-spaces-and-differentials/" next_label="Next page" next_url="/notes/differential-geometry/tensor-fields/" %}

Vector fields and flows records the objects, main results, proof mechanisms, and boundary cases needed for the differential geometry note.

Definition

Definition. A vector field assigns a tangent vector \(X_p\in T_pM\) smoothly to each point. Its integral curves solve \(\dot\gamma(t)=X_{\gamma(t)}\).

Flow theorem

Theorem. Every smooth vector field has a unique local flow through each point, and the flow maps satisfy \(\Phi_{t+s}=\Phi_t\circ\Phi_s\) where both sides are defined.

Proof

Proof. In charts, the problem is an ODE with smooth right-hand side. Existence, uniqueness, and smooth dependence on initial conditions transfer back to the manifold.

Remark

Remark. Flows are how infinitesimal data become local transformations. Lie derivatives and group actions are built from this principle.
