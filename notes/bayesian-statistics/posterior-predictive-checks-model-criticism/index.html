9. Posterior predictive checks and model criticism

{% include note-topic-nav.html back_label="Back" back_url="/notes/bayesian-statistics/" prev_label="Last page" prev_url="/notes/bayesian-statistics/robust-models-missing-data-data-collection/" next_label="Next page" next_url="/notes/bayesian-statistics/model-comparison-cross-validation-bayes-factors/" %}

This page records the main definitions, propositions, and proof ideas for Posterior predictive checks and model criticism.

Definition

Definition. For replicated data \(Y^{\mathrm{rep}}\) and discrepancy \(T\), \[ p_{\mathrm{ppc}}=P(T(Y^{\mathrm{rep}},\theta)\ge T(Y,\theta)\mid Y). \]

Proposition

Proposition. Posterior predictive checks are calibrated to the fitted model, not to a uniform null distribution; their role is model criticism.

Proof

Proof. The data are used both to fit the posterior and to evaluate the discrepancy, so exact Neyman-Pearson calibration is not expected.

Remarks

Good discrepancies target tails, dependence, zero inflation, missingness, residual structure, or group variation.
