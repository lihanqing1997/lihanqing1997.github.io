Classical to sparse

This page is a map for answering literature questions in seminars. It does not replace reading the papers; it records what role each line of work plays in a defense of sparse Bayesian asymptotics.

Proposition. Classical parametric BvM proves posterior normality in fixed dimension. Semiparametric BvM studies finite-dimensional functionals in infinite-dimensional models. Sparse BvM studies a finite-dimensional parameter whose effective dimension is unknown and selected from a high-dimensional ambient space.

Remark. Castillo-Nickl type results are central for understanding when Bayesian credible sets have frequentist meaning in nonparametric and high-dimensional settings. Ghosal-Ghosh-van der Vaart type contraction theory supplies the testing-prior-mass machinery behind concentration. Spokoiny-style finite-sample likelihood theory supplies uniform quadratic expansions beyond classical fixed-dimensional asymptotics.

Comparison questions

Definition. A comparison with an existing BvM theorem should identify four objects: the model class, the prior class, the metric or functional receiving Gaussian approximation, and the regime of \(n,p,s\).

Proposition. For a sparse GLM BvM result, the natural comparison axes are: Gaussian linear model versus GLM likelihood, exact support recovery versus marginal functional normality, fixed support size versus growing sparsity, and asymptotic LAN versus nonasymptotic likelihood control.

Proof. Each axis changes one mathematical obstruction. GLM likelihoods require curvature control depending on the link. Support uncertainty changes mixture weights. Growing sparsity changes normal approximation dimension. Nonasymptotic control changes the statement from pointwise LAN to uniform high-probability quadratic approximation.

Questions that organize the literature

Definition. Sparse BvM papers differ by target: full vector posterior, low-dimensional linear functional, prediction, selected model posterior, or debiased one-step marginal.

Proposition. The strongest oracle BvM results need exact or near-exact model recovery; functional BvM results can hold under weaker conditions if the functional is sufficiently regular and debiased.

Remark. A useful reading order is: posterior contraction, selection consistency, LAN on sparse neighborhoods, BvM on the oracle model, then variants that relax selection by targeting functionals.
