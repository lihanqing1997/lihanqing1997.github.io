11. Monte Carlo integration and simulation error

{% include note-topic-nav.html back_label="Back" back_url="/notes/bayesian-statistics/" prev_label="Last page" prev_url="/notes/bayesian-statistics/model-comparison-cross-validation-bayes-factors/" next_label="Next page" next_url="/notes/bayesian-statistics/markov-chain-monte-carlo/" %}

This page records the main definitions, propositions, and proof ideas for Monte Carlo integration and simulation error.

Definition

Definition. For posterior draws \(\theta^{(1)},\ldots,\theta^{(S)}\), \[ \hat g_S=S^{-1}\sum_{s=1}^Sg(\theta^{(s)}) \] estimates \(E[g(\theta)\mid X]\).

Proposition

Proposition. For independent draws with posterior variance \(\sigma_g^2<\infty\), \[ \sqrt S(\hat g_S-E[g(\theta)\mid X])\rightsquigarrow N(0,\sigma_g^2). \]

Proof

Proof. Condition on the data; the simulated values are i.i.d., so the ordinary CLT applies.

Remarks

More simulation reduces Monte Carlo error, not posterior uncertainty.
