28. Doob, Schwartz, and posterior consistency

{% include note-topic-nav.html back_label="Back" back_url="/notes/bayesian-statistics/" prev_label="Last page" prev_url="/notes/bayesian-statistics/mixture-priors-density-estimation/" next_label="Next page" next_url="/notes/bayesian-statistics/testing-entropy-general-contraction-theorems/" %}

This page records the main definitions, propositions, and proof ideas for Doob, Schwartz, and posterior consistency.

Definition

Definition. A prior has the KL property at \(P_0\) if every set \(\{P:P_0\log(p_0/p)<\varepsilon\}\) has positive prior mass.

Proposition

Proposition. Schwartz theorem: KL support plus uniform tests for the complement of a neighborhood imply posterior consistency.

Proof

Proof. KL support lower-bounds the denominator; tests upper-bound the numerator on the bad set.

Remarks

Doob consistency is prior-almost-sure; Schwartz consistency is useful for fixed frequentist truths.
