8. Robust models, missing data, and data collection

{% include note-topic-nav.html back_label="Back" back_url="/notes/bayesian-statistics/" prev_label="Last page" prev_url="/notes/bayesian-statistics/bayesian-regression-and-glms/" next_label="Next page" next_url="/notes/bayesian-statistics/posterior-predictive-checks-model-criticism/" %}

This page records the main definitions, propositions, and proof ideas for Robust models, missing data, and data collection.

Definition

Definition. For ignorable missingness, the observed-data likelihood is \[ L(\theta;y_{\mathrm{obs}})=\int p_\theta(y_{\mathrm{obs}},y_{\mathrm{mis}})\,\mathrm{d}y_{\mathrm{mis}}. \]

Proposition

Proposition. Student-\(t\) errors are robust because their score is bounded in extreme residuals, unlike the Gaussian score.

Proof

Proof. The \(t\)-score is proportional to \(r/(\nu+r^2)\), which stays bounded as \(|r|\to\infty\).

Remarks

Censoring, truncation, surveys, and selection mechanisms change the experiment whose posterior is computed.
