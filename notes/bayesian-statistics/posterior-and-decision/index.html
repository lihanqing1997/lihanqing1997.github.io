3. Bayes actions, loss, and credible decisions

{% include note-topic-nav.html back_label="Back" back_url="/notes/bayesian-statistics/" prev_label="Last page" prev_url="/notes/bayesian-statistics/posterior-prediction-marginal-likelihood/" next_label="Next page" next_url="/notes/bayesian-statistics/conjugacy-hierarchical-models-and-computation/" %}

This page records the main definitions, propositions, and proof ideas for Bayes actions, loss, and credible decisions.

Definition

Definition. For loss \(L(\theta,a)\), a Bayes action is \[ a^\star(X)\in\operatorname*{argmin}_{a\in\mathcal A}\int L(\theta,a)\,\mathrm{d}\Pi(\theta\mid X). \]

Proposition

Proposition. Under squared error loss, the posterior mean is a Bayes action. Under absolute loss on \(\mathbb R\), every posterior median is a Bayes action.

Proof

Proof. Expand posterior expected squared loss around the posterior mean. For absolute loss, the subgradient condition is exactly the median condition.

Remarks

A credible set is a decision when the action is a set and the loss penalizes missing the truth and reporting excessive volume.
