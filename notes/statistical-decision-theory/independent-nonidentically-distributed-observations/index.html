26. Independent non-identically distributed observations

{% include note-topic-nav.html back_label="Back" back_url="/notes/statistical-decision-theory/" prev_label="Last page" prev_url="/notes/statistical-decision-theory/approximation-by-exponential-families/" next_label="Next page" next_url="/notes/statistical-decision-theory/lower-bounds/" %}

Product experiments are the main application field for Le Cam's general theory. The log likelihood is a sum, but the summands need not be identically distributed.

Product experiment

Definition. A product experiment has \(P_{\theta,n}=\bigotimes_{j=1}^{m_n}P_{\theta,j,n}\). The log likelihood ratio decomposes as

\[ \log\frac{\mathrm{d}P_{\theta,n}}{\mathrm{d}P_{\theta_0,n}} =\sum_{j=1}^{m_n}\log\frac{\mathrm{d}P_{\theta,j,n}}{\mathrm{d}P_{\theta_0,j,n}}. \]

Infinitesimal component condition

Definition. The array is infinitesimal if each single component contributes negligibly to Hellinger distance for fixed parameter pairs. This prevents one observation from dominating the limit.

Central limit mechanism

Proof. Once log likelihoods are sums of negligible independent terms, triangular-array CLTs and Hellinger control give Gaussian or infinitely divisible likelihood-ratio limits.

Non-i.i.d. local asymptotics

The same LAN and minimax conclusions hold when information accumulates unevenly across observations, provided the product experiment has the right likelihood-ratio limit.

26. Observations indépendantes non identiquement distribuées

{% include note-topic-nav.html back_label="Retour" back_url="/notes/statistical-decision-theory/" prev_label="Page précédente" prev_url="/notes/statistical-decision-theory/approximation-by-exponential-families/" next_label="Page suivante" next_url="/notes/statistical-decision-theory/lower-bounds/" %}

Les expériences produits sont le principal domaine d'application de la théorie générale de Le Cam.

Expérience produit

Définition. Une expérience produit vérifie \(P_{\theta,n}=\bigotimes_{j=1}^{m_n}P_{\theta,j,n}\), donc la log-vraisemblance est une somme.

Condition infinitésimale des composantes

Définition. Chaque composante doit contribuer de façon négligeable à la distance de Hellinger pour éviter qu'une seule observation domine la limite.

Mécanisme central limite

Preuve. Les log-vraisemblances étant des sommes indépendantes négligeables, les TCL de tableaux triangulaires donnent les limites.

Asymptotique locale non i.i.d.

Les conclusions LAN et minimax restent valables lorsque l'information s'accumule de manière non uniforme.

26. 独立但非同分布观测

{% include note-topic-nav.html back_label="返回" back_url="/notes/statistical-decision-theory/" prev_label="上一页" prev_url="/notes/statistical-decision-theory/approximation-by-exponential-families/" next_label="下一页" next_url="/notes/statistical-decision-theory/lower-bounds/" %}

乘积实验是 Le Cam 一般理论的主要应用场景。log-likelihood 是和，但各项不必同分布。

乘积实验

定义. 乘积实验满足 \(P_{\theta,n}=\bigotimes_{j=1}^{m_n}P_{\theta,j,n}\)，因此 log-likelihood ratio 分解为各分量之和。

分量无穷小条件

定义. 若每个单独分量对固定参数对的 Hellinger 距离贡献都可忽略，则称阵列无穷小。它防止某个观测支配极限。

中心极限定理机制

证明. log-likelihood 是独立无穷小项之和后，三角阵列 CLT 与 Hellinger 控制给出 Gaussian 或无限可分似然比极限。

非 i.i.d. 局部渐近

只要乘积实验有正确的似然比极限，即使信息在观测之间不均匀累积，LAN 与 minimax 结论仍成立。
