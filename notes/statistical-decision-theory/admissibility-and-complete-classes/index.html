Admissibility and Complete Classes

{% include note-topic-nav.html back_label="Back" back_url="/notes/statistical-decision-theory/" prev_label="Last page" prev_url="/notes/statistical-decision-theory/risk-bayes-and-minimax/" next_label="Next page" next_url="/notes/statistical-decision-theory/shrinkage-and-invariance/" %}

Domination

Definition. \(\delta_1\) dominates \(\delta_0\) if \(R(\theta,\delta_1)\le R(\theta,\delta_0)\) for all \(\theta\), and \(R(\theta,\delta_1)

A rule \(\delta_1\) dominates \(\delta_0\) if \(R(\theta,\delta_1)\le R(\theta,\delta_0)\) for every \(\theta\), with strict inequality for at least one \(\theta\). An inadmissible rule is therefore never better and sometimes worse.

Admissibility

A rule is admissible if no rule dominates it. Admissibility is weak: many rules may be admissible, and an admissible rule can still have large maximum risk. But inadmissibility is decisive evidence against a procedure.

Risk set geometry

For a finite parameter space \(\Theta=\{\theta_1,\ldots,\theta_k\}\), each rule has a risk vector \((R(\theta_1,\delta),\ldots,R(\theta_k,\delta))\in\mathbb R^k\). Domination is coordinatewise comparison. Bayes risk is a linear functional of this vector, while minimax risk is its largest coordinate. This geometric picture explains why supporting hyperplanes and least favorable priors enter complete class arguments.

Bayes rules

Proposition. If \(\delta_\Pi\) is a unique Bayes rule for a prior \(\Pi\) with full support, then \(\delta_\Pi\) is admissible.

Proof. If another rule \(\delta\) dominated \(\delta_\Pi\), then \(\int R(\theta,\delta)d\Pi(\theta)\le\int R(\theta,\delta_\Pi)d\Pi(\theta)\). Full support and strict improvement at some parameter, together with continuity of the risk, make the inequality strict, contradicting Bayes optimality.

Bayes rules under priors with full support are often admissible under regularity. The intuition is that if a rule uniformly improves a Bayes rule, it also improves its integrated Bayes risk, contradicting Bayes optimality.

Proper and generalized Bayes

A proper Bayes rule minimizes Bayes risk for a probability prior. A generalized Bayes rule may arise from an improper prior but still minimizes posterior expected loss formally. Many classical estimators are generalized Bayes. Admissibility of generalized Bayes rules is more delicate and usually requires recurrence or limiting-prior arguments.

Complete classes

A class \(\mathcal C\) of rules is complete if every rule outside \(\mathcal C\) is dominated by a rule inside \(\mathcal C\). Complete class theorems often show that admissible rules are Bayes or limits of Bayes rules. This makes Bayes rules structurally central even for frequentist decision theory.

Essential lesson

Admissibility removes procedures that are uniformly improvable, while complete class theorems describe where non-dominated procedures live. The theory does not say which admissible rule is best; it narrows the search to rules with decision-theoretic support.

Admissibilité et Classes Complètes

{% include note-topic-nav.html back_label="Retour" back_url="/notes/statistical-decision-theory/" prev_label="Page précédente" prev_url="/notes/statistical-decision-theory/risk-bayes-and-minimax/" next_label="Page suivante" next_url="/notes/statistical-decision-theory/shrinkage-and-invariance/" %}

Domination

Définition. \(\delta_1\) domine \(\delta_0\) si \(R(\theta,\delta_1)\le R(\theta,\delta_0)\) pour tout \(\theta\), avec inégalité stricte pour au moins un \(\theta\). Une règle est admissible si elle n'est pas dominée.

Une règle \(\delta_1\) domine \(\delta_0\) si \(R(\theta,\delta_1)\le R(\theta,\delta_0)\) pour tout \(\theta\), avec une inégalité stricte pour au moins un \(\theta\).

Admissibilité

Une règle est admissible si aucune autre règle ne la domine. C'est une condition faible, mais l'inadmissibilité est une critique forte : la règle n'est jamais meilleure et parfois pire.

Géométrie du risque

Si \(\Theta\) est fini, chaque règle correspond au vecteur de ses risques. La domination est une comparaison coordonnée par coordonnée, le risque bayésien est une fonction linéaire de ce vecteur, et le risque minimax est sa plus grande coordonnée. Cette géométrie explique le rôle des a priori les moins favorables.

Règles bayésiennes

Proposition. Une règle bayésienne unique pour un a priori de support complet est admissible, sous continuité des risques.

Preuve. Une règle qui la dominerait diminuerait aussi le risque intégré sous l'a priori, et strictement grâce au support complet, ce qui contredit l'optimalité bayésienne.

Les règles bayésiennes associées à des a priori de support complet sont souvent admissibles. Si une règle les améliorait uniformément, elle améliorerait aussi leur risque bayésien intégré.

Bayes propre et généralisé

Une règle bayésienne propre minimise le risque bayésien pour un a priori de probabilité. Une règle bayésienne généralisée peut provenir d'un a priori impropre. Son admissibilité demande des arguments supplémentaires, souvent par limites d'a priori propres.

Classes complètes

Une classe \(\mathcal C\) est complète si toute règle extérieure est dominée par une règle de \(\mathcal C\). Les théorèmes de classe complète montrent souvent que les règles admissibles sont bayésiennes ou limites de règles bayésiennes.

Message principal

L'admissibilité élimine les règles uniformément améliorables. Les théorèmes de classe complète disent ensuite où chercher les règles qui restent raisonnables.

可容许性与 Complete Class

{% include note-topic-nav.html back_label="返回" back_url="/notes/statistical-decision-theory/" prev_label="上一页" prev_url="/notes/statistical-decision-theory/risk-bayes-and-minimax/" next_label="下一页" next_url="/notes/statistical-decision-theory/shrinkage-and-invariance/" %}

支配

定义. 若对所有 \(\theta\)，\(R(\theta,\delta_1)\le R(\theta,\delta_0)\)，并且至少某个 \(\theta\) 上严格小于，则称 \(\delta_1\) 支配 \(\delta_0\)。若一个规则不被任何其他规则支配，则称其可容许。

若对所有 \(\theta\)，\(R(\theta,\delta_1)\le R(\theta,\delta_0)\)，并且至少在一个 \(\theta\) 上严格小于，则称 \(\delta_1\) 支配 \(\delta_0\)。不可容许规则从不更好，并且有时更差。

可容许性

若没有规则支配 \(\delta\)，则称 \(\delta\) 可容许。可容许性很弱：很多规则都可能可容许，而且可容许规则的最大风险仍可能很大。但不可容许性是强烈的反对理由。

风险集合几何

若参数空间有限，\(\Theta=\{\theta_1,\ldots,\theta_k\}\)，每个规则对应风险向量 \((R(\theta_1,\delta),\ldots,R(\theta_k,\delta))\)。支配是逐坐标比较，Bayes 风险是该向量的线性泛函，minimax 风险是最大坐标。这一几何图像解释了为什么支撑超平面和 least favorable prior 会出现在 complete class 理论中。

Bayes 规则

命题. 若 \(\delta_\Pi\) 是某个全支撑先验 \(\Pi\) 的唯一 Bayes 规则，并且风险函数具有适当连续性，则 \(\delta_\Pi\) 可容许。

证明. 若存在规则支配 \(\delta_\Pi\)，则其 Bayes 风险不大于 \(\delta_\Pi\) 的 Bayes 风险。由于先验全支撑且某处严格改进，积分风险会严格变小，矛盾于 \(\delta_\Pi\) 的 Bayes 最优性。

在适当条件下，全支撑先验对应的 Bayes 规则常常可容许。直觉是：若某规则一致改进 Bayes 规则，它也会降低 Bayes 风险，从而矛盾。

Proper Bayes 与 generalized Bayes

proper Bayes 规则来自概率先验并最小化 Bayes 风险。generalized Bayes 规则可能来自 improper prior，但形式上仍最小化后验期望损失。许多经典估计量是 generalized Bayes；它们的可容许性更微妙，通常需要递归性或极限先验论证。

Complete class

若类 \(\mathcal C\) 外的每个规则都被 \(\mathcal C\) 中某个规则支配，则称 \(\mathcal C\) 是 complete class。Complete class 定理常说明可容许规则是 Bayes 规则或 Bayes 规则的极限。

核心信息

可容许性剔除一致可改进的程序；complete class 定理进一步描述不可被支配的程序大致在哪里。它不直接告诉我们唯一最优规则，但能把搜索范围缩小到有决策论支持的规则。
