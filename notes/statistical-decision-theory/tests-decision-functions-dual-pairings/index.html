A statistical test is not only a rule; it is a bounded measurable decision function paired linearly with probability laws.

Randomized test

Definition. For an experiment \((\mathcal X,\mathcal A,\{P_\theta\})\), a randomized test is a measurable function \(\varphi:\mathcal X\to[0,1]\). The value \(\varphi(x)\) is the conditional probability of rejection after observing \(x\).

Dual pairing

Definition. In a dominated experiment with densities \(p_\theta\in L^1(\mu)\), tests lie in \(L^\infty(\mu)\) and power is the pairing \(E_\theta\varphi=\langle\varphi,p_\theta\rangle=\int\varphi p_\theta\,\mathrm{d}\mu\).

Convexity of tests

Proposition. The set of randomized tests is convex. Size constraints and power objectives are linear in \(\varphi\). Thus many testing problems are infinite-dimensional linear programs.

Neyman-Pearson as separation

Theorem. For simple \(P_0\) versus \(P_1\), the most powerful level-\(\alpha\) test is obtained by a supporting hyperplane argument in the dual pairing: reject where the likelihood ratio \(\mathrm{d}P_1/\mathrm{d}P_0\) is large, with boundary randomization if needed.

Total variation

Corollary. The total variation distance satisfies \(\|P-Q\|_{\mathrm{TV}}=\sup_{0\le\varphi\le1}|P\varphi-Q\varphi|\). Thus distinguishability of experiments is a supremum over bounded tests.

Markov adjoints

Remark. A Markov kernel pushes probability laws forward and pulls tests backward by conditional expectation. Blackwell comparison is therefore naturally expressed by adjoint action on decision functions.
