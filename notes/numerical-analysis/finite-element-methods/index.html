27. Finite element methods

{% include note-topic-nav.html back_label="Back" back_url="/notes/numerical-analysis/" prev_label="Last page" prev_url="/notes/numerical-analysis/finite-difference-methods-pdes/" next_label="Next page" next_url="/notes/numerical-analysis/spectral-methods/" %}

Finite element methods approximate weak formulations by piecewise polynomial spaces on meshes. They are natural for geometry, boundary conditions, and variational problems.

Weak formulation

Definition. A weak formulation seeks \(u\in V\) such that \(a(u,v)=\ell(v)\) for every test function \(v\in V\). It replaces pointwise derivatives by integral identities and is often the right formulation for nonsmooth solutions.

Galerkin method

Definition. Given a finite-dimensional subspace \(V_h\subset V\), the Galerkin approximation \(u_h\in V_h\) satisfies \(a(u_h,v_h)=\ell(v_h)\) for every \(v_h\in V_h\).

Céa lemma

Theorem. If \(a\) is continuous and coercive, then \(\|u-u_h\|_V\le (M/\alpha)\inf_{v_h\in V_h}\|u-v_h\|_V\). Thus finite element error is controlled by best approximation in the chosen space.

Element basis functions

Proposition. Local basis functions lead to sparse stiffness matrices because most basis functions have disjoint support. Mesh geometry influences both approximation error and conditioning of the resulting linear system.

A posteriori error

Remark. Residual-based a posteriori estimates guide adaptive mesh refinement. They measure where the computed solution fails to satisfy the equation or interface conditions strongly enough.
