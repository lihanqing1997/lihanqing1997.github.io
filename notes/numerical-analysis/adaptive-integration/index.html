Adaptive integration spends computational effort where the integrand is difficult, rather than using a uniformly fine grid everywhere.

Local error estimate

Definition. An adaptive quadrature method estimates the local error on an interval by comparing two rules, often a coarse rule and a refined or embedded higher-order rule. Intervals with large estimated error are subdivided.

Adaptive Simpson principle

Proposition. If \(S(a,b)\) denotes Simpson's rule and \(S(a,m)+S(m,b)\) its split version, then their difference estimates the leading error term. Recursion continues until the estimated contribution meets the tolerance.

Reliability caveat

Remark. A posteriori estimates can miss narrow spikes, discontinuities, or cancellation patterns if the sampling never sees them. Adaptive methods are powerful but still depend on assumptions about the integrand.

Termination and tolerance

Proposition. Practical adaptive routines combine absolute and relative tolerances, recursion limits, and safeguards against roundoff stagnation. A returned tolerance is an algorithmic diagnostic, not a theorem unless the required regularity assumptions hold.

Higher-dimensional boundary

Remark. Adaptive deterministic integration scales poorly with dimension. High-dimensional integration often shifts toward sparse grids, quasi-Monte Carlo, or probabilistic methods, which belong partly to numerical analysis and partly to statistics.
