25. Boundary value problems

{% include note-topic-nav.html back_label="Back" back_url="/notes/numerical-analysis/" prev_label="Last page" prev_url="/notes/numerical-analysis/multistep-methods/" next_label="Next page" next_url="/notes/numerical-analysis/finite-difference-methods-pdes/" %}

Boundary value problems constrain a solution at more than one point or along a boundary, making them structurally different from initial value problems.

Boundary value problem

Definition. A boundary value problem seeks a function satisfying a differential equation on a domain together with boundary conditions. For example, \(-u''(x)=f(x)\) on \([0,1]\), with \(u(0)=u(1)=0\).

Shooting method

Definition. Shooting converts a boundary value problem into an initial value problem with unknown initial slope or parameters. The unknowns are adjusted so the terminal boundary condition is met.

Finite difference discretization

Proposition. Replacing derivatives by finite differences turns a linear boundary value problem into a linear algebra problem. For second-order elliptic problems the resulting matrix often has sparse, structured, positive definite form.

Collocation

Definition. Collocation chooses an approximate function in a finite-dimensional space and enforces the differential equation at selected points. It connects interpolation, residual minimization, and finite element ideas.

Well-posedness first

Remark. A discretization can only be expected to converge to the correct object when the continuous boundary value problem is well posed. Boundary layers, nonunique solutions, and incompatible boundary data require special care.
