ODE solvers approximate trajectories of differential equations while controlling local truncation error, global error, and stability.

Initial value problem

Definition. An initial value problem has the form \(\mathrm{d}y/\mathrm{d}t=f(t,y)\), \(y(t_0)=y_0\). A one-step numerical method constructs approximations \(y_n\approx y(t_n)\) on a grid \(t_n\).

Euler methods

Definition. Forward Euler uses \(y_{n+1}=y_n+h f(t_n,y_n)\). Backward Euler uses \(y_{n+1}=y_n+h f(t_{n+1},y_{n+1})\). The first is explicit; the second requires solving an equation but is much more stable for stiff problems.

Local and global error

Proposition. The local truncation error measures the error made in one step starting from the exact solution. The global error accumulates these local errors through the stability behavior of the method.

Runge-Kutta methods

Theorem. Runge-Kutta methods approximate the integral form \(y(t+h)=y(t)+\int_t^{t+h} f(s,y(s))\,\mathrm{d}s\) by several staged slope evaluations. Order conditions determine which Taylor terms are matched.

Adaptive step size

Remark. Embedded Runge-Kutta pairs estimate local error by comparing two orders. Step-size adaptation is essential because trajectory difficulty is usually nonuniform in time.
