Arithmetic functions and convolution records the objects, main results, proof mechanisms, and boundary cases needed for the number theory note.

Definition

Definition. An arithmetic function is a function on positive integers. Dirichlet convolution is \((f*g)(n)=\sum_{d\mid n}f(d)g(n/d)\).

Proposition

Proposition. Multiplicative functions are closed under Dirichlet convolution, and a function with \(f(1)\ne0\) has a convolution inverse.

Proof

Proof. Factor divisors prime-by-prime for multiplicativity. The inverse is constructed recursively from the equation \(f*g=\varepsilon\), where \(\varepsilon(1)=1\) and \(\varepsilon(n)=0\) for \(n>1\).

Remark

Remark. Convolution is algebra for divisors. It organizes Möbius inversion, divisor sums, and Euler products.
