5. Quadratic residues and reciprocity

{% include note-topic-nav.html back_label="Back" back_url="/notes/number-theory/" prev_label="Last page" prev_url="/notes/number-theory/chinese-remainder-theorem/" next_label="Next page" next_url="/notes/number-theory/arithmetic-functions-and-convolution/" %}

Quadratic residues and reciprocity records the objects, main results, proof mechanisms, and boundary cases needed for the number theory note.

Definition

Definition. A nonzero class \(a\) modulo an odd prime \(p\) is a quadratic residue if \(x^2\equiv a\pmod p\) has a solution. The Legendre symbol records this sign.

Quadratic reciprocity

Theorem. Quadratic reciprocity relates \((p/q)\) and \((q/p)\) for odd primes: \((p/q)(q/p)=(-1)^{((p-1)/2)((q-1)/2)}\).

Proof

Proof. Proofs count lattice points, compare Gauss sums, or use finite Fourier analysis. Each proof converts solvability of quadratic congruences into a symmetry count.

Remark

Remark. Quadratic reciprocity is the first deep reciprocity law. It suggests that congruence solvability is governed by hidden symmetry.
