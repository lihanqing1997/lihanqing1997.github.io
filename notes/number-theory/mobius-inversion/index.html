Mobius inversion records the objects, main results, proof mechanisms, and boundary cases needed for the number theory note.

Definition

Definition. The Möbius function \(\mu\) is the Dirichlet inverse of the constant function \(1\). It detects squarefree parity of prime factors.

Proposition

Proposition. If \(F(n)=\sum_{d\mid n}f(d)\), then \(f(n)=\sum_{d\mid n}\mu(d)F(n/d)\).

Proof

Proof. Write \(F=1*f\) under Dirichlet convolution. Convolve both sides with \(\mu\), using \(\mu*1=\varepsilon\).

Remark

Remark. Möbius inversion is inclusion-exclusion on the divisor lattice. It is a recurring method in elementary and analytic number theory.
