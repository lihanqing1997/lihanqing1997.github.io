Number fields and algebraic integers records the objects, main results, proof mechanisms, and boundary cases needed for the number theory note.

Definition

Definition. A number field is a finite extension of \(\mathbb Q\). An algebraic integer is a root of a monic polynomial with integer coefficients.

Proposition

Proposition. The algebraic integers in a number field form a ring, the ring of integers \(\mathcal O_K\), which is a free abelian group of rank \([K:\mathbb Q]\).

Proof

Proof. Closure under addition and multiplication follows from finite generation of \(\mathbb Z\)-algebras generated by algebraic integers. The rank statement uses embeddings and trace/discriminant arguments.

Remark

Remark. Rings of integers are the correct replacement for \(\mathbb Z\) inside number fields.
