Class group and failure of unique factorization records the objects, main results, proof mechanisms, and boundary cases needed for the number theory note.

Definition

Definition. The ideal class group is the quotient of fractional ideals by principal fractional ideals. It measures the failure of every ideal to be principal.

Proposition

Proposition. A Dedekind domain has unique factorization of elements exactly when its ideal class group is trivial. For number fields, the class group is finite.

Proof

Proof. If every ideal is principal, ideal factorization descends to element factorization. Conversely, a UFD that is Dedekind has all height-one prime ideals principal. Finiteness uses geometry of lattices.

Remark

Remark. The class group is an invariant of arithmetic complexity. It is often the first global obstruction in a number field.
