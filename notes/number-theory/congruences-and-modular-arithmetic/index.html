Congruences and modular arithmetic records the objects, main results, proof mechanisms, and boundary cases needed for the number theory note.

Definition

Definition. Integers \(a,b\) are congruent modulo \(n\) if \(n\mid a-b\). Congruence classes form the quotient ring \(\mathbb Z/n\mathbb Z\).

Proposition

Proposition. An element \(a\) is invertible modulo \(n\) exactly when \(\gcd(a,n)=1\). The invertible classes form the group \((\mathbb Z/n\mathbb Z)^\times\).

Proof

Proof. Bezout gives \(ua+vn=1\) when the gcd is one, so \(u\) is the inverse of \(a\) modulo \(n\). Conversely, a modular inverse gives a Bezout relation.

Remark

Remark. Modular arithmetic turns divisibility into algebra on finite rings and groups. It is the basic language of congruence constraints.
