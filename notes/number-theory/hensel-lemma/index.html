Hensel lemma records the objects, main results, proof mechanisms, and boundary cases needed for the number theory note.

Definition

Definition. Hensel lemma lifts approximate roots modulo powers of a prime to exact roots in \(\mathbb Z_p\).

Hensel lemma

Theorem. If \(f(a)\equiv0\pmod p\) and \(f'(a)\not\equiv0\pmod p\), then there is a unique \(p\)-adic root \(\alpha\equiv a\pmod p\).

Proof

Proof. Construct corrections recursively modulo \(p^n\). The nonzero derivative is invertible modulo \(p\), so each correction is uniquely solvable. Completeness gives the limit.

Remark

Remark. Hensel lemma is Newton's method in nonarchimedean geometry. It is the basic local lifting tool.
