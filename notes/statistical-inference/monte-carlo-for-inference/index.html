22. Monte Carlo for inference

{% include note-topic-nav.html back_label="Back" back_url="/notes/statistical-inference/" prev_label="Last page" prev_url="/notes/statistical-inference/higher-order-asymptotics/" next_label="Next page" next_url="/notes/statistical-inference/missing-data-em/" %}

Monte Carlo methods approximate sampling distributions, power, coverage, and numerical expectations. They are inferential tools only when their own simulation error is measured.

Definition. If \(Y_1,\ldots,Y_B\) are simulation outputs estimating \(m=EY\), the Monte Carlo estimator is \(\bar Y_B=B^{-1}\sum_bY_b\), with Monte Carlo standard error \[ \widehat{\operatorname{mcse}}(\bar Y_B)=s_Y/\sqrt B. \]

Simulation CLT. If \(Y_b\) are i.i.d. with variance \(\tau^2\lt\infty\), then \[ \sqrt B(\bar Y_B-m)\rightsquigarrow N(0,\tau^2). \]

Proof. This is the ordinary CLT applied to the simulation draws. The point is that \(B\), not the data sample size \(n\), controls Monte Carlo error.

Power and coverage studies

For a rejection indicator \(Y_b\), \(\bar Y_B\) estimates power or size and has standard error \(\sqrt{\hat p(1-\hat p)/B}\). Simulation reports should include this error, otherwise numerical noise can be mistaken for statistical behavior.

22. Monte Carlo pour l’inférence

{% include note-topic-nav.html back_label="Retour" back_url="/notes/statistical-inference/" prev_label="Page précédente" prev_url="/notes/statistical-inference/higher-order-asymptotics/" next_label="Page suivante" next_url="/notes/statistical-inference/missing-data-em/" %}

Les méthodes de Monte Carlo approximent les lois d’échantillonnage, puissances, couvertures et espérances numériques. Elles ne sont fiables que si l’erreur de simulation est mesurée.

Définition. Si \(Y_1,\ldots,Y_B\) sont des sorties de simulation estimant \(m=EY\), alors \(\bar Y_B=B^{-1}\sum_bY_b\), avec erreur standard Monte Carlo \[ \widehat{\operatorname{mcse}}(\bar Y_B)=s_Y/\sqrt B. \]

TCL de simulation. Si \(Y_b\) sont i.i.d. de variance \(\tau^2\lt\infty\), alors \[ \sqrt B(\bar Y_B-m)\rightsquigarrow N(0,\tau^2). \]

Preuve. C’est le TCL ordinaire appliqué aux tirages de simulation. C’est \(B\), et non la taille d’échantillon \(n\), qui contrôle l’erreur Monte Carlo.

Puissance et couverture

Pour un indicateur de rejet, \(\bar Y_B\) estime la puissance ou la taille, avec erreur standard \(\sqrt{\hat p(1-\hat p)/B}\).

22. 推断中的蒙特卡罗

{% include note-topic-nav.html back_label="返回" back_url="/notes/statistical-inference/" prev_label="上一页" prev_url="/notes/statistical-inference/higher-order-asymptotics/" next_label="下一页" next_url="/notes/statistical-inference/missing-data-em/" %}

蒙特卡罗方法用于近似抽样分布、power、coverage 和数值期望。只有同时量化模拟误差时，它才是可靠的推断工具。

定义. 若 \(Y_1,\ldots,Y_B\) 是用于估计 \(m=EY\) 的模拟输出，则蒙特卡罗估计量为 \(\bar Y_B=B^{-1}\sum_bY_b\)，其蒙特卡罗 standard error 为 \[ \widehat{\operatorname{mcse}}(\bar Y_B)=s_Y/\sqrt B. \]

模拟 CLT. 若 \(Y_b\) 独立同分布且方差 \(\tau^2\lt\infty\)，则 \[ \sqrt B(\bar Y_B-m)\rightsquigarrow N(0,\tau^2). \]

证明. 这是普通 CLT 作用于模拟样本。关键是蒙特卡罗误差由 \(B\) 控制，而不是由数据样本量 \(n\) 控制。

Power 与 coverage 研究

若 \(Y_b\) 是拒绝指示变量，则 \(\bar Y_B\) 估计 power 或 size，标准误为 \(\sqrt{\hat p(1-\hat p)/B}\)。模拟报告应包含这个误差，否则数值噪声会被误认为统计行为。
