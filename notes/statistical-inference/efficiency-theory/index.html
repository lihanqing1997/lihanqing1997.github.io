26. Efficiency theory

{% include note-topic-nav.html back_label="Back" back_url="/notes/statistical-inference/" prev_label="Last page" prev_url="/notes/statistical-inference/local-asymptotic-normality-contiguity/" next_label="Next page" next_url="/notes/statistical-inference/semiparametric-inference-interface/" %}

Efficiency theory asks whether an estimator uses all first-order information available in the experiment. In regular parametric models, Fisher information gives the lower bound.

Cramér-Rao bound. If \(T\) is unbiased for scalar \(g(\theta)\) in a regular model, then \[ \operatorname{Var}_\theta(T)\ge \frac{\{g'(\theta)\}^2}{I(\theta)}. \]

Proof. Differentiating \(E_\theta T=g(\theta)\) gives \[ g'(\theta)=E_\theta\{T s_\theta(X)\}=E_\theta\{(T-g(\theta))s_\theta(X)\}. \] Cauchy-Schwarz gives \(\{g'(\theta)\}^2\le\operatorname{Var}_\theta(T)I(\theta)\).

Asymptotic efficiency

An estimator sequence is asymptotically efficient for \(\theta\) if \(\sqrt n(\hat\theta-\theta)\) has limiting variance \(I(\theta)^{-1}\). LAN and Le Cam convolution theorems generalize this: regular estimators have a Gaussian information-bound component plus possible extra noise.

26. Théorie de l’efficacité

{% include note-topic-nav.html back_label="Retour" back_url="/notes/statistical-inference/" prev_label="Page précédente" prev_url="/notes/statistical-inference/local-asymptotic-normality-contiguity/" next_label="Page suivante" next_url="/notes/statistical-inference/semiparametric-inference-interface/" %}

La théorie de l’efficacité demande si un estimateur utilise toute l’information de premier ordre disponible dans l’expérience. Dans les modèles paramétriques réguliers, l’information de Fisher donne la borne.

Borne de Cramér-Rao. Si \(T\) est sans biais pour \(g(\theta)\) dans un modèle régulier scalaire, alors \[ \operatorname{Var}_\theta(T)\ge \frac{\{g'(\theta)\}^2}{I(\theta)}. \]

Preuve. En dérivant \(E_\theta T=g(\theta)\), \[ g'(\theta)=E_\theta\{T s_\theta(X)\}=E_\theta\{(T-g(\theta))s_\theta(X)\}. \] L’inégalité de Cauchy-Schwarz donne la borne.

Efficacité asymptotique

Un estimateur est asymptotiquement efficace si la variance limite de \(\sqrt n(\hat\theta-\theta)\) est \(I(\theta)^{-1}\). Les théorèmes de convolution de Le Cam généralisent cette borne.

26. 效率理论

{% include note-topic-nav.html back_label="返回" back_url="/notes/statistical-inference/" prev_label="上一页" prev_url="/notes/statistical-inference/local-asymptotic-normality-contiguity/" next_label="下一页" next_url="/notes/statistical-inference/semiparametric-inference-interface/" %}

效率理论研究估计量是否使用了实验中所有一阶信息。在正则参数模型中，费雪信息给出下界。

克拉美-拉奥下界. 若 \(T\) 在正则标量模型中无偏估计 \(g(\theta)\)，则 \[ \operatorname{Var}_\theta(T)\ge \frac{\{g'(\theta)\}^2}{I(\theta)}. \]

证明. 对 \(E_\theta T=g(\theta)\) 求导得 \[ g'(\theta)=E_\theta\{T s_\theta(X)\}=E_\theta\{(T-g(\theta))s_\theta(X)\}. \] 柯西-施瓦茨不等式给出 \(\{g'(\theta)\}^2\le\operatorname{Var}_\theta(T)I(\theta)\)。

渐近效率

若 \(\sqrt n(\hat\theta-\theta)\) 的极限方差达到 \(I(\theta)^{-1}\)，则估计量渐近有效。LAN 与勒卡姆 convolution 定理将这一思想推广到更一般实验。
