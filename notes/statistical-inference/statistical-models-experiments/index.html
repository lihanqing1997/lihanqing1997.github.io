Statistical models and experiments

{% include note-topic-nav.html back_label="Back" back_url="/notes/statistical-inference/" next_label="Next page" next_url="/notes/statistical-inference/likelihood-score-information/" %}

A statistical model is a mathematical description of how data could have been generated. In this note, probability is used as the language for data-generating mechanisms; the inferential question is how the unknown element of the model can be learned from the observed sample.

Definition. A statistical experiment is a measurable sample space \((\mathcal X,\mathcal A)\) together with a family of probability measures \(\mathcal P=\{P_\theta:\theta\in\Theta\}\). A statistic is a measurable map \(T:\mathcal X\to\mathcal T\). The model is identifiable if \(P_\theta=P_{\theta'}\) implies \(\theta=\theta'\).

Dominated models

The model is dominated if there exists a \(\sigma\)-finite measure \(\mu\) such that \(P_\theta\ll\mu\) for every \(\theta\). Then \(p_\theta=\mathrm{d}P_\theta/\mathrm{d}\mu\) is a density and likelihood methods can be written using \(p_\theta(x)\). For independent observations, the product experiment has density \(\prod_{i=1}^np_\theta(x_i)\).

Proposition. If \(\theta_0\) is the true value in a dominated identifiable model, then the expected log-likelihood contrast satisfies \[ E_{\theta_0}\log p_{\theta_0}(X)-E_{\theta_0}\log p_\theta(X) =\operatorname{KL}(P_{\theta_0},P_\theta)\ge0, \] with equality only at \(\theta=\theta_0\), whenever the expectations are well-defined.

Proof. The displayed difference is \(E_{\theta_0}\log(p_{\theta_0}/p_\theta)\). Since \(\log u\) is concave, \[ E_{\theta_0}\log\frac{p_\theta}{p_{\theta_0}}\le \log E_{\theta_0}\frac{p_\theta}{p_{\theta_0}}=\log\int p_\theta\,\mathrm{d}\mu=0. \] Multiplying by \(-1\) gives nonnegativity. Equality in Jensen requires \(p_\theta/p_{\theta_0}\) to be constant \(P_{\theta_0}\)-a.s.; normalization makes the constant \(1\), and identifiability gives \(\theta=\theta_0\).

Model classes

Parametric models have finite-dimensional \(\Theta\). Nonparametric models allow infinite-dimensional distributions or functions. Semiparametric models contain a finite-dimensional target and an infinite-dimensional nuisance component. This note focuses on frequentist inference; Bayesian priors and posterior analysis belong to the Bayesian statistics note, and risk-based optimality belongs to statistical decision theory.

Modèles statistiques et expériences

{% include note-topic-nav.html back_label="Retour" back_url="/notes/statistical-inference/" next_label="Page suivante" next_url="/notes/statistical-inference/likelihood-score-information/" %}

Un modèle statistique décrit mathématiquement les mécanismes possibles de génération des données. Ici, la probabilité fournit le langage du mécanisme aléatoire; l’inférence étudie ce que l’échantillon observé permet d’apprendre sur l’élément inconnu du modèle.

Définition. Une expérience statistique est un espace mesurable \((\mathcal X,\mathcal A)\) muni d’une famille de probabilités \(\mathcal P=\{P_\theta:\theta\in\Theta\}\). Une statistique est une application mesurable \(T:\mathcal X\to\mathcal T\). Le modèle est identifiable si \(P_\theta=P_{\theta'}\) implique \(\theta=\theta'\).

Modèles dominés

Le modèle est dominé s’il existe une mesure \(\sigma\)-finie \(\mu\) telle que \(P_\theta\ll\mu\) pour tout \(\theta\). On écrit alors \(p_\theta=\mathrm{d}P_\theta/\mathrm{d}\mu\). Pour des observations indépendantes, l’expérience produit a densité \(\prod_{i=1}^np_\theta(x_i)\).

Proposition. Si \(\theta_0\) est la vraie valeur dans un modèle dominé identifiable, alors \[ E_{\theta_0}\log p_{\theta_0}(X)-E_{\theta_0}\log p_\theta(X) =\operatorname{KL}(P_{\theta_0},P_\theta)\ge0, \] avec égalité seulement pour \(\theta=\theta_0\), lorsque les espérances existent.

Preuve. La différence vaut \(E_{\theta_0}\log(p_{\theta_0}/p_\theta)\). Par concavité de \(\log\), \[ E_{\theta_0}\log\frac{p_\theta}{p_{\theta_0}}\le \log E_{\theta_0}\frac{p_\theta}{p_{\theta_0}}=\log\int p_\theta\,\mathrm{d}\mu=0. \] On obtient la positivité en changeant de signe. Le cas d’égalité impose \(p_\theta=p_{\theta_0}\) presque sûrement; l’identifiabilité donne \(\theta=\theta_0\).

Types de modèles

Les modèles paramétriques ont un paramètre de dimension finie. Les modèles non paramétriques autorisent des distributions ou fonctions infiniment dimensionnelles. Les modèles semiparamétriques combinent une cible finie et un nuisance infini. Les priors et postérieurs relèvent de la note bayésienne; l’optimalité par risque relève de la théorie de la décision statistique.

统计模型与统计实验

{% include note-topic-nav.html back_label="返回" back_url="/notes/statistical-inference/" next_label="下一页" next_url="/notes/statistical-inference/likelihood-score-information/" %}

统计模型用数学方式描述数据可能如何生成。在本笔记中，概率论是描述数据生成机制的语言；统计推断关心的是：从观测样本中能学到模型中未知对象的哪些信息。

定义. 统计实验是一个可测样本空间 \((\mathcal X,\mathcal A)\) 以及一族概率测度 \(\mathcal P=\{P_\theta:\theta\in\Theta\}\)。统计量是可测映射 \(T:\mathcal X\to\mathcal T\)。若 \(P_\theta=P_{\theta'}\) 推出 \(\theta=\theta'\)，则模型称为可识别。

支配模型

若存在 \(\sigma\)-有限测度 \(\mu\)，使得每个 \(P_\theta\ll\mu\)，则模型称为支配模型。此时可写密度 \(p_\theta=\mathrm{d}P_\theta/\mathrm{d}\mu\)。对独立观测，乘积实验的密度为 \(\prod_{i=1}^np_\theta(x_i)\)。

命题. 若 \(\theta_0\) 是支配可识别模型中的真实值，则在期望有定义时， \[ E_{\theta_0}\log p_{\theta_0}(X)-E_{\theta_0}\log p_\theta(X) =\operatorname{KL}(P_{\theta_0},P_\theta)\ge0, \] 且等号只在 \(\theta=\theta_0\) 时成立。

证明. 上式差值为 \(E_{\theta_0}\log(p_{\theta_0}/p_\theta)\)。由 \(\log\) 的凹性， \[ E_{\theta_0}\log\frac{p_\theta}{p_{\theta_0}}\le \log E_{\theta_0}\frac{p_\theta}{p_{\theta_0}}=\log\int p_\theta\,\mathrm{d}\mu=0. \] 变号即得非负性。Jensen 等号要求 \(p_\theta/p_{\theta_0}\) 为常数，归一化使常数为 \(1\)；由可识别性得 \(\theta=\theta_0\)。

模型类型

参数模型有有限维参数；非参数模型允许无限维分布或函数；半参数模型有有限维目标参数和无限维 nuisance 成分。本笔记聚焦 frequentist inference；先验和后验放在 Bayesian statistics，风险意义下的最优性放在 statistical decision theory。
