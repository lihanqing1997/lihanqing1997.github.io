10. Confidence sets and pivots

{% include note-topic-nav.html back_label="Back" back_url="/notes/statistical-inference/" prev_label="Last page" prev_url="/notes/statistical-inference/likelihood-based-tests/" next_label="Next page" next_url="/notes/statistical-inference/finite-sample-optimality/" %}

A confidence set is a random set designed to cover the true parameter with a specified long-run probability. It is a procedure, not a probability statement about a fixed realized parameter.

Definition. A \(1-\alpha\) confidence set \(C_n(X)\subset\Theta\) satisfies \[ \inf_{\theta\in\Theta}P_\theta\{\theta\in C_n(X)\}\ge1-\alpha. \] A pivot is a statistic \(R(X,\theta)\) whose distribution does not depend on \(\theta\).

Test inversion. If \(\varphi_\theta\) is a level-\(\alpha\) test of \(H_0:\vartheta=\theta\) for every \(\theta\), then \[ C_n(X)=\{\theta:\varphi_\theta(X)=0\} \] is a \(1-\alpha\) confidence set.

Proof. Under the true parameter \(\theta\), the event \(\theta\notin C_n(X)\) is exactly the event that the level-\(\alpha\) test of \(H_0:\vartheta=\theta\) rejects. Its probability is at most \(\alpha\). Therefore \(P_\theta\{\theta\in C_n(X)\}\ge1-\alpha\).

Asymptotic pivots

If \((T_n-\theta)/\widehat{\operatorname{se}}(T_n)\rightsquigarrow N(0,1)\), then \[ T_n\pm z_{1-\alpha/2}\widehat{\operatorname{se}}(T_n) \] is an asymptotic confidence interval. Likelihood ratio regions use \(\{2(\ell_n(\hat\theta)-\ell_n(\theta))\le\chi^2_{r,1-\alpha}\}\), often with better invariance properties than Wald intervals.

10. Régions de confiance et pivots

{% include note-topic-nav.html back_label="Retour" back_url="/notes/statistical-inference/" prev_label="Page précédente" prev_url="/notes/statistical-inference/likelihood-based-tests/" next_label="Page suivante" next_url="/notes/statistical-inference/finite-sample-optimality/" %}

Une région de confiance est un ensemble aléatoire conçu pour couvrir le vrai paramètre avec une probabilité de long terme prescrite. Ce n’est pas une probabilité a posteriori sur un paramètre fixé.

Définition. Une région de confiance de niveau \(1-\alpha\) satisfait \[ \inf_{\theta\in\Theta}P_\theta\{\theta\in C_n(X)\}\ge1-\alpha. \] Un pivot est une statistique \(R(X,\theta)\) dont la loi ne dépend pas de \(\theta\).

Inversion des tests. Si \(\varphi_\theta\) est un test de niveau \(\alpha\) de \(H_0:\vartheta=\theta\) pour tout \(\theta\), alors \[ C_n(X)=\{\theta:\varphi_\theta(X)=0\} \] est une région de confiance de niveau \(1-\alpha\).

Preuve. Sous le vrai paramètre \(\theta\), l’événement \(\theta\notin C_n(X)\) est exactement le rejet du test de \(H_0:\vartheta=\theta\). Sa probabilité est au plus \(\alpha\). Donc la couverture est au moins \(1-\alpha\).

Pivots asymptotiques

Si \((T_n-\theta)/\widehat{\operatorname{se}}(T_n)\rightsquigarrow N(0,1)\), alors \[ T_n\pm z_{1-\alpha/2}\widehat{\operatorname{se}}(T_n) \] est un intervalle de confiance asymptotique. Les régions par rapport de vraisemblance sont souvent plus invariantes que les intervalles de Wald.

10. 置信集与枢轴量

{% include note-topic-nav.html back_label="返回" back_url="/notes/statistical-inference/" prev_label="上一页" prev_url="/notes/statistical-inference/likelihood-based-tests/" next_label="下一页" next_url="/notes/statistical-inference/finite-sample-optimality/" %}

置信集是随机集合，设计目标是在长期重复抽样中以指定概率覆盖真实参数。它不是对已经固定的参数给出的后验概率陈述。

定义. 若随机集合 \(C_n(X)\subset\Theta\) 满足 \[ \inf_{\theta\in\Theta}P_\theta\{\theta\in C_n(X)\}\ge1-\alpha, \] 则称它为 \(1-\alpha\) 置信集。若统计量 \(R(X,\theta)\) 的分布不依赖 \(\theta\)，则称它为枢轴量。

检验反演. 若对每个 \(\theta\)，\(\varphi_\theta\) 都是检验 \(H_0:\vartheta=\theta\) 的 level-\(\alpha\) 检验，则 \[ C_n(X)=\{\theta:\varphi_\theta(X)=0\} \] 是 \(1-\alpha\) 置信集。

证明. 在真实参数 \(\theta\) 下，事件 \(\theta\notin C_n(X)\) 正是检验 \(H_0:\vartheta=\theta\) 被拒绝的事件。其概率不超过 \(\alpha\)。因此覆盖概率至少为 \(1-\alpha\)。

渐近枢轴量

若 \((T_n-\theta)/\widehat{\operatorname{se}}(T_n)\rightsquigarrow N(0,1)\)，则 \[ T_n\pm z_{1-\alpha/2}\widehat{\operatorname{se}}(T_n) \] 是渐近置信区间。似然比区域 \(\{2(\ell_n(\hat\theta)-\ell_n(\theta))\le\chi^2_{r,1-\alpha}\}\) 通常比 Wald 区间有更好的不变性。
