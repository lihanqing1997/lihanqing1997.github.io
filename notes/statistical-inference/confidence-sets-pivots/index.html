A confidence set is a random set designed to cover the true parameter with a specified long-run probability. It is a procedure, not a probability statement about a fixed realized parameter.

Definition. A \(1-\alpha\) confidence set \(C_n(X)\subset\Theta\) satisfies \[ \inf_{\theta\in\Theta}P_\theta\{\theta\in C_n(X)\}\ge1-\alpha. \] A pivot is a statistic \(R(X,\theta)\) whose distribution does not depend on \(\theta\).

Test inversion. If \(\varphi_\theta\) is a level-\(\alpha\) test of \(H_0:\vartheta=\theta\) for every \(\theta\), then \[ C_n(X)=\{\theta:\varphi_\theta(X)=0\} \] is a \(1-\alpha\) confidence set.

Proof. Under the true parameter \(\theta\), the event \(\theta\notin C_n(X)\) is exactly the event that the level-\(\alpha\) test of \(H_0:\vartheta=\theta\) rejects. Its probability is at most \(\alpha\). Therefore \(P_\theta\{\theta\in C_n(X)\}\ge1-\alpha\).

Asymptotic pivots

If \((T_n-\theta)/\widehat{\operatorname{se}}(T_n)\rightsquigarrow N(0,1)\), then \[ T_n\pm z_{1-\alpha/2}\widehat{\operatorname{se}}(T_n) \] is an asymptotic confidence interval. Likelihood ratio regions use \(\{2(\ell_n(\hat\theta)-\ell_n(\theta))\le\chi^2_{r,1-\alpha}\}\), often with better invariance properties than Wald intervals.
