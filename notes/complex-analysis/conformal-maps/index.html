12. Conformal maps

{% include note-topic-nav.html back_label="Back" back_url="/notes/complex-analysis/" prev_label="Last page" prev_url="/notes/complex-analysis/argument-principle-and-rouche-theorem/" next_label="Next page" next_url="/notes/complex-analysis/mobius-transformations/" %}

Conformal maps records the objects, main results, proof mechanisms, and boundary cases needed for the complex analysis note.

Definition

Definition. A conformal map is a holomorphic map with nonzero derivative; it preserves oriented angles locally and is locally biholomorphic.

Local inverse theorem

Theorem. If \(f'(z_0)\ne0\), then \(f\) is injective in a neighborhood of \(z_0\) and has a holomorphic local inverse.

Proof

Proof. Write the first-order expansion \(f(z)=f(z_0)+f'(z_0)(z-z_0)+o(|z-z_0|)\). The inverse theorem follows by contraction or by the argument principle on small circles.

Remark

Remark. Conformal maps translate geometry into holomorphic function theory. Local angle preservation is only the first layer; global domains matter.
