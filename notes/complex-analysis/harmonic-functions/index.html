15. Harmonic functions

{% include note-topic-nav.html back_label="Back" back_url="/notes/complex-analysis/" prev_label="Last page" prev_url="/notes/complex-analysis/riemann-mapping-theorem/" next_label="Next page" next_url="/notes/complex-analysis/analytic-continuation/" %}

Harmonic functions records the objects, main results, proof mechanisms, and boundary cases needed for the complex analysis note.

Definition

Definition. A real-valued function is harmonic if \(\Delta u=0\). Real and imaginary parts of holomorphic functions are harmonic, and locally many harmonic functions have harmonic conjugates.

Proposition

Proposition. Harmonic functions satisfy the mean-value property and maximum principle. On simply connected domains, a harmonic function has a harmonic conjugate exactly when the associated closed one-form is exact.

Proof

Proof. Cauchy-Riemann equations imply the Laplacian vanishes. Conversely, solve for a conjugate by integrating the closed form \(-u_y\,\mathrm{d}x+u_x\,\mathrm{d}y\).

Remark

Remark. Harmonic functions connect complex analysis to potential theory and elliptic PDE.
