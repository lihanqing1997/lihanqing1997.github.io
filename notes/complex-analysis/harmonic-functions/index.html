Harmonicity

Definition. A real-valued \(C^2\) function \(u\) on a plane domain is harmonic if \(\Delta u=u_{xx}+u_{yy}=0\). Real and imaginary parts of holomorphic functions are harmonic because the Cauchy-Riemann equations imply the Laplace equation.

Harmonic conjugates

Proposition. On a simply connected domain, every harmonic function \(u\) has a harmonic conjugate \(v\) such that \(u+iv\) is holomorphic. The one-form \(-u_y\,\mathrm{d}x+u_x\,\mathrm{d}y\) is closed, and simply connectedness makes it exact.

Mean value property

Theorem. A harmonic function equals its average over every sufficiently small circle centered at the point. Conversely, a continuous function with the mean value property is harmonic.

Maximum principle

Corollary. A nonconstant harmonic function has no interior maximum or minimum. Apply the mean value property: if the average equals an extreme value, the function must be locally constant.

Poisson integral formula

Theorem. For continuous boundary data \(\varphi\) on the unit circle, the harmonic solution in the disc is \[ u(re^{i\theta})=\frac{1}{2\pi}\int_0^{2\pi}P_r(\theta-t)\varphi(e^{it})\,\mathrm{d}t, \] where \(P_r\) is the Poisson kernel.

Dirichlet problem

Remark. Harmonic functions connect complex analysis with potential theory. Boundary values determine interior values in well-behaved domains, mirroring Cauchy's boundary representation for holomorphic functions.
