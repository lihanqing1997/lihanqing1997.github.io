Paths and contour integrals

Definition. A path is a piecewise \(C^1\) map \(\gamma:[a,b]\to U\). For a continuous function \(f\), define \[ \int_\gamma f(z)\,\mathrm{d}z=\int_a^b f(\gamma(t))\gamma'(t)\,\mathrm{d}t. \] Reparametrization preserving orientation does not change the integral.

Length estimate

Proposition. If \(M=\sup_{z\in\gamma}|f(z)|\) and \(L(\gamma)\) is the path length, then \(|\int_\gamma f\,\mathrm{d}z|\le M L(\gamma)\). This estimate is the basic analytic control behind limiting arguments with contours.

Primitives

Theorem. If \(f=F'\) on \(U\), then \(\int_\gamma f\,\mathrm{d}z=F(\gamma(b))-F(\gamma(a))\). Hence every closed contour integral of a function with a primitive is zero.

Path independence

Corollary. On a connected open set, a continuous function has a primitive if and only if its integral over every closed path is zero. One constructs the primitive by fixing a base point and integrating along paths.

Integral of reciprocal powers

Calculation. On the circle \(\gamma(t)=a+re^{it}\), \(0\le t\le2\pi\), one has \(\int_\gamma (z-a)^n\,\mathrm{d}z=0\) for \(n\ne-1\), while \(\int_\gamma (z-a)^{-1}\,\mathrm{d}z=2\pi i\). This single exceptional exponent is the source of residues.

Orientation

Remark. Reversing a contour changes the sign of the integral. Positively oriented boundary curves keep the region on the left as they are traversed.
