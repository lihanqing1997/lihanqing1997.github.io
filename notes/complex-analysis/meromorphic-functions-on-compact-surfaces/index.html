Meromorphic functions on compact surfaces records the objects, main results, proof mechanisms, and boundary cases needed for the complex analysis note.

Definition

Definition. A meromorphic function is holomorphic except for poles. On compact Riemann surfaces, zeros and poles are global data constrained by compactness.

Proposition

Proposition. On a compact Riemann surface, a nonconstant meromorphic function has as many zeros as poles when counted with multiplicity, after viewing it as a map to the Riemann sphere.

Proof

Proof. Apply the argument principle in coordinate charts or use degree theory for holomorphic maps to the sphere. Compactness prevents boundary contributions.

Remark

Remark. This page points beyond plane complex analysis toward Riemann surfaces, divisors, and algebraic geometry.
