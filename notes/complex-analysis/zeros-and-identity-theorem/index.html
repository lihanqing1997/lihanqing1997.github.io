9. Zeros and identity theorem

{% include note-topic-nav.html back_label="Back" back_url="/notes/complex-analysis/" prev_label="Last page" prev_url="/notes/complex-analysis/maximum-principle/" next_label="Next page" next_url="/notes/complex-analysis/isolated-singularities/" %}

Order of a zero

Definition. If \(f\) is holomorphic near \(a\) and not identically zero, then \(a\) is a zero of order \(m\) when \(f(z)=(z-a)^m g(z)\) with \(g\) holomorphic and \(g(a)\ne0\). The order is the first nonzero Taylor coefficient.

Isolated zeros

Theorem. Zeros of a nonzero holomorphic function are isolated. If zeros accumulated inside the domain, the Taylor expansion at the accumulation point would have all coefficients zero, forcing the function to vanish nearby.

Identity theorem

Theorem. If two holomorphic functions on a connected domain agree on a set with an accumulation point in the domain, then they agree everywhere. Apply isolated zeros to their difference.

Analytic continuation uniqueness

Corollary. A holomorphic continuation, if it exists along a connected overlap, is unique. Local agreement propagates across overlapping discs by the identity theorem.

Multiplicity and products

Proposition. Orders add under multiplication: \(\operatorname{ord}_a(fg)=\operatorname{ord}_a f+\operatorname{ord}_a g\), with poles treated later as negative orders. This arithmetic of orders underlies the argument principle.

Real-variable contrast

Remark. A smooth real function can vanish on an interval and then become nonzero. A holomorphic function cannot do this unless it is globally zero on the connected component.
