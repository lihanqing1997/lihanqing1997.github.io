4. Naive set theory and paradoxes

{% include note-topic-nav.html back_label="Back" back_url="/notes/math-history/" prev_label="Last page" prev_url="/notes/math-history/dedekind-cantor-and-arithmetization/" next_label="Next page" next_url="/notes/math-history/zermelo-fraenkel-set-theory-and-choice/" %}

Cantor opened a new universe: sets could be counted, compared, and organized even when infinite. But the first language of sets was too generous. If every property defines a set, then some properties define contradictions.

Naive comprehension

Definition. Naive comprehension is the principle that for every property \(P(x)\) there is a set \(\{x:P(x)\}\) consisting exactly of the objects satisfying it.

This principle expresses a natural habit of thought: collect all objects with a feature. The paradoxes show that unrestricted collecting is not innocent.

Russell's paradox

Theorem. Unrestricted comprehension is inconsistent.

Proof. Let \(R=\{x:x\notin x\}\). If \(R\in R\), then by definition \(R\notin R\). If \(R\notin R\), then by definition \(R\in R\). Both alternatives contradict themselves.

Other warnings

Russell's paradox was not alone. Burali-Forti's paradox involved the class of all ordinals; Cantor's paradox involved the class of all sets and the power-set theorem. Each warning pointed to the same moral: the universe of sets cannot itself be handled as an ordinary set.

Repair by restriction

Proposition. Restricted separation avoids Russell's contradiction: from an already given set \(A\), one may form \(\{x\in A:P(x)\}\), but one may not form the set of all objects satisfying \(P\).

The crisis did not reject sets. It made set theory more disciplined. Modern axioms preserve Cantor's productive ideas while blocking self-referential totalities from becoming sets.

Paradox as diagnosis

The paradoxes were not mere puzzles; they diagnosed a hidden assumption that every meaningful condition names a completed object. Once this assumption was exposed, mathematicians could separate language from existence: a formula may be meaningful without determining a set.

Different repairs emphasize different lessons. Type theory blocks self-reference by sorting objects into levels; Zermelo-Fraenkel set theory blocks unrestricted totalities by axiomatizing allowed constructions; category-theoretic foundations later shift attention from membership to maps and universal properties.

Mathematicians to remember

Cantor created the paradise of sets; Frege tried to ground arithmetic in logic and was struck by Russell's letter; Russell isolated the paradox; Cesare Burali-Forti's work on ordinals exposed another fracture; Zermelo began the axiomatic repair.

4. Théorie naïve des ensembles et paradoxes

{% include note-topic-nav.html back_label="Retour" back_url="/notes/math-history/" prev_label="Page précédente" prev_url="/notes/math-history/dedekind-cantor-and-arithmetization/" next_label="Page suivante" next_url="/notes/math-history/zermelo-fraenkel-set-theory-and-choice/" %}

Cantor ouvrit un nouvel univers : les ensembles, même infinis, pouvaient être comptés, comparés et organisés. Mais le premier langage des ensembles était trop permissif. Si toute propriété définit un ensemble, certaines propriétés définissent des contradictions.

Compréhension naïve

Définition. La compréhension naïve affirme que pour toute propriété \(P(x)\), il existe un ensemble \(\{x:P(x)\}\) formé exactement des objets qui la satisfont.

Ce principe formalise une habitude naturelle : rassembler tous les objets ayant un trait commun. Les paradoxes montrent que ce geste doit être limité.

Paradoxe de Russell

Théorème. La compréhension illimitée est contradictoire.

Preuve. Posons \(R=\{x:x\notin x\}\). Si \(R\in R\), alors \(R\notin R\). Si \(R\notin R\), alors \(R\in R\). Les deux possibilités se contredisent.

Autres avertissements

Le paradoxe de Russell n'est pas isolé. Celui de Burali-Forti concerne la totalité des ordinaux ; celui de Cantor concerne la totalité des ensembles et le théorème de l'ensemble des parties. Tous disent que l'univers des ensembles ne peut pas être traité comme un ensemble ordinaire.

Réparation par restriction

Proposition. La séparation restreinte évite la contradiction : à partir d'un ensemble déjà donné \(A\), on peut former \(\{x\in A:P(x)\}\), mais non l'ensemble de tous les objets satisfaisant \(P\).

La crise ne détruit pas la théorie des ensembles. Elle la discipline : les axiomes modernes gardent la fécondité de Cantor tout en empêchant les totalités auto-référentielles de devenir des ensembles.

Le paradoxe comme diagnostic

Les paradoxes ne sont pas de simples énigmes : ils révèlent l'hypothèse cachée selon laquelle toute condition signifiante détermine un objet achevé. Une formule peut avoir un sens sans définir un ensemble.

Les réparations tirent des leçons différentes. La théorie des types empêche l'auto-référence par des niveaux ; Zermelo-Fraenkel remplace les totalités sans restriction par des constructions axiomatisées ; les fondements catégoriques déplaceront plus tard l'attention de l'appartenance vers les applications et les propriétés universelles.

Mathématiciens à retenir

Cantor crée l'univers des ensembles ; Frege tente de fonder l'arithmétique sur la logique ; Russell met au jour le paradoxe ; Burali-Forti révèle une fracture avec les ordinaux ; Zermelo commence la réparation axiomatique.

4. 朴素集合论与悖论

{% include note-topic-nav.html back_label="返回" back_url="/notes/math-history/" prev_label="上一页" prev_url="/notes/math-history/dedekind-cantor-and-arithmetization/" next_label="下一页" next_url="/notes/math-history/zermelo-fraenkel-set-theory-and-choice/" %}

康托尔打开了一个新世界：集合可以被计数、比较、组织，即使它们是无限的。但最初的集合语言过于慷慨。如果任意性质都能定义一个集合，那么某些性质会定义出矛盾。

朴素概括原则

定义. 朴素概括原则认为，对任意性质 \(P(x)\)，都存在集合 \(\{x:P(x)\}\)，其元素恰好是满足该性质的对象。

这符合日常思维：把具有同一特征的东西收集起来。但悖论说明，“任意收集”在数学宇宙中并不安全。

罗素悖论

定理. 不受限制的概括原则是不一致的。

证明. 令 \(R=\{x:x\notin x\}\)。若 \(R\in R\)，按定义应有 \(R\notin R\)；若 \(R\notin R\)，按定义又应有 \(R\in R\)。两种情况都矛盾。

其他警告

罗素悖论不是孤例。布拉利-福尔蒂悖论涉及所有序数的总体；康托尔悖论涉及所有集合的总体和幂集定理。它们共同说明：集合宇宙本身不能被当作普通集合处理。

通过限制来修复

命题. 受限制的分离原则可以避免罗素矛盾：从已经给定的集合 \(A\) 出发，可以形成 \(\{x\in A:P(x)\}\)，但不能直接形成“所有满足 \(P\) 的对象”的集合。

这场危机没有废弃集合，而是让集合论变得更有纪律。现代公理保留了康托尔思想的创造力，同时阻止自指总体成为集合。

悖论作为诊断

这些悖论不只是智力游戏，而是诊断出一个隐藏假设：只要一个条件有意义，它就决定一个完成的对象。悖论之后，人们学会区分“公式有意义”和“公式定义了集合”。

不同修复方案强调不同教训。类型论用层级阻止自指；策梅洛-弗兰克尔集合论用公理规定允许的构造，避免无约束总体；后来的范畴论基础则把注意力从属于关系转向映射和泛性质。

应当记住的人

康托尔创造集合论；弗雷格试图以逻辑奠基算术，却被罗素来信击中；罗素明确提出悖论；布拉利-福尔蒂在序数中发现裂缝；策梅洛开始公理化修复。
