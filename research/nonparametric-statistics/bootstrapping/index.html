Bootstrapping

{% include research-topic-nav.html back_label="Back" back_url="/research/nonparametric-statistics/" prev_label="Last page" prev_url="/research/nonparametric-statistics/empirical-likelihood/" next_label="Next page" next_url="/research/nonparametric-statistics/jackknife/" %}

The bootstrap approximates the sampling distribution of a statistic by resampling from the empirical distribution.

Nonparametric bootstrap

Given observations \(X_1,\ldots,X_n\), define the empirical distribution \(P_n=n^{-1}\sum_{i=1}^n\delta_{X_i}\). A bootstrap sample \(X_1^*,\ldots,X_n^*\) is drawn i.i.d. from \(P_n\). For a statistic \(T_n=T(X_1,\ldots,X_n)\), compute \(T_n^*=T(X_1^*,\ldots,X_n^*)\).

Consistency idea

The bootstrap is consistent when the conditional law of \(\sqrt n(T_n^*-T_n)\) given the data approximates the law of \(\sqrt n(T_n-\theta)\). For smooth functionals \(T(P)\), this follows from an asymptotic linear expansion

\[T(P_n)-T(P)=\frac1n\sum_{i=1}^n \phi(X_i)+o_p(n^{-1/2}),\]

where \(E\phi(X)=0\) and \(E\phi(X)^2<\infty\).

Use

The bootstrap can estimate standard errors, quantiles, and confidence intervals. It is especially useful when the analytic variance formula is complicated.

Caution

The ordinary bootstrap may fail for non-smooth statistics, boundary parameters, heavy tails, or dependent data unless the resampling scheme is modified.

Le bootstrap approxime la loi d'échantillonnage d'une statistique en rééchantillonnant depuis la distribution empirique.

Bootstrap non paramétrique

Pour \(X_1,\ldots,X_n\), définir \(P_n=n^{-1}\sum_{i=1}^n\delta_{X_i}\). Un échantillon bootstrap \(X_1^*,\ldots,X_n^*\) est tiré i.i.d. de \(P_n\). Pour une statistique \(T_n=T(X_1,\ldots,X_n)\), on calcule \(T_n^*=T(X_1^*,\ldots,X_n^*)\).

Idée de consistance

Le bootstrap est consistant si la loi conditionnelle de \(\sqrt n(T_n^*-T_n)\) donnée les données approxime la loi de \(\sqrt n(T_n-\theta)\). Pour les fonctionnelles lisses \(T(P)\), cela vient d'une expansion asymptotiquement linéaire

\[T(P_n)-T(P)=\frac1n\sum_{i=1}^n \phi(X_i)+o_p(n^{-1/2}),\]

où \(E\phi(X)=0\) et \(E\phi(X)^2<\infty\).

Usage

Le bootstrap permet d'estimer erreurs standards, quantiles et intervalles de confiance, surtout lorsque la variance analytique est compliquée.

Attention

Le bootstrap ordinaire peut échouer pour des statistiques non lisses, des paramètres au bord, des queues lourdes ou des données dépendantes sans modification du schéma de rééchantillonnage.

Bootstrap 通过从经验分布中重新抽样，近似统计量的抽样分布。

非参数 Bootstrap

给定 \(X_1,\ldots,X_n\)，定义经验分布 \(P_n=n^{-1}\sum_{i=1}^n\delta_{X_i}\)。Bootstrap 样本 \(X_1^*,\ldots,X_n^*\) 独立同分布地从 \(P_n\) 抽取。若 \(T_n=T(X_1,\ldots,X_n)\)，则计算 \(T_n^*=T(X_1^*,\ldots,X_n^*)\)。

一致性思想

若给定数据后 \(\sqrt n(T_n^*-T_n)\) 的条件分布能够近似 \(\sqrt n(T_n-\theta)\) 的分布，则 bootstrap 一致。对光滑泛函 \(T(P)\)，这通常来自渐近线性展开

\[T(P_n)-T(P)=\frac1n\sum_{i=1}^n \phi(X_i)+o_p(n^{-1/2}),\]

其中 \(E\phi(X)=0\) 且 \(E\phi(X)^2<\infty\)。

用途

Bootstrap 可用于估计标准误、分位数和置信区间，特别适合解析方差公式复杂的情形。

注意

普通 bootstrap 对非光滑统计量、边界参数、重尾数据或相关数据可能失效，需要修改重抽样方案。
